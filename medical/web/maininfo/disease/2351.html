病态窦房结综合征
病态窦房结综合征是指由于窦房结及其邻近组织缺血、炎症等因素引起窦房结起搏功能和（或）窦房传导功能障碍，临床上出现窦停、传导阻滞、心动过速等心律失常和各种症状的一组症候群，轻者只出现轻度的头晕、心悸、活动耐力降低，严重者发生晕厥甚至猝死。大多数患者在40岁以上出现症状，60～70岁最多见。对严重症状的患者目前最有效、最安全的治疗方法是安装永久性心脏起搏器。

发病机制

病因

1.病因 病窦综合征的病因可分为心脏病、药物、神经性及其他。

（1）心脏疾病：冠心病、心肌病导致窦房结及其周围组织发生缺血，风湿性心瓣膜病、心肌炎、心脏手术及放射治疗也可损伤窦房结及其周围组织，窦房结的纤维化、炎症、脂肪浸润与退行性变等造成窦房结功能障碍。

（2）药物：抗心律失常药物，如洋地黄、美托洛尔、奎尼丁、胺碘酮、维拉帕米抑制窦房结结功能可发生窦房结功能障碍，锂、西咪替丁、阿米替林也发生窦房结功能障碍。

（3）神经源性：迷走神经张力增高、颈动脉窦综合征、血管迷走神经性晕厥。

（4）其他：高钾血症、高碳酸血症、甲状腺功能减退、颅内压增高、低温、脓毒血症等均可引起窦房结功能障碍。

院前急救判断

1.电话询问

n 年龄＞40岁？

n 突然发病

n 原发疾病情况

n 起病时是否出现过头晕、黑蒙、抽搐、胸痛

n 有无恶心、呕吐、血压低等

n 生命体征稳定吗

2.途中思考

n 原发病轻重

n 心律失常的原因是什么

3.可危及生命的事件

n 晕厥

n 阿斯综合征

n 原发病加重？

n 猝死

4.现场评估

4.1临床症状除原发疾病症状外

①重症患者心绞痛，心力衰竭加重，晕厥。

②血压降低。

4.2查体所见

①皮肤黏膜苍白，冰凉。

②心率加快或缓慢。

4.3急危重症评估

出现以下症状为危急重症：

①患者发病急，窦性停搏，血压降低。

②患者呼吸不规则。

③意识障碍或有抽搐。

院前急救处置

1.立即解除窦性停搏，血压降低，呼吸不规则、抽搐等危及生命的情况。

2.立即开放气道，保持气道通畅。

3. 患者取舒适位，坐、卧或半坐位。

4. 监测血压、心率、呼吸、氧饱和度。力争保持在理想状态：血压90～160/60～100mmHg，心率50～100次/分，呼吸12～25次/分。

5. 开放静脉通道，对症用药。

选择医疗机构与转运

1.选择医疗机构

1.1对伴有呼吸衰竭心衰的重症患者，有严重缓慢心律，阿斯综合征的可能发生心脏骤停，转运不及时可能会导致预后不良，应该选择转送到能24h对患者进行相关检查，有专科治疗能力，可急诊处理的具备CCUICU的急救医疗机构。

1.2出现意识障碍、休克状态、心跳呼吸停止等，在紧急处置的同时应短时间内将患者转运到就近的急救医疗机构。

1.3对意识清楚的、无危及生命的情况，可送往日常就诊的医疗机构。

2.转运时注意转运途中严密监控神志、呼吸、血压、心率、心律和SpO2等。

急诊科诊断与治疗

初步评估与急救

病态窦房结综合症的临床症状差异较大，可以仅有轻度的头晕、心悸、活耐力降低，到严重的血流力学障碍，引起黑蒙、晕厥甚至死亡，根据病情的需要给予恰当治疗，短期使用阿托品，异丙肾上腺可以用物提高心率，但要注意物的副作用。严重症状的安装心脏起搏器。

全面诊断与检查

1.出现乏力、头昏、眼花、失眠、记忆力差、反应迟钝或易激动等轻症，年轻者要与神经官能症鉴别，老年人还易被误诊为脑卒中或衰老综合征。

2.出现眼黑蒙、晕厥、抽搐等急病症。注意与癫痫发作、体位性低血压等鉴别。

3.出现心动过缓或心动过速，原有心脏病症状加重而忽视病态窦房结综合征。

4.慢-快综合征还可能导致血管栓塞，出现瘫痪中风。

5.心排出量过低时还严重影响肾脏灌注而致尿少、肾功能障碍。

【辅助检查】

1.24小时心电监测

可记录到实时的心电图表现，心律失常发生时间、持续长短，如恶性心律失常严重的窦性心动过缓，小于50次/分；窦性停搏；完全性房室传导阻滞、室性心动过速、室颤。为临床医生判断治疗提供依据，以及直接观察药物的治疗作用。

2.药物试验

为排除心动过缓是否神经张力因素抑制窦房结，常做阿托品试验和异丙肾上腺素试验，若注射后心率增加达90次/分者提示窦房结功能尚好。有青光眼或明显前列腺肥大的患者慎用。

3.心房调搏方法

经食管或直接心房调搏检测窦房结功能，对于病窦综合征患者，测定的窦房结恢复时间（SNRI）和窦房传导时间（SACT）常显著延长超过正常高限。

4.24小时动态心电图监测

可检测到最快和最慢心率、窦性停搏、窦房阻滞、房室传导阻滞等心律失常表现。同时可了解到患者症状与心电改变的关系。

5．心脏电生理检查 可检测窦房结功能、心房、房室结、室内传导系统功能。区别窦房结的冲动形成异常和冲动传导异常。

6.实验室检查和其他检查

6.1电解质:可发现高钾、低磷、低钙及酸中毒。

6.2血常规:可发现贫血、感染性疾病。

6.3动脉血气:可确定缺氧程度，血液酸碱度，作为给氧或机械通气的依据。

6.4肝肾功能:可了解肝肾功能。

6.5心电图:可提供心脏疾病、心律失常信息。

6.6心脏标记物及D-二聚体:可协助诊断心肌缺血或肺栓塞。

6.7胸部X线 :可过滤许多心、肺疾病；怀疑肺栓塞时可做增强螺旋CT检查。

6.8超声心动图:可确定心包、心内结构异常及心功能异常。

6.9脑钠肽:鉴别肺源性与心源性呼吸困难。

6.10.心血管造影 可发现病变血管。

窦房结功能损害的部位和严重程度不同，SSS的心电图表现不同，临床上常根据心电图表现将SSS分为以下类型。

（1）一型：严重而持久的窦性心动过缓。最为常见，占75%~80%，也常为SSS的早期表现。其窦性频率低至30～40次/分，且白昼及活动前后无明显变化。

（2）二型：窦房传导阻滞。严重窦性心动过缓患者常伴有窦房传导阻滞，其中以二度窦房传导阻滞最为常见。

（3）三型：窦性停搏。当窦房结功能进一步恶化，造成周期性窦性激动形成障碍，导致窦性停搏。其心电图表现为P波脱落和较长时间的窦性静止。其间歇时间与基础窦性周期无倍数关系。这点在与二度窦房传导阻滞鉴别时非常重要。

（4）双结病变：当病变累及房室结时出现双结病变，心电图表现为在以上3型的基础上，不能及时出现交界区逸搏（逸搏周期＞1.5s）或逸搏频率＜40次/分。

（5）慢快综合征：是指在一、二、三型的基础上，反复出现阵发性室上性心动过速、心房扑动或心房颤动。其中以心房颤动最为常见。

（6）全传导系统病变：病变累及多个传导系统时表现为窦房结传导阻滞、房室结传导阻滞及心室内传导阻滞。为SSS的特殊类型。

急诊处理

1.治疗原则

治疗原发病是关键，提高心率药物治疗、植入心脏起搏器仅能缓解临床症状。各种抗心律失常药物常有明显和不能耐受的不良反应，不适合长期治疗。故在药物治疗中要把握时机及控制剂量。

2.对因治疗

如冠状动脉硬化者应抗动脉硬化，应用硝酸甘油等改善冠脉供血，严重者植入支架。对于急性心肌炎可短期使用激素减轻水肿改善传导。严重感染者加强抗感染治疗。

3.对症治疗

（1）无快速性心律失常的患者：无禁忌证时可短期用阿托品或异丙肾上腺素以提高心率缓解症状。避免使用β受体阻滞剂及钙拮抗剂等减慢心率的药物。

（2）植入心脏起搏器：根据病情需要选择各种类型起搏器。

疗效标准

1.全愈 症消失，心率在60次/分，心电图正常。

2.好转 症好转，心率在60次/分左右。

3.未愈 症无改善。

出院标准

1.可逆性经治纠正病因，恢复窦性心律，心率在60次/分，临床症状消失可出院。

2.永久性经安装起搏器治，起搏心律，心率在60次/分，临床症状消失可出院。

3.心率在60次/分以下，临床症状无好转，出院需要患者签名说有猝死的风险。

急诊建议
1.SSS的病因较复杂，因此，在诊断该综合征时应仔细录找病因，对原发疾病治疗。

2.对于因急性心肌梗死、急性心肌炎等引起的暂时性窦房结功能紊乱可采用临时人工起搏器治疗。

3.严重的SSS药物治疗效果不理想，符合以下况之一者应安装永久起搏器：SSS伴有阿斯综合征发作或虽无症状但心电图示窦房阻滞或窦性停搏≥3.0s； SSS合二度Ⅱ型以上的A-VB有阿斯综合征或晕厥先兆；潜在性SSS合各种顽固性心动过速需要长期药物治疗。
