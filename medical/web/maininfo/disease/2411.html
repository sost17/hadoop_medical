急性胆道感染与胆石症
急性胆道感染与胆石症是肝胆系统的常见疾病，两者有时不能截然分开。急性胆道感染是指胆道系统内发生的急性细菌性炎症，临床上常分为急性胆囊炎和急性胆管炎。胆石症指胆道系统（包括胆囊和胆管）任何部位发生结石的疾病，在临床上也属于常见病。胆石症按发病部位可分为胆囊结石和胆管结石，其中胆管结石又分为肝内胆管结石和肝外胆管结石（胆总管结石和肝总管结石）。胆道感染与胆石病互为因果关系。胆石症可致使胆道感染。反复发作的胆道感染又是胆石形成的重要因素。本章主要论述急性胆囊炎、急性胆管炎、胆囊结石、胆管结石。急性梗阻性化脓性胆管炎（AOSC）作为急性胆管炎的严重阶段，常威胁患者生命，本章也作为主要论述。

发病机制

病因

急性胆道感染是指胆囊或胆管细菌性感染炎症。

急性胆囊炎绝大多数（95%）为结石性胆囊炎，极少数（5%）为非结石性胆囊炎。引起急性结石性胆囊炎主要原因是①胆囊管梗阻，多由于胆囊结石影胆汁排出不畅，最终引起胆囊炎症及水肿。②细菌感染，主要致病菌为大肠埃希菌、克雷伯菌、铜绿假单胞菌以及厌氧菌等。多由于胆道逆行感染或经血流感染。急性非结石胆囊炎病因目前不清楚，通常多在危重患者中发病，可能与胆汁淤滞和缺血有关。

急性胆管感染最常见的病因是肝内外胆管结石，其次是胆道寄生与胆管狭窄有关。近年来，胆道内镜操作后引起者多见。

胆石症包括胆囊结石及肝内外胆管结石。胆石通常分为三类：①胆固醇类结石（约占所有种类的80%）；②胆色素类结石；③其他结石。胆石在哪个部位就称为哪个部位的结石。如结石在胆囊内就称为胆囊结石。肝内外胆管结石以左右肝管汇合部解剖标记。胆囊结石的成因复杂，多种因素可造成。任何影胆固醇、胆汁酸与磷脂浓度比例和造成胆汁淤滞的因素都能导致结石的形成。肝外胆管结石病因多由于胆道感染、梗阻、寄生诱因以及胆囊内结石排出并停留在胆管内所致。肝内胆管结石病因复杂，一般认为与胆道感染、寄生、胆汁停滞、营养不良等有关。

病理

一、急性胆囊炎

1. 急性结石性胆囊炎

病变开始时结石阻塞胆囊管，从而造成胆囊内胆汁不能排出，胆囊肿大，压力增高，黏膜充血水肿、渗出增加。如及时梗阻解除，炎症消退，大部分组织恢复原来结构，胆囊不留疤痕，称为急性单纯性胆囊炎。如果梗阻未解除或炎症未控制，胆囊内可出现充满脓液，成为急性化脓性胆囊炎。最终，胆囊可发生坏疽、穿孔，成为坏疽性胆囊炎。坏疽胆囊穿孔部位多发生在底部或颈部。急性胆囊炎炎症可浸润至周围器官，也可穿破肠腔形成内瘘。如胆囊炎反复发作及好转，胆囊壁纤维组织增生，胆囊黏膜消失，形成慢性胆囊炎，甚至胆囊萎缩。

2. 急性非结石性胆囊炎

病理过程与急性结石性胆囊炎相似，但病情发展更为讯速。如病情得不到有效控制，一般在24h内即发展成坏疽性胆囊炎。

二、急性梗阻性化脓性胆管炎

又称急性重症胆管炎（ACST），是急性胆管炎的严重阶段。基本病理改变是胆管急性完全性梗阻和胆管内化脓性感染。梗阻部位可在肝外和(或)肝内胆管。梗阻情况下，经胆汁进入肝内的细菌大部分被单核-吞噬细胞系统所吞噬，10%细菌可逆流入血，成菌血症。另外，胆血反流可致细菌或感染胆汁进入循环，引起全身化脓性感染，大量内毒素可引全身炎症反应、MODS。胆管改变主要以梗阻部位以上胆管扩张、充血、水肿，炎症细胞浸润，形成黏膜坏死、溃疡。进而肝大，晚期肝细胞大片坏死，胆小管可破破裂。致病菌以革兰阴性杆菌为主，其中以大肠埃菌、克雷伯菌常见。25%～30%合并厌氧菌感染。其中感染性休克、胆源性肝脓肿、脓毒败血症及多器官功能衰竭为导致患者死亡的三大主要原因。

三、胆石症

以结石在不同部位形成，所造成的影响不同。多表现为结石梗阻胆囊、胆管造成相应感染炎症及梗阻病变，如急性胆囊炎、胆管炎、胆源性胰腺炎、胆汁性肝硬化等，也能造成不同程度的肝损害以及全身感染。肝胆管长期受结石、肝汁及炎症刺激，甚至可发生癌变。

院前急救判断与处置

院前急救判断

1.电话询问

n 年龄，性别，婚育史？

n 是否为上腹部或右上腹疼痛，呈阵发性？向肩胛部和背部放射？伴恶心、呕吐？

n 起病前是否进食油腻食物或饱餐？

n 是否突然发病？

n 生命体征稳定吗？

2.途中思考

n 胆囊结石和急性梗阻性化脓性胆管炎？

n 是否穿孔？

3.可危及生命的事件

n 神经源性休克

n 胆囊穿孔，败血症，脓毒症休克

4.现场评估

4.1临床症状

①上腹部或右上腹疼痛，呈阵发性，向肩胛部和背部放射，伴恶心、呕吐。

②起病前是否进食油腻食物或饱餐。

4.2查体所见

①右上腹可有不同程度、不同范围的压痛、反跳痛及肌紧张。

②Murphy征阳性，扪及肿大而有触痛的胆囊。

③若胆囊被大网膜粘连包裹，可形成边界不清、固定的压痛包块。

④若胆囊发生坏死、穿孔，可出现肌紧张、全腹压痛、反跳痛等弥漫性腹膜炎表现。

4.3急危重症评估

出现以下症状为危急重症：

全腹刀割样剧痛，血压下降，心率下降，面色苍白等休克表现。

院前急救处置

1．心电监护，严密监测意识状况、血压和呼吸等主要生命体征的变化。

2．建立静脉通道，维持有效血液循环，要时使用血管活性药物。

3．消除胆绞痛，可硝酸甘油片，0.6mg，口服，每4h1次；杜冷丁25～100mg，肌注或皮下注射；阿托品0.5mg，肌注，每4h1次；或加用非那根25mg，临时肌注。

选择医疗机构与转运

1．选择医疗机

急性胆道感染、胆石症，若怀疑有穿孔、急性化脓性胆管炎、急性坏死性胰腺炎等并发症或患者有弥漫性腹膜炎体征时，诊断延迟，转运不及时可能会导致预后不良，无论患者症状轻重，都应该选择转送到能24h对患者进行B超检查，可重症监护、急诊手术处理的医疗机。

2.转运时注意

要及时转送，转送时移动应避免刺激，尽量注意安静状态下转送。在搬运患者时，尽量使用转送担架。对伴有呕吐的患者要注意转送时患者体位的摆放。转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1.半坐卧位，禁食水。

2.保持呼吸道通畅，吸氧。

3.放置胃管持续胃肠减压。

4.留置尿管，记每小时尿量。

5.休克患者放置中心静脉导管，使用血管活性药物维持有效血流灌注。

6.及时请外科会诊协助诊治。

全面诊断与检查

一、急性胆囊炎

（一）急性结石性胆囊炎

1．症状：急性发作时腹痛多位于右上腹，持续性疼痛，阵发性加重，可向右肩背部放射，逐渐发展为右上腹阵发性绞痛，常伴恶心、呕吐等消化道症状，可有发热、恶寒、黄疸等。常因饱餐、进油腻食物后、劳累所诱发，多在在夜间发作。既往有类似发作史。

2．体格检查：右上腹压痛、反跳痛，肌紧张，Murphy征阳性，有时可触及肿大而有触痛的胆囊。若胆囊被大网膜粘连包裹，可形成边界不清、固定的压痛包块；若胆囊发生坏死、穿孔，可出现肌紧张、全腹压痛、反跳痛表现。

3、实验室检查：多数患者（约占85%）血白细胞及中性粒细胞计数增多，核左移，平均在（10～15）×109/L，若白细胞总数在20×109/L以上时，应考虑有胆囊坏死或穿孔存在。老年人不一定升高。当合并有急性胰腺炎时，血，尿淀粉酶含量亦增高。血细菌培养可阳性。40%左右的患者血清转氨酶升高，但多数在400U以下。

4．影像学检查：B超检查特征为：①胆囊的长径宽径可正常或稍大，由于张力增高常呈椭圆形；②胆囊壁增厚，轮廓模糊；有时多数呈双环状，其厚度大于3mm；③胆囊内容物透声性降低，出现雾状散在的回声光点；还可发现结石、蛔虫、大于lcm直径的肝脓肿、膈下脓肿等。其对急性胆囊炎诊断的诊断准确率为85%～90%。

此外，还可行胸、腹X线检查有助于诊断脓胸、肺炎、肺脓肿、心包积脓、膈下脓肿、胸膜炎等，ERBD、PTCD既可确定胆道阻塞的原因部位，又可做应急的减压引流，但有加重胆道感染或使感染淤积的胆汁溢漏进腹腔的危险。　　

（二）急性非结石性胆囊炎

对危重、严重创伤及长期静脉营养支持患者，出现右上腹痛同时伴有发热时要注意本病。体查与结石性胆囊多有相似。但早期B超检查不易诊断，CT检查有帮助，而97%的患者可在肝胆系统核素扫描检查中获得确诊。

二、急性梗阻性化脓性胆管炎的检查

1．症状：腹痛为主要表现，以右上腹或中上腹阵发性疼痛，有时伴肝区剧烈疼痛，常放射至腰背部及肩部，伴有寒战高热，最高体温可大39～41℃，多数患者可有黄疸，但程度不一，可进行性加重。神经系统症状可表现为神志淡漠、嗜睡、甚至昏迷等，合并休克时可有烦躁不安等表现。上述腹痛、寒战高热、黄疸、休克、神经系统症状又称为Reynolds五联征。

2．体格检查：体温持续升高高达39～40℃以上；脉快（120次/分以上）而弱、血压下降、口唇、指甲床发绀、全身湿冷、烦躁不安等休克表现，伴有神志淡漠、嗜睡或昏迷神经系统改变。剑突下或右上腹有压痛，右上腹肌紧张，可有腹膜刺激征。常肝大并有压痛叩击痛。胆囊肿大见于胆总管梗阻者。

3实验室检查：白细胞计数升高，多大于20×109/L，中性粒细胞升高，胞浆内可出现中毒颗粒。血小板计数降低，最低可达(10～20)×109/L，凝血酶原时间延长，提示有DIC倾向，表示预后严重；胆道梗阻肝细胞坏死可引起血清胆红素、血清转氨酶、乳酸脱氢酶等肝功能指标升高，如血清淀粉酶升高，表示伴有胰腺炎。常见肾功能受损、低氧血症、失水、酸中毒、电解质紊乱等，特别是在老年人合并休克者。血细菌培养可呈阳性。

4.影像学检查：B超检查是最常应用的简便快捷、无创伤性辅助诊断方法，可在床旁进行，对诊断很有帮助。B超可显示胆管扩大范围程度以估计梗阻部位，可发现结石蛔虫、大于lcm直径的肝脓肿膈下脓肿等。CT检查不仅能显示肝胆管扩张、结石、肿瘤、肝脏增大或萎缩，还可能显示肝脓肿。若怀疑急性重症胰腺炎，必要做CT检查以评估病情。

三、胆石症

1．症状：胆囊结石的典型临床表现为胆绞痛、上腹隐痛、胆囊积液、黄疸、Mirizzi综合征，胆绞痛的特点多在饱餐、进食油腻食物后或睡眠中体位改变后开始发作，呈右上腹持续性疼痛伴阵发性加剧，可有放射痛。肝外胆管结石一般无症状，但结石梗阻胆管，并发胆管炎时，出现腹痛、寒战高热、黄疸临床表现，称为Charcot三联征。肝内胆管结石一般无症状，并发胆管炎时出现寒战高热、腹痛、有或无黄疸症状。

2．体格检查：一般平时不发作时，可无阳性体征。合并炎症时，可出现不同程度腹膜炎表现，主要为右上腹压痛，胆囊、肝脏或可以触及，可有触痛。肝区叩击痛。

3．实验室检查：但胆石症发作时，白细胞计数及中性粒细胞升高常见，肝功能检查出现异常改变。

4.影像学检查：B超为首选检查项目，临床表现如前述，不再复述。

急诊处理

1.急性胆囊炎的治疗

急性结石性胆囊为最终需术治疗，原则上应争取择期术。急性非结石性胆囊炎一经诊断，应及早术治疗，因为易发生胆囊坏疽穿孔。

1.1非术治疗 也可作为术前的准备。方法包括禁食、解痉、止痛、利胆、输液、营养支持、补充维生素、纠正水电解质及酸碱失衡等。

1.2术治疗 急诊术并非立即术。在实施术前，需要4～8h的快速准备，以控制感染、稳定血压及微循环的灌注，保护重要器官，使患者更好地承受麻醉和术，以免发生顽固性低血压及心搏骤停，更有利于术后恢复。尤其对年老体弱、合并多个重要脏器疾病者，选择术方法更应慎重小心，术方式应以简单、快速、有效为原则。

1.2.1术适应证：①经积极非术治疗，感染不易控制，病情无明显好转鶒，黄疸加深、腹痛加剧，体温在39℃以上，胆囊胀大并有持续压痛；②出现嗜睡、昏迷等精神症状或预示出现脓毒性休克；③有胆囊穿孔、弥漫性腹膜炎、急性化脓性胆管炎、急性坏死性胰腺炎等并发症者。对于年老体弱或有全身重要脏器疾病者因代偿功能差易引起脏器损害，一旦发生，难以逆转，故应放宽适应证，尽早术。

1.2.2术方法：①胆囊切除术（腔镜为首选）；②部分胆囊切除术；③胆囊造口术；④超声导引下经皮经肝胆囊穿刺引流术PTGD。

2.急性梗阻性化脓性胆管炎的治疗

应以快速解除胆道梗阻并有效引流为原则，引流胆管是治疗本病的最重要措施。

2.1非术治疗 既是治疗段，也可作为术前的准备。主要包括：①维持有效的输液道，迅速扩充血容量，可晶体与胶体液联合使用；②全身静脉足量使用抗生素：因急性梗阻性化脓性胆管炎的致病菌主要为革兰阴性杆菌及一些厌氧菌，故应选用针对革兰阴性杆菌及厌氧菌的抗生素；③对症治疗：如降温、足量维生素、纠正水、电解质紊乱和酸碱失衡等；④血管活性药物，如去甲肾上腺素可以提高血压，维持全身血流灌注。⑤经积极非术治疗，感染不易控制，病情无明显好转，黄疸加深、腹痛加剧体温在39℃以上胆囊胀大并有持续压痛；应紧急行胆道引流治疗。

2.2胆总管减压引流 是最直接而有效的术式，可以清除结石和蛔虫，主要为抢救患者生命、简单、快捷有效。减压术包括：①胆总管切开取石、T管引流术；②ENBD；③PTCD。

2.3后续治疗 急诊胆管减压引流仅为对处理措施，病因一般不可能完全去除，考虑到过多的操作和过于复杂的术，如术中胆道造影胆囊切除、胆肠内引流术等，对患者创伤大术时间延长，反而可加重病情；对于复杂的胆道病变，难以在急症情况下解决者可留做二期术处理。一般在患者术后1～3个月后根据病因选择彻底的术治疗。强调应根据患者具体情况采用个体化的术方法。

3.胆囊结石的治疗

3.1对于有症状和（或）并发症的胆囊结石，首选腹腔镜胆囊切除治疗。

3.2无症状的胆囊结石一般不需要预防性术治疗，但是长期观察表明，约30%以上的患者会出现症状及合并症而需要术。

3.3需要考虑术治疗的情况：①结石数量多及结石直径≥2～3mm；②胆囊壁钙化或瓷性胆囊；③伴胆囊息肉＞1cm；④胆囊壁增厚（＞3mm）即伴有慢性胆囊炎；⑤儿童胆囊结石：无症状者，原则不术。

4.肝外胆管结石的治疗

胆外胆管结石仍以术治疗为主。术中应尽量取尽结石、解除胆道梗阻、术后保持胆汁引流畅。

4.1非术治疗 也可作为术前的准备。争取在胆道感染控制后才行择期术。

4.1.1应用抗生素，应根据敏感细菌选择用药，经验治疗可选用胆汗浓度高、主要针对革兰阴性细菌的抗生素。

4.1.2解痉。

4.1.3利胆，包括一些中药及中成药。

4.1.4纠正水、电解质及酸碱平衡紊乱。

4.1.5加强营养及补充维生素，禁食患者应使用肠外营养。

4.1.6护肝及纠正凝血功能异常。

4.2术治疗

4.2.1胆总管切开取石、T管引流术：可采用开腹或腹腔镜术。适用于单纯胆总管结石。若胆管上、下端畅，无狭窄等其他病变者伴有胆囊结石者或当胆管问题解决后胆囊可恢复其形态及正常功能，可同时行胆囊切除术。但不可随意切除严重急性胆囊炎症如坏疽、穿孔或合并明显慢性病变的胆囊。术中应尽量取尽结石，如条件不允许，也可以在胆总管内留置橡胶T管（不提倡应用硅胶管），但必须置于狭窄以上的胆管内，忌遗漏肝内病灶。术后行造影或胆道镜检查、取石。

4.2.2胆肠内引流术：此术应慎重，由于胆肠内引流术废弃了Oddi括约肌的功能，有多种缺点，目前使用已经逐渐减少。

5.肝内胆管结石的治疗

5.1无症状的肝胆管结石可不治疗，仅定期观察、随访即可。

5.2术治疗：临床症状反复发作者应术治疗。术方式主要根据患者的具体情况而定，应力求简单、快捷有效，其基本原则是以抢救生命为主，关键是行胆道减压，解除梗阻，畅胆流，达到充分减压和引流的目的即可。对于复杂的胆道病变，难以在急症情况下解决者可留做二期术处理。

5.3术方法：①胆管切开取石术：是最直接而有效的术式，可以清除结石和蛔虫，但必须探查肝内胆管有无梗阻，沿胆总管身上切开甚至可达2级胆管，尽力去除阻塞因素,应争取切开狭窄的部位，直视下或过术中胆道镜取出结石，直至取净。②胆肠内引流术不能作为替代对胆管狭窄、结石病灶的处理方法。当Oddi括约肌仍有功能时，应尽量避免胆肠吻合术。③肝切除术：切除病变部分的肝，包括结石和感染病灶、不能切开的狭窄胆管，去除了结石的再发源地，并防止病变肝段、肝叶癌变。

疗效标准

1．治愈：症状体征消失。腹部B超恢复正常，各项并发症均已治愈。

2．好：病情明显好，症状体征尚未完全恢复正常，或腹部B超处在好中，各项并发症尚未完全治愈。

3．未愈：未达到上述标准者。

出院标准

达到临床治愈者可出院，好或未愈者需家属签字要求出院者方可院或出院。
急诊建议
1．患者腹痛原因未明确时，应慎用止痛药，以防止掩盖病情，要详细向患者或家属说明，争取合作。

2．问患者有无胆道疾病病史，尽早做出诊断。

3．首选B超检查，尽早明确病情，必要时行急诊手术。

4．存在腹膜炎时，如短时间不能诊断，可行剖腹探查术。
