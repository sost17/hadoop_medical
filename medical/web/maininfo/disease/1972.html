蜡样芽孢杆菌食物中毒
蜡样芽孢杆菌食物中毒是由进食含有蜡样芽孢杆菌所产生的肠毒素所致的急性胃肠炎。污染的食物主要为含淀粉较多的各类食物。蜡样芽孢杆菌为革兰阳性大肠杆菌，能形成芽孢，无荚膜，有鞭毛，属需氧菌，最适生长温度为28~35°C，10°C以下不能繁殖，pH值为4.9~9.3时可以繁殖。该菌繁殖体不耐热，食物中毒菌株的芽孢能耐受100°C 30分钟，而干热120°C经60分钟才能被杀死。蜡样芽孢杆菌广泛分布于土壤、水、垃圾等中。该菌能产生致吐肠毒素和致泻肠毒素，前者耐热。

发病机制

病因

蜡样芽孢杆菌中毒是由摄入带有大量的蜡样芽孢杆菌及该菌产生的肠毒素所引起的。该菌的致病菌株所产生的耐热毒素能引起呕吐型胃肠炎。不耐热肠毒素能产生腹泻型胃肠炎。中毒季节以夏、秋季为多。中毒食品主要为剩米饭、米粉、剩菜、甜点心及乳、肉类制品。中毒多因食用保存环境温度较高（20°C以上）或放置时间较长而被蜡样芽孢杆菌污染的食品。

诊断

病史要点

蜡样芽孢杆菌潜伏长短不一，一般为0.5～16小时。

症状要点

由于感染菌株所产生的毒素不同，临床上主要可分为：

1. 呕吐型 潜伏期较短，仅0.5~5小时。起病急骤，以恶心、呕吐、腹痛、上腹不适为主，并有头晕、四肢无力，少数患亦可有腹泻，无明显发热。

2. 腹泻型 潜伏期较长，约8~16小时。症状以腹痛、腹泻为主，粪便呈水样，有里急后重及恶心，呕吐等。

辅助检查

实验室检查

样品的检测应在BSL-2级生物安全实验室中进行。

1. 快速检测方法

（1）直接涂片染色镜检：将残余食物或患者排泄物直接涂片染色镜检。如见到大量革兰阳性芽孢杆菌，芽孢呈椭圆形，并位于菌体中央或偏端，不膨大于菌体，菌体两端较平整，多呈短链或长链状排列者，高度怀疑蜡样芽孢杆菌食物中毒。

（2）直接划线分离培养：将残余食物或患者呕吐物直接划线于琼脂平板上，于（30±1）°C培养24小时，可见到典型蜡样芽孢杆菌菌落。

2. 常规检测方法 参照《食品卫生微生物学检验——蜡样芽孢杆菌检验（GB/T4789.14）》进行定性和定量检测。

3. 毒素检测方法：

（1）腹泻毒素：可采用家兔结扎实验、血管通透性反应、毒素致死实验、免疫反应测定等方法。

（2）呕吐毒素：将含有0.95%氯化钠的米饭，经高压灭菌后接种待检菌株，（30±1）°C培养18~20小时，经淀粉酶水解，聚乙二醇透析浓缩后，用导管注入猴子或小猫的胃内，如含有呕吐毒素即发生呕吐。

4. 流行病学分型技术

（1）生化分型：根据柠檬酸盐利用、硝酸盐还原、淀粉酶水解、V-P实验反应、明胶液化试验，将蜡样芽孢杆菌分成32个生化型别。

（2）血清学分型：利用蜡样芽孢杆菌H抗原进行分型，国外已有23个分型的诊断血清，引起腹泻综合征的菌株主要为2、6、8、9、10和12血清型；引起呕吐综合征的菌株主要为1、3、4和5血清型。

临床类型

临床主要分为吐型及腹泻型。

鉴别诊断

1. 急性细菌性痢疾 急性起病，发热、腹痛、腹泻、黏液脓血便、里急后重，大便次数达每日十余次，常伴有恶心、呕吐、乏力等全身症状，粪便检查可见大量脓细胞，其中有红细胞及巨噬细胞，培养可分离出痢疾杆菌。

2. 霍乱 夏季流行，有切接触史或集体发病，临床表现为先泻后吐，无明显全身中毒症状，无明显腹痛、里急后重等症状，大便次数不多但量大，初为黄色水样便，后为典型的“米泔水”样便，并迅速出现脱水、电解质紊乱、酸中毒和循环衰竭。粪便镜检常无红、白细胞，外周血象呈血液浓缩表现。早期对呕吐物及排泄物悬滴标本进行动力试验和制动试验可实现早期快速诊断。

3. 非细菌性食物中毒：包括化学性食物中毒和有毒动植物食物中毒。主要根据病史鉴别，患者多有进食被化学毒物（有机磷农药、砷、汞、亚硝酸盐等）污染食物史，或进食毒蕈、河豚、发芽马铃薯、苦杏仁等食物病史。潜伏期短（数分钟至数小时），临床表现除急性胃肠炎症状外，可有头晕、头痛、乏力、发绀、肢端麻木、惊厥等神经系统症状及肾功能损害等表现。可疑食物、呕吐物与粪便标本检出毒物可确诊。

治疗

治疗原则

本病一般为自限性，仅需终止进食疑污染食物，并行对症治疗，如口服或静脉补液，应用解剂等。对重症病例考虑应用抗生素治疗。

一般治疗

有失水或酸中毒的患者应根据病情进行补液。轻、中度脱水可采用服补液，重度脱水者给予静脉补液，血压下降时，可视病情应用血管活性药。

药物治疗

一般无须使用抗生素治疗。对病情较重者可考虑使用氯霉素，静脉滴注，成人每日2~3g，分2次给予。也可用红霉素、庆大霉素等。

预防
因蜡样芽孢杆菌分布广泛，因而希望杜绝食物污染是困难的，应以正确处理食物，控制细菌繁殖及产生毒素为主要措施。

1. 不进食腐败变质和保存不当的食物。

2. 食物应充分加热，对蜡样芽孢杆菌污染的食物应加热30分钟以上，食物加热后应立即进食，室温保留时间不超过1小时。

3. 做好熟食后如不立即食用，应尽快冷藏或冷冻，低温保存，食用前再加热。

4. 严格剩饭菜的处理和凉拌菜的卫生要求。

疑难病例
患者，男性，43岁。主诉：恶心、呕吐3小时。患者于3小时前进食炒米饭后出现恶心、呕吐，伴头晕、腹痛、腹泻。呕吐物为胃内容物，腹痛以脐周隐痛为主，腹泻多为水样便，无发热。就诊于齐鲁医院，详细询问病史后，将呕吐物、排泄物直接涂片，革兰染色镜检，见大量呈链状革兰阳性芽孢杆菌。普通琼脂平板上，将患者呕吐物连续划线法接种，经31℃ 24小时培养，优势菌落为圆形、隆起、灰白色、不透明、蜡滴样大肠杆菌落，无荚膜，氧化酶试验阳性，V-P试验阳性，分解葡萄糖、蔗糖产酸不产气。将该菌落涂片，革兰染色镜检，为有芽孢无荚膜的革兰阳性大肠杆菌。初步诊断为蜡样芽孢杆菌食物中毒。给予静脉补充水电解质，静滴氯霉素抗菌及对症支持治疗后病情好转，住院3天后痊愈出院。
