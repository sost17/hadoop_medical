肺炎链球菌性肺炎
链球菌(streptococcus)是化脓性球菌中的一大类常见细菌，为链状或者成双排列的革兰阳性球菌。广泛分布于自然界和人类的上呼吸道、胃肠道、泌尿道及生殖道中。致病者可发生各种化脓性炎症以及链球菌感染后的变态反应性疾病，如猩红热、丹毒、风湿热、肾小球肾炎等。链球菌中的致病菌主要为A族，亦称化脓性链球菌(pyogenic streptococcus)，可引起人类肺炎。肺炎链球菌性肺炎散发于世界各地，一般少有真正的流行，但1918年流感大流行后继发的链球菌肺炎呈小流行，病死率高达30%～35%。多为内源性感染，传染源为带菌者本身，当机体抵抗力降低时，吸入上呼吸道含菌的分泌物而发病。儿童和老年体弱者易患本病。

发病机制

病因

肺炎链菌为链菌科，链菌属细菌，革兰氏染阳性。它不产生内外毒素，其致病性主要是荚膜的侵袭作用。

病理

病理改变有充血、红肝变、灰肝变及消散。病理以肺泡炎为主，很少涉及肺泡壁或支气管壁的间质。
诊断

病史要点

急性起病，发病前常有淋雨受凉、过度疲劳、上呼吸道病毒感染、醉酒等。多有上呼吸道感染的前驱症状。

吸烟者、痴呆者、慢性支气管炎、支气管扩张、充血性心力衰竭、慢性病患者以及免疫抑制者均易受感染。

症状要点

典型临床表现：高热，寒战，咳嗽，咳痰，呼吸急促，胸痛，全身肌肉酸痛。

1. 咳嗽开始为干咳，之后出现脓痰，典型为铁锈色，有时痰中带血丝小血斑。

2. 高热、寒战，体温在数小时内升至 39~40℃，高峰在下午傍晚，呈稽留热，脉率随之增速。

3.气促与病变范围较广、高热及基础肺功能减退相关。

4. 胸痛常见，在深呼吸咳嗽时加重，下叶肺炎刺激膈肌胸膜，疼痛放射至肩背部下腹部。

5. 胃纳锐减，偶有恶心、呕吐、腹痛，易被误诊为急腹症。

查体要点

1. 病人高热病容，气急、鼻翼扇动、发绀、口角可有疱疹。

2. 根据病变的范围，可有典型肺部实变体征。早期肺部体征无明显异常，仅有胸廓呼吸运动幅度减小，叩诊稍浊，听诊可有呼吸音减低、胸膜摩擦音、干湿啰音较广泛。肺实变时叩诊浊音，触觉语颤增强并可闻及支气管呼吸音。消散期可闻及湿啰音。合并胸腔积液时可出现相应的体征。

3. 有脓毒症者，可出现皮肤、结膜出血点，巩膜黄染。心率增快，有时心律不齐。

4. 重症患者有肠胀气，上腹部压痛多与炎症累及膈胸膜有关。

5. 重症感染时可伴休克、急性呼吸窘迫综合征及神经精神症状。

辅助检查

实验室检查

1. 白细胞总数及中性粒细胞百分比增，可见核左移。

2. C-反应蛋白升高，ESR快。

3. 抗链球菌素O的效价明显升高。

4. 痰涂片发现典型的链状排列的革兰阳性球菌可初步诊断。

5. 痰培养鉴定出化脓性链球菌。

6. 聚合酶链反应(PCR) 及荧光标记抗体检测可提高病原学诊断率。

7. 尿肺炎链球菌抗原可阳性。约 10% ～20% 患者合并菌血症，故重症肺炎应做血培养。如合并胸腔积液，应积极抽取积液进行细菌培养。

影像学检查

1. X线表现 分布于两下肺中内，带沿支气管树分布的边缘模糊的密度增高阴影，呈叶段实。病灶可融合为大片状，但密度常不均匀，细支气管阻塞可导致局限性肺不张/肺气肿，可合并脓胸。

2. CT诊断 早期，病显示为肺叶内磨玻璃影或稍高密度影，病内部密度不均，边缘模糊；病发展，CT表现为肺叶内全部或大部分实，病灶密度可均匀或不均匀，部分病灶内可见含气支气管气象，增强后病灶可见结构完整的肺血管影；治疗后，由于炎症吸收，病范围较实缩小，密度降低，病灶内密度更不均匀，形成大小不等的斑片状病灶。绝大部分病例可完全吸收，少数病例吸收缓慢，甚至形成慢性炎症。

3.MRI表现 T1WI与渗出物相关，以浆液渗出或细胞性渗出为主时呈中低信号，以蛋白性渗出为主或发生出血时为偏高信号。T2WI为较高信号。

其他检查

1. 经纤维支气管镜或人工气道吸引 受口咽部细菌污染的机会较咳痰为少，如吸引物细菌培养其浓度≥105cfu/ml可认为是致病菌，低于此浓度者则多为污染菌。

2. 防污染样本毛刷（protected specimen brush,PSB）如所取标本培养细菌浓度≥103cfu/ml，可认为是致病菌。

3. 支气管肺泡灌洗（bronchial alveolar lavage,BAL） 如灌洗液培养细菌浓度≥104cfu/ml，防污染BAL标本细菌浓度≥103cfu/ml，可认为是致病菌。

4. 经皮细针吸检（percutaneous fine-needleaspiration,PFNA）和开胸肺活检 两种方法所取标本检测的敏感性和特异性很好，但由于是创伤性检查，容易引起并发症，如气胸、出血等，临床一般用于对抗菌药物经验性治疗无效或其他检查不能确定者。

鉴别诊断

1. 葡萄球菌肺炎 感染中毒症状重，咳黄色粘稠痰或脓血痰，X线呈现多发性大小不等斑片状阴影，可有空洞形成，可有多变肺气囊肿形成。

2.干酪性肺炎 患者常有低热、乏力，痰中易找到结核杆菌。X线显示病变多在肺上部，呈大片浓密阴影，历久不消失，可形成空洞和肺内播散。而肺炎球菌肺炎经历抗菌治疗后肺内炎症吸收较快。

3. 急性肺脓肿 随着病程的发，大量脓臭痰为肺脓肿的特征，X线显示大片浓密浸润阴影，合并脓腔和液平形成。

4. 肺癌 可伴有阻塞性肺炎，一般不发热或仅有低热，周围血白细胞计数不高，抗菌治疗后炎症吸收缓慢或炎症吸收后出现肿块阴影，痰中找到癌细胞可以确诊。

治疗

治疗原则

综合治疗：抗感染治疗、呼吸治疗（吸氧和MV）、持治疗以及痰液引流等，以抗感染治疗最重要。

一般治疗

1. 应卧床休息，监测患者生命体征，注意能量的摄入。鼓励饮水每日1～2L。

2. 有明显痛，可适当给予少量止痛剂。烦躁不安、谵妄、失眠者可服安定5mg或水合氯醛1～1.5g，不可使用抑制呼吸的镇静剂。

3. 对于重症患者有呼吸困难等应及时给予氧疗。高热脱水者给予静脉补液，纠正电解质紊乱。

药物治疗

抗菌治疗：青霉素G为首选。对于成年轻症患者，80万单位每8小时肌注1次。稍重，则宜240万~480万单位青霉素G静脉注，每6小时1次。重症及并发胸膜炎时，可加至每日1000万~3000万单位，均分4次静脉注；注时每次量尽可能在1小时内完，以便产生有效血浓度。

如患者对青霉素过敏，轻症可用红霉素每日1~2g口服，或每日1.5g静脉注；亦可用林可霉素每日2g口服或静脉注。重症患者应使用头孢唑啉每日2~6g，分次静脉注或肌注。左氧氟沙星，每日200mg，分2次口服或静脉注；环丙沙星0.25g或0.5g，日服2次，或0.2g每日2次静脉注。

抗菌药物疗程一般为5~7天，或退热后3天停药。

常见并发症
1. 用适当抗菌药物后，体温再升或3天后仍不退者，应考虑肺炎球菌的肺外感染及混杂细菌感染，药物热或存在其他并存的疾病。

2. 约10%~20%肺炎球菌性肺炎伴发胸腔积液，应进行胸部X线检查排查。

3. 慢性包裹性脓胸应考虑外科肋间切开水封瓶闭式引流。可有5％并发脓胸，须局部加用青霉素并积极排脓。

4. 严重服毒症或毒血症患者易发生感染性休克，尤其是老年人。表现为血压降低、四肢厥冷、多汗、心动过速、心律失常等，而高热、胸痛、咳嗽等症状并不突出。

预后
肺炎球菌肺炎预后良好。但有以下因素存在时，预后差：年老、原先患有慢性心、肺、肝肾疾病者，体温和白细胞计数不高者以及免疫缺陷者；病变广泛、多叶受累者、并发症严重，如有周围循环衰竭者。
随访
复查血常、胸片，检测肺阴影吸收情况。
预防
1.积极治疗病人及带菌者，可以减少传染源。对急性咽喉炎、扁桃体炎等上呼吸道感染等彻底治疗。

2.意空气消毒。避免淋雨受寒、疲劳、醉酒等诱发因素。

3. 对于老弱体衰和免疫功能减退者，如糖尿病、慢性肺病、慢性肝病、脾切除者，可射肺炎免疫疫苗。
