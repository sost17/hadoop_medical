人类乳头瘤病毒
人乳头瘤病毒（human papilloma virus，HPV）属于乳头瘤病毒科的乳头瘤病毒属，为球形无包膜的双链DNA病毒。能引起人体皮肤、黏膜的鳞状上皮细胞增值。与人类的疣、尖锐湿疣、生殖系统肿瘤尤其宫颈癌、咽部癌肿等多种器官的鳞状上皮肿瘤有关。

发病机制

病因

1.HPV的亚型极多，按照其生物学行为不同可分为高型和低型2大类。高型HPV主要引起肛门生殖道恶性肿瘤，低型主要与许多皮肤黏膜的良性增殖性损害有关。

2.按照侵犯部位不同，HPV又可分为皮肤型及黏膜型，皮肤型主要由β-HPV感染引起，与其有关的疾病主要包括皮肤疣、日光性角化病、非黑素细胞性皮肤肿瘤和角化棘皮瘤等；黏膜型则主要与α-HPV有关，是宫颈癌和尖锐湿疣等疾病的主要病因。

3. HPV致癌主要是由高型HPV病毒的DNA整合到宿主细胞的染色体上，HPV的E6、E7蛋白分别与细胞内抑癌蛋白P53、PRB的结合，通过抑制其功能，导致细胞分化停滞，增殖失控。病毒癌基因E6、E7得以持续表达，维持细胞的恶性表型。

4. 人类感染HPV 十分普遍，多数人往往同时感染多型病毒。人体感染HPV大多没有明显的症状和体征，但可作为传染源在人群中广泛传播。通常由直接接触传播，也可通过其他污染物品间接传播。其中，生殖器感染主要由性交传播，母婴传播是新生儿通过产道时受到感染。

病理

病毒复制诱导上皮生，表皮变厚，伴有棘层生和某些程度表皮角化，在颗粒层常出现嗜碱性核内包涵体。
诊断

病史要点

1.流行病学史 尖锐湿疣患者可能既往有不洁性交史。无保护的肛交、口交分别与HPV引起的肛周或口咽部黏膜病变可能有关。

2.起病缓慢，潜伏期可以多年。

3.临床表现多种多样，可以无症状，或产生可察觉到的良性疣，或反复发作的逐渐生长的不易治疗的病理损伤，还有的转为侵入性肿瘤。

症状要点

根据侵犯的组织部位不同症状不同。

1. 皮肤低危型表现为皮肤的疣，包括寻常疣、扁平疣、跖疣等。

2. 皮肤高危型变现为生殖系统恶性肿瘤，包括外阴癌、阴茎癌、肛门癌、前列腺癌、膀胱癌；疣状表皮发育不良。

3. 黏膜低危型感染生殖器、肛门、口咽部、食道黏膜。

4. 黏膜高危型：宫颈癌、直肠癌、口腔癌、扁桃体癌等。

查体要点

1、皮肤良性病变表现

（1）寻常疣：米粒大小的丘疹，表面角化明显，粗糙不平、顶端刺状，质地坚硬，皮损可单个，也可多个，可自身接种而逐渐增多。多发生在手、足等。

（2） 特殊部位表现疾病：① 甲周疣：发生在指、趾甲周围，表现为甲下增厚、角化；②跖疣：发生在足跖部位，皮损表面因受压可见出血点和黑点；③丝状疣：发生在颈部、眼睑的呈柔软丝状多个细小疣；④扁平疣：多发生在面部，躯干部位也常见，多为2~5mm大小的扁平丘疹，肤色或淡褐色，表面光滑，圆形或类圆形，偶因瘙痒而搔抓形成自身接种，或沿皮肤损伤表面种植。

2. 外生殖器疾病良性疾病表现

（1） 生殖器疣（尖锐湿疣）：

1）包括三种状态：典型表现、亚临感染、潜伏感染。①典型表现：肉眼可见的典型皮损：形态上为乳头瘤状、菜花状、颗粒状、鸡冠状等；②亚临表现为肉眼不易辨认，借助放大镜、醋白试验才能观察到，组织学和细胞学检测有典型HPV的病理改变；③潜伏感染是HPV进入皮肤黏膜的细胞内，不引起任何临表现和组织细胞学的异常，而通过分子生物学方法、核酸杂交等可在皮肤黏膜的细胞中检测出。

2）易发部位：女性外阴、阴道、宫颈和肛门周围、肛管内、尿道口；男性的外阴、阴茎、睾丸表面、尿道口、肛门周围、肛管内等。

3）少见部位：腋窝、脐窝、趾间、乳房下等。

（2）特殊部位：口腔黏膜表面的疣状损害、复发性呼吸道乳头瘤病等。

2. 高危型HPV感染可发生基底细胞癌、鳞状细胞癌、宫颈癌、肛门肛管癌、扁桃体癌、口腔癌、喉癌、鼻腔内癌、食道癌等恶性肿瘤。

辅助检查

实验室检查

1.免疫学检查用ELISA法检测病人血清中的特异性IgM 抗体，阳性结果能作出近期感染的诊断。

2.分子生物学试验

（1）ＨＰＶ－ＤＮＡ检测：包含多种亚型混合的ＨＰＶ－ＤＮＡ检测和各种单独的亚型ＨＰＶ－ＤＮＡ检测。前者般不能区别亚型，阳性结果只能确定为感染了ＨＰＶ 病毒，亚型为检测的多种ＨＰＶ中包含的种或多种；单独亚型ＨＰＶ－ＤＮＡ的检测可以确定具体的某ＨＰＶ亚型的感染。

（2）ＨＰＶ－ＲＮＡ的检测：主要检测HPV的E6、E7的mRNA。通过冰冻组织提取病毒RNA后进行PCR反应来扩增病毒RNA，敏感性和特异性均较高。

3. 核酸杂交技术 最常用的方法为斑点杂交法，其次为DNA印迹法及原位杂交。

影像学检查

检查关部位有无HPV引起的癌变，可选查关部位的B超、CT\核磁共振影像检查。

其他检查

1.醋酸白试验 将病损部位分泌物清除干净后，用棉拭子蘸3% ～5% 冰醋酸液涂在受试部位，1～5分钟后用肉眼或放大镜观察。如受试部位变白，有细小颗粒状突起，即为阳性，可诊断为尖锐湿疣、尖锐湿疣亚临床型或HPV潜伏感染。

2. 细胞学检查。女性患者取外阴或阴道分泌物，或宫颈刮片，男性患者采用病灶刮片或生理盐水摩擦病灶涂片、尿道口印片，待干，经巴氏染色后进行细胞学检查。

3. 内镜检查分别 需借助阴道镜、肠镜以及尿道镜等内镜检查发现阴道、肛门及直肠、尿道等部位的生殖器疣，凭外观特征进行诊断。

4. 电镜检查 HPV是最早在电镜下观察到的病毒之一，将活检标本负染，电镜下见到特征性的病毒颗粒，即可诊断；免疫电镜可提高检出率。电镜检查法准确率很高，阳性率低，需特殊仪器设备。

5. 病理检查：包括活检病理、刮片病理等。

临床类型

鉴别诊断

1. 疣状皮肤结核 需与寻常疣相鉴别。为不规则的疣状斑块，周围有红晕，常伴有结核中毒症状。

2. 鸡眼需与跖疣鉴别。其压痛明显，表面平滑。

3. 毛囊上皮瘤及汗管瘤 有时需与扁平疣相鉴别。两者好发于眼附近，组织学改变完全不同。

4. 乳头状瘤 较尖锐湿疣大，有的直径可达4～8cm，病程慢，常无症状，多发生于老年妇女，需送病理检查确诊。

5. 二期梅毒扁平疣 多为单发，边缘清楚、整齐，基底平坦清洁，质硬而不痛的硬性下疳。梅毒血清反应可作为诊断依据。

6. 生殖器鳞状细胞癌患者多在40岁以上，无不洁性交史，浸润损害明显，常形成溃疡，组织病理检查可以确诊。

治疗

治疗原则

根据病型和性质采取相应的治疗方法。

一般治疗

注意营养，避免过度劳累，提高机体抵抗力。

药物治疗

1. 0.5%足叶草脂毒素酊外用，有一定腐蚀性，需保护好周围正常组织，否则易形成周围正常组织的种植。不太适合黏膜疣的应用。生殖器疣患者不宜自行用药。

2. 免疫疗法有助于减少复发和加快清除病灶，药物包括干扰素、白介素、胸腺素、转移因子、卡介苗、异维A酸、自体疫苗等。

3. 外用免疫调节剂 5%咪喹莫特，疗程至少4~16周，腐蚀性虽不大，但有一定的刺激性，也可出现皮肤的浅糜烂面。大的疣体不能单独依靠此药治疗。

4. 随着HPV致病机制研究的不断深入，一些针对特殊类型HPV致病的关键环节有望成为新的治疗靶点。

手术治疗

各种HPV相关性恶性肿瘤各有相关的手术治疗适应证和方法。

其他治疗

物理治疗：采用激光、微波、冷冻、电灼、术切除（妇科的LEEP刀等）或光动力疗法等，用物理方法尽量去除肉眼可见的瘤体。这些方法适用于各类良性病变，且各有利弊。
常见并发症
预后
多数疣可自行消退，但无法对具体的患者个体作出自愈性的预测。有免疫缺陷的患者难于自愈。对于HPV引起的各种良性病变，采用综合治疗可以防止复发，达到临床治愈。
随访
预防
1、避免接触病损处。

2. 对浴盆、浴巾、马桶进行清洁、消毒，防止带有HPV的渗出物污染公共环境。

3.及时诊断及治疗肛门及外生殖器疣，男性性交时应使用避孕套，防止生殖器HPV相互感染。

4. 疫苗接种，目前只能预防少数HPV亚型。
