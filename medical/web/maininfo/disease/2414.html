急性肺栓塞/深静脉血栓
急性肺栓塞（acute pulmonary embolism，APE）是由内源性或外源性栓子堵塞肺动脉主干或分支引起肺循环障碍的临床和病理生理综合征,常见的栓子是血栓，其余为少见的新生物细胞、脂肪滴、气泡、静脉输入的药物颗粒甚至导管头端引起的肺血管阻断。由于肺组织受支气管动脉和肺动脉双重血供，而且肺组织和肺泡间也可直接进行气体交换，所以大多数肺栓塞不一定引起肺梗塞，其发病率仅次于冠心病及高血压；死亡率居第三位,仅次于肿瘤及心肌梗死。 深静脉血栓（deep venous thrombosis，DVT）实质上是和肺血栓栓塞症为同一疾病在不同阶段、不同部位的表现。主要指血液非正常地在深静脉内凝结，属于下肢静脉回流障碍性疾病。

发病机制

病因

绝大数APE或者DVT患者都有致病的诱因,如下肢或盆腔静脉血栓形成,长期卧床或不活动，或者慢性心肺疾病、手术、创伤、恶性肿瘤、妊娠及口服避孕药等，一般来说，在静脉血栓形成中内皮损伤起着重要的初始和持续作用。静脉内皮损伤可因机械性创伤，长期缺氧及免疫复合物沉着等引起，使胶原组织暴露，刺激血小板附着和集聚，激活血凝反应链。血液停滞能激活凝血机制，触发血栓形成。血液的高凝状态也是血栓形成的重要机制之一。

病理

大多数急性肺栓塞可累及多支肺动脉，栓塞的部位为右肺多左肺，下叶多上叶，但少见栓塞在右或左肺动脉主干或骑跨在肺动脉分叉处。血栓栓子机化差时，通过心脏途径中易形成碎片栓塞在小血管。若纤溶机制不能完全溶解血栓，24小时后栓子表面即逐渐为内皮样细胞被覆，2～3周后牢固贴动脉，血管重建。早期栓子退缩，血流再通的冲刷作用，覆盖栓子表面的纤维素、血小板凝集物及溶栓过程，都可以产生新栓子进一步栓塞小的血管分支。栓子是否引起肺梗塞由受累血管大小，阻塞范围，支气管动脉供给血流的能力及阻塞区通气适当与否决定。肺梗塞的组织学特征为肺泡内出血和肺泡坏死，但很少发现炎症，原来没有肺部感染或栓子为非感染性时，极少产生空洞。梗塞区肺表面活性物质丧失可导致肺不张，胸膜表面常见渗出，1/3为血性。若能存活，梗塞区最后形成瘢痕。

肺栓塞后引起生理死腔增加，通气效率降低，但由急性肺栓塞可刺激通气，增加呼吸频率和每分钟通气量，从而抵消了生理死腔的增加，保持PaCO2不升高甚至降低。肺泡过度通气与低氧血症无关，甚至不能由吸氧消除。其机理尚不清楚，推测与血管栓塞区域肺实质的反射有关。虽然PaCO2通常降低，但神经肌肉疾患，胸膜剧烈疼痛和肺栓塞严重患者不能相应增加通气代偿增加的生理死腔时，可出现CO2潴留。急性肺栓塞时常见PaO2降低，通气/血流比值失调可能是其主要机制，局部支气管收缩，肺不张和肺水肿为其解剖基础。如果心输出量不能与代谢需要保持一致，混合静脉血氧分压将降低，可进一步加重通气/血流比值失调和低氧血症。

院前急救判断与处置

院前急救判断

1.电话询问

n 是然发病？

n 起病前有无长期卧床或者然用力排便等诱因？

n 有无发呼吸困难、严重胸痛、咯血或者晕厥等表现？

n 生命体征是稳定？

2.途中思考

n 原发性还是继发性

n 属于肺栓塞的那种类型

n 是存在心肌梗塞的可能

3.可危及生命的事件

n 有无气道阻塞引起的窒息

n 有无大量体表出血，甚至休克可能

4.现场评估

4.1临床症状

①有无发呼吸困难

②有无气道阻塞

③有无体表可见大量出血．

④有无脉搏，循环功能是良好

4.2查体所见

①是有气道阻塞

②是有呼吸困难

③是存在呼之无反应，大动脉搏动消失，无心跳等表现

4.3急危重症评估

出现以下症状为危急重症：

4.3.1患者发病急，短时间内迅速出现意识障碍。

4.3.2患者呼吸不规则，或者无呼吸。

4.3.3大动脉搏动消失。

院前急救处置

1．取平卧头侧位，利于口粘液及呕吐物流出。

2．保持呼吸道通畅，及时清理呼吸道分泌物，防止误吸。如出现舌后坠影响呼吸道通畅者，应放置口咽通气管，必要时气管插管。

3．给予吸氧，保持供氧充分。

4．严密监测意识状况、瞳孔、血压和呼吸等主要生命体征的变化。

5．维持有效血液循环（静脉补液避免脱水，主张入量略大于出量）；

选择医疗机构与转运

1.选择医疗构

考虑患者出现急性肺栓塞的情况，应采取就近原则，选择能够有条件处理该病的医疗构迅速转运。

2.转运时注意

要及时转送，转送时移动应避免刺激，尽注意安静状态下转送。在搬运患者时，尽保证身体的平稳性，同时在转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1.绝对卧床休息，选择侧卧位，患侧向下避免误吸以及窒息可能。

2.保持呼吸道通畅，吸氧，昏迷患者应将头歪向一侧，以利于口腔、气道分泌物及呕吐物流出，并可防止舌根后坠阻塞呼吸道，必要时行气管插管或行气管切开术。

3.高流量吸氧，保证血氧饱和度在95%以上。

4.建立静脉通道

5.密切监测患者生命体征

6.必要时机械通气

全面诊断与检查

1.症状

1.1起病形式：起病突然（数秒或数分钟内发生），多数患者发病前有明显诱因（剧烈运动、过度疲劳、用力排便、情绪激动等）。

1.2主要症状：不明原因的呼吸困难及气促，尤其以活动后明显，伴有突发剧烈胸痛，持续不能缓解或进行性加重；可伴有恶心、呕吐等消化道症状；有时可出现晕厥的表现，常伴有咯血和咳嗽等症状。

2.体格检查

2.1呼吸急促最常见，有时可伴有发绀，肺部可出现哮鸣音或者细湿啰音，有时可闻及血管杂音。

2.2严重时刻出现血压下降甚至休克，颈静脉充盈或者异常的搏动，肺动脉区的第二心音亢进或者分裂，三尖瓣区收缩期杂音。

3.实验室检查和其他检查

3.1胸片可显示：1）肺动脉阻塞征 局部肺纹理变细或消失，肺部透亮区变大。2）肺动脉高压及右心扩大征 出现肺动脉膨隆及右心室扩大。3）肺组织的其他改变 可出现肺野的局部片状阴影，肺不张等表现，可伴有胸腔积液。

3.2血气分析可协助诊断，多表现为低氧血症或低碳酸血症，肺泡-动脉血氧分压差增大。

3.3心电图最常变现为窦性心动过速，伴有完全或者不完全性右束支传导阻滞。

3.4超声心动图在提示或者排除其他心血管疾病的情况下可协助诊断，若在肺动脉近端发现血栓克直接确诊，长期存在肺动脉高压的情况下，可出现右心室肥厚。

3.5 D-二聚体的指标在急性期的时候其值低于500μg/L,可基本排除。

4. 诊断与鉴别诊断

4.1冠状动脉硬化性心脏病：两者都可出现胸闷，心绞痛样的表现，但冠心病行冠状动脉造影时可见动脉硬化、管腔堵塞的影像。

4.2心肌梗塞：心肌梗死发病后心电图表现为II、III、aVF的ST段上升，V1～V5的ST段下降，呈下壁心肌梗死图形，3h后ST段抬高更为明显，24h后II、III、aVF出现病理性Q波；而肺栓塞II、III、aVF的P波增高(肺性P波)，I、II、aVF、V1～V5的ST段降低，而后逐渐恢复。

4.3肺炎：两者都有咳嗽、咯血、呼吸困难及胸痛的表现，但肺炎可有相应的肺部和全身感染的表现，如高热、寒战、外周血白细胞增高等，抗菌药物治疗有效。

4.4主动脉夹层：主动脉夹层多伴有高血压，疼痛更为剧烈，胸片常显示纵膈增宽，心血管超声和胸部CT造影可见主动脉夹层表现。

5. 发病后的主要并发症

急性肺栓塞可引起急性的肺动脉高压及有心衰竭，从而导致肺缺血、缺氧，左心输出量下降最终引起心源性休克。

急诊处理

1.一般治疗

肺栓塞发病后应密切监测患者血压、心率、呼吸、心电图和动脉血气等。

2.对症治疗

2.1镇静止痛：为减轻患者疼痛，必要时可给吗啡、杜冷丁等。

2.2治疗急性右心功能不全：可选用多巴胺和多巴酚丁胺，增加心博出量，若出现血压下降，可增大剂量或者加用血管加压药物，如去甲肾上腺素等。

2.3抗休克治疗：补充血容量，必要时可加用多巴胺等，维持体液循环。

2.4保持呼吸道通畅：改善患者的通气功能，必要时机械通气。

3.溶栓治疗：主要适用于大面积肺栓塞的患者，对于血压和右心功能正常的患者，不推荐溶栓，同时注意出血等溶栓的并发症，最严重的为颅内出血，发生率为1%-2%。

3.1链激酶：负荷量250000单位，30min静脉注射，然后100000单位/h，连续24h静脉给药。

3.2尿激酶：负荷量4400单位/kg，10min静脉注射，然后2200单位/kg/h，连续12h静脉给药。

3.3t-PA：50-100mg，2h内连续静脉注射。

4.抗凝治疗：抗凝药物主要有肝素、低分子肝素和华法林。

4.1肝素治疗：予3000-5000单位或按80单位/kg静注，继而以18单位/（kg/h）持续静滴。

4.2低分子肝素：用药剂量一般在4000～8000单位/12h皮下注射。

4.3华法林：华法令的起效时间为2～3天，因此肝素停药前的3～4天开始给药，华法令的使用剂量使PT值比对照值延长1.5～2.5倍，国际标准化比率至2.0～2.5之间。

疗效标准

1.治愈：发病内症状体征消失。各项指标、实验室检查及并发症均已治愈。

2.好转：病情明显好转，症状体征尚未完全恢复正常，各项并发症尚未完全治愈。

3.未愈：未达到上述标准者。

出院标准

凡达到临床治者可出院，好或未者需家属签字要求出院者方可院或出院。
急诊建议
1.对于不明原因呼吸困难、咯血、胸痛患者，必须考虑到肺栓塞可能，同时必须向家属交代病情严重性及可能出现呼吸、心跳骤停风险。

2.在初步怀疑患者可能存在肺栓塞情况下，应尽可能保证呼吸功能及生命体征稳定。

3.在转运过程中，尽量避免可能加重病情各种可能，同时选择有条件就近医院迅速转运。

4.条件具备医院应争取完善相关检查，同时做好一切可能出现发症抢救措施。

5.依据病情及医疗条件等，来考虑选用内科保守、抗凝或者溶栓等治疗。
