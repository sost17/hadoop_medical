牙磨损
牙齿自萌出建牙合后，硬组织每天都会因摩擦而丧失。这种现象常见于牙合面，也会发生在邻面，且丧失程度与随着年龄增加而加重。根据牙硬组织丧失的原因、速度及危害，将其分为两种类型：一为磨耗，二为磨损。 磨耗（attrition）指在正常咀嚼过程中，牙体硬组织的缓慢丧失。由于牙髓腔相应部位有不断形成的继发性牙本质（secondary dentin）,因此牙体硬组织的厚度并无明显降低。磨耗又被称为咀嚼磨损，是生理性的增龄性变化，无明显危害，无需专门处理。 磨损（abrasion）指除正常咀嚼之外，高强度、反复的机械摩擦造成的牙体硬组织快速丧失。磨损发生时牙髓腔相应的部位则形成反应性牙本质（reactionary dentin）。磨损又称非咀嚼磨损，属于病理性变化，应针对病因采取治疗措施。

发病机制

病因

1. 刷牙不当 在刷牙过程中，刷毛过硬、牙膏中摩擦剂的颗粒过大、刷牙速度快或用力过大以及不正确的刷牙方式等都会造成牙体硬组织的磨损。2. 不良咬合习惯 因职业习惯或不良习惯，用较大力度反复咬硬物是导致牙体硬组织磨损的原因之一。比如木匠用牙咬铁钉、吸烟的人习惯咬烟斗，嗑瓜子、有人习惯用后牙咬啤酒瓶盖、核桃等会造成牙齿特定部位的明显缺损。 3. 医源性损伤反 复进行牙根表面的刮治和根面整可能导致牙根表面的磨损；而活动义齿的卡环也会造成基牙牙颈部及邻面的磨损。 4. 其他磨牙症（bruxism） 又称夜磨牙，是指非进食情况下发生的不自主的嚼运动，多在睡眠中发生。而在夜磨牙过程中由于没有食物的缓冲及唾液的润滑，且用力大，多导致明显的牙体硬组织磨损。

症状要点

可能出现牙齿敏感、食物嵌塞、软组织损伤、颞下颌关弹响、疼痛及牙髓根周病变的症状。

查体要点

后牙磨损一般比前牙严重，尤以牙合面最明显。磨损可导致牙齿原本的尖窝沟嵴结构破坏或消失，牙本质外露。因磨损不均匀，常见牙齿边缘锐利。磨损处质硬，与未磨损部位无明显分界线。后牙磨损严重者原邻牙的点接触可能变为较松的面接触。在临床检查中可能发现有食物嵌塞、邻面龋、牙周病等体征。

前牙磨损多见于咬合关系不好及有不良咬合习惯者，严重者可见临床牙冠明显变短。磨损可作为病因导致以下症状或疾病：

1. 牙本质敏感症磨损导致牙本质暴露，患者自觉受冷热刺激及咀嚼时出现酸痛感，且磨损越快酸痛越明显。

2. 食物嵌塞磨损导致邻面点接触变成面接触，且牙合面正常的沟嵴结构消失，食物的溢出通道作用减弱，容易造成食物嵌塞。

3. 牙髓根尖周病过度磨损导致髓腔暴露，口腔内的细菌进入髓腔导致牙髓感染及根尖周病。

4. 颞下颌关节功能紊乱综合征牙合面的重度磨损可导致颌垂直距离过短，从而引起颞下颌关节病损，出现相应的关节弹响、疼痛。

5. 创伤不均匀的磨损会导致牙冠出现高耸的牙尖和锐利的边缘嵴，这些牙尖和边缘嵴抗折性差，极易折断且容易导致临近的软组织，如舌、颊黏膜的损伤。

治疗

治疗原则

根据牙齿磨的因干预，并对患者表现的临床症状进行治疗。

一般治疗

1.改变不良的刷牙方式，戒除不良咬合习惯。

2. 调磨高耸牙尖锐利边缘。

3. 对食物塞的患者，应通过调牙合、恢复接触关系等措施加以改善。

4. 对出现牙本质过敏的患者应选择脱敏治疗。

5. 对于出现牙髓炎、根尖周炎的患者应进行相应的根管治疗。

6. 对磨牙症患者可通过配戴咬合垫、肌电反馈治疗以精神、心理干预等治疗方案加以改善。
