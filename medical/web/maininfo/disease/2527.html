心肌病
心肌病是由于心脏心肌壁功能受损和心室的结构改变所导致心脏功能进行性障碍的一组病变。其临床表现常见有心律失常、心脏扩大、栓塞及心力衰竭等。其病因一般与病毒感染、遗传、自身免疫反应、代谢异常和药物中毒等有关系。按病理特点可分为扩张型心肌病、限制型心肌病和肥厚型心肌病等几类。

发病机制

病因

心肌病是一种原因不明的心肌疾病，一般认为病因与病毒感染、遗传、自身免疫反应、药物中毒和代谢异常等有关。它不包括病因明确的或者继发于全身性疾病的特异性心肌病。心肌病大概可分为三种：扩张型心肌病，限制型心肌病和肥厚型心肌病。中以扩张型心肌病和肥厚型心肌病较为常见，但是心肌病的发病原因至今未明。

1.扩张型心肌病

扩张型心肌病可能和某些特殊因素有关，比如病毒、细菌感染，药物中毒、代谢异常等所致的心肌损伤，中病毒性心肌炎被认为是最主要的致病原因。而持续性病毒感染对心肌组织的损伤与自身免疫所包括细胞、自身抗体或者细胞因子介导的心肌损伤等可导致或者诱发扩张型心肌病。除此之外，围生期、酒精中毒、抗癌药物、心肌能量代谢紊乱和神经激素受体异常等诸多因素也可能引起本病。

2.肥厚型心肌病

肥厚型心肌病可能与常染色体的显性遗传有关，约1/3的患者有明显家庭史，心脏肌球蛋白重链及心脏肌钙蛋白T因突变则是主要的致病因素。此外，儿茶酚胺的代谢异常，细胞内的钙调节异常或者高血压、高强度运动等均可作为本病发病的促进因子。

3.限制型心肌病

限制型心肌病特征为心内膜心肌纤维化、心肌僵硬及心室舒张充盈受阻。起病缓慢，早期可能有低发热、乏力、头晕、气急等症状，晚期出现全心衰竭。心房颤动也是比较常见的，部分还合并与内脏栓塞。 此病多见于热带和温带地区，我国仅有散发病例。

4.致心律失常性右室心肌病

致心律失常性右室心肌病，旧称致心律失常性右室发育不良，特征为进行性纤维脂肪组织替代右心室的心肌，临床常表现有右心室扩大、心律失常和猝死。1995年WHO/ISFC工作组专家委员会关于心肌病定义及分类的报告中将ARVD/C列为与扩张型心肌病、限制型心肌病、肥厚型心肌病并列的第4类原发性心肌病。

病理

院前急救判断与处置

院前急救判断

1.电话询问

n 年龄多大？

n 是否突然发病？

n 起病前有无上呼吸道感染、情绪激动或劳累、用力等诱因？

n 起病时是否出现呼吸困难、心悸、晕厥？

n 有无水肿、血栓栓塞症状？

n 生命体征稳定吗？

2.途中思考

n 是急性还是慢性？

n 是原发性还是继发性？

3.可危及生命的事件

n 心衰

n 心律失常

n 血栓栓塞

n 感染性心内膜炎

4.现场评估

4.1临床症状

①呼吸困难，大汗淋漓，不能平卧，烦躁不安，发绀，有恐慌或濒死感。

②咳嗽，咯血，咯粉红色泡沫样痰。

③心悸，胸闷，头昏，意识改变。

4.2查体所见

①心脏浊音界可扩大。

②心率增快，心音弱，可出现交替脉，心脏杂音。

③血压改变：血压升高或降低。

④双肺多数散在大中湿性啰音。

4.3急危重症评估

出现以下症状为危急重症：

①患者发病急，短时间内迅速出现急性心衰、心律失常、血栓栓塞、感染性心内膜炎。

②患者呼吸困难，烦躁不安。

③心音弱，出现交替脉，提示心脏有严重损害。

院前急救处置

1.取半卧位或端坐位，下肢下垂。

2.吸氧，纠正缺氧。

3.建立静脉通道。

4.镇静：应用吗啡，吗啡注射液3～5mg静脉推注。

5.快速利尿：呋塞米注射液20～40mg静推，2min内推完，10min内起效。

6.血管扩张剂：硝酸甘油、硝普钠，注意监测血压。

选择医疗机构与转运

1.选择疗机

怀疑心肌病时，要考虑到发病后早期可出现急性肺水肿、心律失常、血栓栓塞。诊断延迟，转运不及时可能会导致预后不良，无论患者症状轻重，都应该选择转送到能对患者进行心血管专科治疗的院，以及可急诊处理的疗机。

2.转运时注意

要及时转送，转送时取端坐位或半卧位，双腿下垂，移动应避免刺激，尽量注意安静状态下转送。转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

一、初步评估

1.评估血流动力学是否稳定。

2.急性肺水肿：端坐呼吸、面色苍白、口唇发绀、大汗、咳粉红色泡沫痰，满肺水泡音，可出现血压下降。

3.致命性心律失常：室性心动过速、高度或Ⅲ度房室传导阻滞、窦性停搏等。

二、紧急处理措施

1.急性心衰：取半卧位或端坐位，双下肢下垂。吸氧，纠正缺氧。镇静：应用吗啡，吗啡注射液3～5mg静脉推注。快速利尿：呋塞米注射液20～40mg静推，2min内推完，10min内起效。血管扩张剂：硝酸甘油、硝普钠、重组人脑利钠肽。

2.维持血流动力学稳定，存在致命性心律失常者给予抗心律失常治疗。

3.保持气道通畅，必要时机械通气。

全面诊断与检查

一、临床表现

1.扩张型心肌病的主要临床表现如下：

充血性心力衰竭：心力衰竭是本病最突出的表现。其发生主要是由于心室收缩力下降、顺应性降低和体液潴留导致心排血量不足和（或）心室同时出现充盈压过度增高所致。可出现左心功能不全的症状，常见的为进行性乏力或劳动耐力下降、劳力性呼吸困难、端坐呼吸以及阵发性夜间呼吸困难等左心衰竭的临床表现，疾病晚期可同时出现右心衰竭的临床症状和体征，如肝大、上腹部不适以及周围性水肿等。

心律失常：本病在各种病程阶段可发生各种快速或缓慢型心律失常，甚至可为本病首发临床表现。其中严重恶性心律失常尤其是室性心律失常是导致该病猝死的常见病因。

栓塞：可发生心、脑、肾或肺栓塞。血栓来源于扩大的心室或心房，尤其是伴有心房颤动时。外周血管栓塞偶见可为该病首发症状。

胸痛：虽然冠状动脉主要分支正常，但仍有约1／3的患者出现胸痛，其发生可能与肺动脉高压、心包受累、微血管性心肌缺血以及其他不明因素有关。

2.肥厚型心肌病的临床表现为胸闷、胸痛、气促、晕厥或黑蒙、头晕心源性猝死。

3.限制性心肌病：乏力、呼吸困难和运动耐力下降是限制性心肌病的常见主诉，严重者还会出现水肿、端坐呼吸、肝脏肿大、少尿、腹水及消化道淤血的症状。

4.致心律失常性右室心肌病以青壮年男性多见，常以室性期前收缩、室速等起病，是年轻人猝死的常见原因。致心律失常性右室心肌病的临床表现取决于心肌病变范围，病变范围广泛可出现心脏扩大和充血性心力衰竭（右心衰竭常见）。

二、体格检查

1.扩张型心肌病常见的体征有：心尖搏动常明显向左侧移位，但左心室明显向后增大时可不出现，常可听到第三、第四心音（奔马律）。心功能失代偿期时会出现明显的二尖瓣反流杂音。右心功能不全时可见发绀、颈动脉怒张、肝大、下肢浮肿、少数有胸腔积液、腹水等。

2.肥厚型心肌病的体征可见：（1）心脏轻度增大（心浊音界可向左扩大）；（2）听诊闻及病理性第四心音及第二心音反常分裂，后者是由于左心室射血受阻、主动脉瓣延迟关闭所致。有左心室流出道梗阻者可在胸骨左缘第3～4肋间闻及心尖内侧听到粗糙的收缩中晚期喷射性杂音，并可沿胸骨左缘向下及向心尖部传导，可伴有阵颤。

3.限制性心肌病的体征可见血压偏低、脉压差小、颈静脉怒张、Kussmaul征阳性（吸气时静脉压升高）。心脏浊音界扩大、心律失常、可闻第三心音、第四心音。当合并有二、三尖瓣关闭不全时，常会听到二、三尖瓣收缩期反流性杂音。双肺可闻湿啰音。肝脏肿大，有时会有腹水。双下肢水肿。

4. 致心律失常性右室心肌病在体格检查早期常常无任何异常，有时可见各种心律失常的表现，可有心力衰竭的体征。

三、辅助检查

1.X线胸片：扩张型心肌病X线检查可有心影增大，肺部淤血，胸腔积液等表现。肥厚型心肌病表现为左心室增大或正常。

2.心电图：改变各不相同，扩张型心肌病常显示左心房和右心室增大，可见房性或室性心律失常。肥厚型心肌病主要表现ST-T改变，病理性Q波。心电图是致心律失常性右室心肌病的重要诊断手段之一，V1～V3导联QRS终末部分出现Epsilon波或QRS波延长（＞110ms）为特征性改变。

3.多普勒超声心动图检查：扩张型心肌病可见左右心室扩大和心肌收缩力下降，并有助于同其他类型的心肌病以及瓣膜病、先心病等进行鉴别。肥厚型心肌病可见左心室壁弥慢性或呈节段性的增厚，非扩张性伴收缩末容积减小。超声心动图对右心室结构及功能异常程度的评价具有很高的价值，但由于该病早期病变较局限，使得超声心动图观察右心室结构有一定的局限性，故漏检率较高。

4.心脏磁共振成像：心脏磁共振成像检查：可观察局部心肌肥厚、瘢痕、纤维化，并可定量测量心肌肥厚程度，特别适用于超声心动图难以检出或确诊者，是目前最敏感、最可靠的无创诊断方法。

5.心导管检查及血管造影检查：流出道梗阻时左心室与左心室流出道之间出现压力阶差，与左心室流出道梗阻程度呈正相关。左心室造影对肥厚型心肌病有诊断价值。

四、诊断

心肌病的临床表现多样，确诊主要依靠实验室检查，其中包括超声心动图检查，心脏的磁共振成像以及心脏心导管检查，结合临床症状和体征做出综合判断。

五、并发症

心肌病常见的并发症有心律失常、心衰、栓塞、感染性心内膜炎及猝死：

(1)感染性心内膜炎和猝死多发生于有心肌肥厚者。

(2)栓塞多发生于心肌纤维化及收缩力下降、合并心房颤动、久卧不动或用利尿药的患者。

(3)猝死是常见的致命性并发症。

急诊处理

1.扩张型心肌病

改善症状：限制体力活动，给予低盐饮食。多数患者可应用洋地黄制剂，但易发生洋地黄中毒，用量宜小，地高辛常用量为0.125mg/d。血管紧张素转换酶抑制剂（ACEI）是心力衰竭的一线药物，可改善心力衰竭的血流动力学异常。

预防并发症：β受体阻制剂能降低心源性猝死或死于心力衰竭恶化的风险。

2.肥厚型心肌病治疗

非梗阻性肥厚型心肌病患者主要以药物治疗为主，常用的药物包括β受体阻制剂和非二氢吡啶钙离子拮抗剂（CCB），并发心律失常者可采用抗心律失常药物和起搏治疗，应避免使用正性肌力药物及减轻心脏前负荷的药物。

3.限制性心肌病

治疗包括对因治疗、对症治疗、手术治疗。本病病程长短不一，轻者存活期可达25年，死亡原因多为心力衰竭或肺栓塞。病变累及左心室、心功能Ⅲ级～Ⅳ级（NYHA）、严重二尖瓣与三尖瓣关闭不全及栓塞多提示预后不良。

4.致心律失常右室心肌病

其治疗方面除改善心脏功能、纠正充血性心力衰竭外，主要是控制室性心律失常、预防心脏性猝死。

疗效标准

1.治愈：心脏形态结构及能完全恢复正常，无任何后遗症，如栓塞等。

2.转：心脏形态结构未能完全恢复正常，心能恢复至二级。

3.未愈：治疗后心脏形态及能状态均无显著改善或呈恶化趋势。

出院标准

凡达到临床治愈者可出院，好转未愈者需家签字要求出院者方可转院出院。
急诊建议
1.心肌病患者容易出现急性心衰、心律失常、栓塞、感染性心内膜炎，使病情加重或恶化，要详细向患者或家属说明，争取合作。

2.条件具备的医院应争取做超声心动图、心脏磁共振成像检查。

3.依据心肌病的情况、病情及医疗条件，考虑选用介入治疗或内科保守治疗。
