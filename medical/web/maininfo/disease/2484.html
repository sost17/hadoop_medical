弥漫性血管内凝血
弥散性血管内凝血（DIC）是在许多疾病基础上，以微血管体系损伤为病理基础，凝血剂溶系统被激活，导致全身微血管血栓形成，凝血因子大量消耗继发纤溶亢进，引起全身出血、微循环衰竭的临床综合征。
【标准名称】：
弥漫性血管内凝血
【英文名称】：
Disseminated intravascular coagulation；DIC

发病机制

病因

易发生DIC的基础疾病很多，其中以感染性疾病、恶性肿瘤、严重创伤、病理产科最为常见，约占发病总数的80%。

1. 严重感染 ①细菌性感染，革兰阴性菌感染如脑膜炎球菌、大肠埃希菌等；革兰阳性菌感染如金黄色葡萄球菌；②病毒感染如流行性出血热；③其他感染如钩端螺旋体。

2. 恶性肿瘤常见肿瘤如急性白血病、淋巴瘤、肺癌等。

3. 病理产科如胎盘早剥、羊水栓塞等

4. 手术及创伤如脑、前列腺的手术或创伤。

5. 其他。

病理

1．微血栓形成 微血栓形成是DIC的基本和特异性病理变化，是在致病因素的作用下，外源性和内源性凝血系统被激活，破坏了体内凝血和抗凝过程的平衡，维蛋白原形成维蛋白在微循环内广泛沉积，形成微血栓。

2. 凝血功能异常 DIC早期出现高凝状态，后由于大量血小板和凝血因子被消耗，止血成分生存期缩短，溶系统被激活。大量维蛋白降解产物的产生，妨碍了血小板的功能和正常维蛋白的聚合，血液从高凝状态转变为低凝状态，引起广泛而严重的出血。

3. 微循环障碍 毛细血管微血栓形成、血容量减少、血管舒张功能失调、心功能受损等因素造成微循环障碍。

院前急救判断与处置

院前急救判断

1.电话询问

（1）是否患有易并发DIC的疾病。

（2）是否有出血倾向，如全身发性出血。

（3）是否有器官功能不全的临床表现。

（4）生命体征稳定吗？

2.途中思考

（1）是否出现休克。

（2）是否需要与重症肝炎、血小板减少性紫癜、原发性纤维蛋白溶解亢进相鉴别。

3.可危及生命的事件

（1）休克

（2）重要脏器出血

（3）脏器功能衰竭

4.现场评估

（1）临床症状： ①出血倾向；②休克或微循环障碍；③微血管栓塞；④微血管病性溶血； ⑤原发病临床表现

（2）查体所见： ①皮肤、黏膜、伤口及穿刺部位出血；②血压下降、肢体湿冷、发绀、神智改变； ③器官衰竭的其他表现；④大量腹水时，腹部膨隆，移动性浊音阳性。

（3）急危重症评估：出现以下症状为危急重症： ①顽固性休克；②器官功能衰竭；③重要脏器出血，如脑出血。

院前急救处置

1．平卧位，合并呕血的患者，取平卧头侧位，利于口腔黏液及呕吐物流出。

2．给予吸氧，保持供氧充分。

3．严密监测识状况、血压和呼吸等主要生命体征的变化。

4. 维持有效血液循环

选择医疗机构与转运

1.选择医疗机构 怀疑DIC时，应考虑到患者可能出现重要脏器出血，顽固性休克和微循环衰竭的可能。诊断延迟，转运不及时可能会导致预后不良。应该选择转送设立血液内科的专科医院或综合型医疗治疗。

2. 转运时注意 要及时转送，转送时移动应避免刺激，尽量注意安静状态下转送。转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1. 就地诊治，卧床休息，稳定患者情绪。

2. 密切监测生命体征的变化。

3. 给予吸氧，保持供氧充分。

4. 维持有效血液循环。

全面诊断与检查

1.临床表

（1）出血倾向：出血是DIC最常见的症状，特点为自发性、多发性出血，常出血不止。部位可遍及全身，多见于皮肤、黏膜，可有瘀点、大片瘀斑。其次是内脏出血，如胃肠道出血、血尿、咯血等，甚至发生颅内出血。

（2）休克或微循环障碍：为一过性或持续性血压下降，发生率约为30%~80%。早期出肾、肺、大脑等器官功能不全，表为血压下降、肢体湿冷、呼吸困难、发绀、神智改变、少尿等。休克程度与出血量常不成正比。原发病与DIC均可发生休克，休克可与DIC形成恶性循环。顽固性休克是DIC预后不良的征兆。

（3） 微血管栓塞及脏器功能衰竭：微血管栓塞分布广泛，浅层微血栓表为皮肤坏死性瘀斑，多见于眼睑、四肢、胸背、会阴等部位。黏膜坏死或溃疡多见于口腔、消化道、肛门等部位。深部器官微血管栓塞可导致器官功能衰竭，如急性肾功能衰竭、呼吸衰竭、肝功能障碍、意识障碍等。

（4）微血管病性溶血：表为进行性贫血，贫血程度与出血量不成比例，偶见轻微黄疸，周围血检查可见畸形或破碎红细胞。

按临床经过可将DIC分为急性型、慢性型。急性型多起源于感染、手术、创伤、病理产科等，病程在7天以内，临床表凶险。慢性型多起因于肿瘤、变态反应性疾病等，病程一般超过14天，主要表为轻、中度出血。

2. 实验室检查和其他检查

DIC常用的筛选指标包括：血小板计数、凝血酶原时间（PT）、活化部分凝血酶原时间（APTT）、纤维蛋白原（FIB）、鱼精蛋白副凝（3P）实验、纤维蛋白（原）降解产物等。同时有以下三项以上异常可作为确诊依据：①血小板＜100×109/L或进行性下降，肝病、白血病患者血小板＜50×109/L；②血浆纤维蛋白原含量＜1.5g/L或进行性下降，或＞4g/L，白血病及其他恶性肿瘤＜1.8g/L，肝病＜1.0g/L；③3P试验阳性或血浆FDP＞20mg/L，肝病、白血病FDP＞60mg/L，或D-二聚体水平升高或阳性；④PT缩短或延长3秒以上，肝病、白血病延长5秒以上，或APTT缩短或延长10秒以上。

急诊处理

1. 积极治疗基础疾病及消除诱因治疗基础疾病、消除或钝化促发DIC的病理过程是治疗DIC最为关键和根本的治疗措施，如控制感染、治疗肿瘤、产科及伤、纠正缺血、缺氧及酸中毒等

2. 抗凝治疗：抗凝治疗是终止DIC病理过程，防止微血栓栓塞、减轻器官损害，重建凝血-抗凝平衡的重要治疗措施。临床上常用抗凝药物包括普通肝素和低分子量肝素。普通肝素：急性DIC每日10000~30000U/d，一般为12500U/d左右，每6小时用量不超过5000U，静脉点滴，根据病情可连续使用3~5天。低分子肝素：常用剂量为每天75~150IUAXa/kg，1次或分2次皮下注射，连用3~5天。肝素使用的适应证包括：①DIC早期；②血小板和凝血因子进行性下降，微血管栓塞；③消耗性低凝期，病因短期内不能消除，在补充凝血因子情况下使用。在下列情况下慎用肝素：手术或损伤面未止血者；近期有严重出血；蛇毒所致DIC；DCI晚期，患者有多种凝血因子缺乏；纤溶亢进。

3. 补充血小板及凝血因子适用于有明显血小板或凝血因子减少，或已进行病因及抗凝治疗，DIC未能良好控制，有明显出血表现者。可酌情选用以下制品：①新鲜冷冻血浆，每次10~15ml/kg；②血小板悬液：未出血患者的血小板计数＜20×109/L，或者存在活动性出血且血小板＜50×109/L的DIC患者；③纤维蛋白原：首次剂量2.0~4.0g，静脉滴注。24小时内给予8.0~12.0g，可使血浆纤维蛋白原升至1.0g/L，一般每3天用药一次；④其他，如凝血因子Ⅷ、凝血酶原复合物等。

4. 纤溶抑制药物临床应用较少。主要用于确实存在纤溶亢进的特殊病例如白血病等出血倾向的主要原因是纤溶亢进；或DIC晚期，继发纤溶亢进已经成为迟发性出血的主要原因。常用药物包括：氨基己酸、氨甲环酸等。

5. 溶栓疗法主要用于DIC后期、器官衰竭明显经上述治疗无效，或有确切血栓栓塞的证据。但由于DIC多伴有纤溶亢进，原则上慎用溶栓剂。

疗效标准

1. 治愈 DIC症状体征消失。凝血功能指标恢复正常，各项并发症均已治愈。

2. 好转 病情明显好转，症状体征尚完全恢复正常，凝血功能指标处在好转中，各项并发症尚完全治愈。

3.愈 达到上述标准者。

出院标准

凡达到临床治愈者可出院，转未愈者需家属签字要求出院者方可转院出院。
