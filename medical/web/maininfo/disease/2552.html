疝气嵌顿
体内脏器或组织离开其正常解剖位，通过先天或后天形成的薄弱点、缺损或孔隙进入另一倍位，称为疝。疝气嵌顿是指由于腹内压突然增高，小肠、大网膜等疝内容物强行通过疝囊颈而进入疝囊，随后疝囊颈弹性回缩，将小肠等内容物卡住，使其不能回纳。疝发生嵌顿后，肠管等受压水肿，如能及时解除嵌顿，病变肠管可恢复正常。如不及时解除嵌顿，肠管及其系膜受压情况不断加重可使动脉血流减少，最后导致完全血流阻断，即为绞窄性疝。最终肠管缺血坏死。

发病机制

病因

疝气按疝门的部位可分为腹股沟疝、股疝、脐疝、切口疝等。腹股沟疝又可分为斜疝及直疝。腹外疝中以斜疝是最为常见，占全部腹外疝的75%～90%；男性患者以腹股沟疝发生占大多数，右侧较左侧多见。男性较女性生病率之比约为15︰1；在腹外疝中，股疝嵌顿者最多，可高达60%，但肌疝只占腹外疝发病率的3%～5%，。腹外疝发生有两个主要原因：（1）腹壁强度降低 常见有因素有：①某些组织穿过腹壁的部位，灵精索或子宫圆韧带穿过腹股沟管、股动静脉穿过股管、脐血管穿过脐环等处；②腹白线因发育不全也可以成为腹壁的薄弱点。③手术切口感至愈合不良、腹壁外伤及感，老年、肥胖所致肌萎缩等也常是腹壁强度降低的原因。（2）腹内压力增高 如前列增生所至排尿困难、慢性咳嗽、慢性便秘、经常搬运重物、妊娠、腹水、婴儿啼哭等是引起腹腔内压力增高的常见原因。正常人虽偶有腹内压增高的情况，但如腹壁强度正常，则不会发生疝。

病理

疝气嵌顿发生之后，疝囊颈下受压的管及系膜可发生静脉回流受阻，导致管淤血和水肿，壁颜色由正常的淡红色逐渐转为深红，疝囊内可有少量淡黄色液积聚。管受压而壁水肿加重使疝内容物更难回纳。管嵌顿时系膜内动脉的搏动可扪及，嵌顿如能及时解除，病变管可恢复正常。管嵌顿如不能及时解除嵌顿，系膜动脉可因管及其系膜受压情况不断加重而致血流减少至完全血流阻断，如为后者即为绞窄性疝。此时系膜逐渐失去其光泽，弹性和蠕动能力，最终变黑坏死。疝囊内渗液将变为淡红色或暗红色。如果继发感染，疝囊内的渗液将变为脓性。股疝发生嵌顿者最多，可占达股疝的60%，本病发生后可迅速发展为绞窄性疝，要特别关注。疝气嵌顿后可出现全身变化，如水、电解质和酸碱失衡，血容量下降、休克、呼吸及心脏功能障碍等。
院前急救判断与处置

院前急救判断

1.电话询问

n 年龄？

n 突然发病？

n 起病前有无剧烈咳嗽、用力排便等诱因？

n 是否有腹部包块不能自行回纳？

n 有无腹痛、恶心、呕吐、停止排排便等？

n 生命体征稳定吗？

2.途中思考

n 易得性和难复性？

n 嵌顿性和绞窄性？

3.可危及生命的事件

n 呕吐物窒息

n 脓毒症休克

4.现场评估

4.1临床症状

①腹股沟部有一突出肿块，平卧或用手推送不能使肿块回纳。

②有腹部持续绞痛、恶心、呕吐、停止排排便、腹胀等机械性肠梗阻表现。

③发生脓毒症者出现低血压、神志改变。

4.2查体所见

①腹股沟肿块明显增大，紧张发硬，且有明显触痛，如为大网膜，疼痛常较轻微。

②腹痛明显，早期肠鸣音亢进，有过水声或金属音。

③可有腹膜刺激征表现。

4.3急危重症评估

出现以下症状为危急重症：

4.3.1患者发病急，短时间内迅速出现意识障碍。

4.3.2低血压、休克表现。

院前急救处置

1．取平卧头侧位，利于口腔黏液及呕吐物流出。

2．保持呼吸道通畅，及清理呼吸道分泌物，防止误吸。

3．给予吸氧，保持供氧充分。

4．严密监测意识状况、血压和呼吸等主要生命体征的变化。

5．建立静脉通道，维持有效血液循环，低血压者应使用血管活性药物维持有效血流灌注。

6．可尝试手法回纳肿块。

选择医疗机构与转运

1.选择医疗机构

怀疑疝气嵌顿时，要考虑到发病后需要手术治疗。转运不及时可能会导致预后不良，无论患者症状轻重，都应该选择转送到能24h对患者进行急诊处理，有普通外科的二级医疗机构。

2.转运时注意

要及时转送，转送时移动应避免刺激，尽量注意安静状态下转送。对伴有呕吐的患者要注意转送时患者体位的摆放。转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1.卧床休息，禁食水。

2.必要时开通中心脉通道。

3.密切监测生命体征的变化。

4.胃肠减压

5.留置尿管

6.吸氧，保持呼吸道畅通

7.请外科医师进行会诊

全面诊断与检查

1.症状

1.1起病形式：起病突然，多数患者发病前有明显诱因（用力咳嗽、用力排尿排便、搬运重物、妊娠等腹腔内压力增高）。

1.2主要症状：疝块突然增大，并伴有明显疼痛。多伴有腹痛、腹胀、停止排气排便、恶心、呕吐、便血等。

2.体格检查

2.1腹股沟区可摸到明显肿块或包块，有时腹部可见胃肠型或蠕动波。

2.2腹膜刺激征：患者出现腹肌紧张、压痛、反跳痛、拒按压等腹膜刺激征。

3.实验室检查和其他检查

3.1血常规示白细胞计数升高。

3.2 B检查可为判断嵌顿的程度及肠梗阻的类型提示有用信息，能够发现疝内容物的大小、类型及性质，也能够发现嵌顿性腹外疝所致的肠腔积气积液、肠蠕动及疝囊积液情况。还能提示腹腔积液情况及相应鉴别信息。

3.3腹平片多显示小肠袢扩张及梗阻。

3.4腹腔穿刺可抽出血性液体。

4.鉴别诊断

4.1睾丸鞘膜积液 多表现为一侧阴囊肿大，一般可以清楚地摸到肿块完全局限在阴囊内，鞘膜积液多为透光试验阳性，而疝块则不能透光。

4.2隐睾 本病以睾丸下降停留在腹股沟管而容易被诊为斜疝或精索鞘膜积液。如发现患侧阴囊内睾丸缺如，则可明确诊断。B检查有较大价值。

4.3急性肠梗阻 急性肠梗阻的病因中有因肠管被嵌顿所致，但临床工作中不应满足于肠梗阻的诊断而忽略疝的存在；特别是患者比较肥胖或疝块较小时，容易导致诊治。

4.4肿大的淋巴结 嵌顿性股疝常诊为腹股沟区淋巴结，B检查有助鉴别。

4.5髂腰部结核性脓肿 表现为腹股沟的外侧部、偏髂窝外波动性肿块，易被认为是腹股沟疝，病变是由于脊柱或骶髂关节结核感染所致寒性脓肿可沿腰大肌流至腹股沟区所致。多有低热、盗汗等结核中毒症状，检查脊柱常可见腰椎病变。

急诊处理

1.手法复位 指征：①嵌顿时间较短（3～4h内），局部压痛不明显，无腹肌紧张、无腹膜炎指征者。②年老体弱或伴其他较严重疾病（不宜手术的）而肠袢尚未绞窄坏死者。方法：先注射吗啡或哌替啶，止痛和镇静以达到腹肌松施。患者取头低足高卧位，术者右手托起阴囊，持续缓慢地将疝块推向腹腔，同时用左手轻轻按摩浅环和深环以协助疝内容物回纳。注意事项：在疝内容物回纳后仍要严密观察腹部情况以判断有无腹膜炎或肠梗阻表现，如在上述情况，则应尽早行手术探查。但手术复位仅对症处理，病因未根治，复位指征要严格把握。

2.手术治疗 嵌顿疝原则上需紧急手术，以防止疝内容物坏死并解除伴发的肠梗阻。绞窄疝的内容物已坏死，必须手术。术前做好必要的准备，如有脱水和电解质紊乱，应迅速补速加以纠正。手术的关键在于正确判断疝内容物的活力，术中根据疝内容物的活力情况确定手术方法。如肠坏死则需要切除坏死肠管。如不能判断是否在坏死，可在肠系膜根部注射0.25%～0.5%普鲁卡因60～80ml，温湿盐水纱布覆盖肠段后进行观察判断活力。手术中注意事项：①逆行性嵌顿疝的处理。②活力可疑的肠管避免遗留在腹腔内。③凡因嵌顿绞窄的肠管因坏死而切除吻合的患者，予行高位疝囊结扎，为免因感染而致修补失败，一般不再做疝修补术。

疗效标准

1.治愈：腹股沟肿块消失，手术伤口已经拆线，肠道畅，各项并发症均已治愈。

2.好转：病情明显好转，症状体征尚未完全恢复正常，各项并发症尚未完全治愈。

3.未愈：未达到上述标准者

出院标准

凡达到临床治愈者出院，好转或愈者需家属签字要求出院者方转院或出院。
急诊建议
1．患者手法复位后，疼痛不能及缓解，应及手术探查。

2．如疝容物不能自回纳，患者尽早到医院就诊，避免病情加重延误。

3．B超、腹平片等检查尽早完善。
