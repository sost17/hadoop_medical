胃肠型食物中毒
胃肠型食物中毒是细菌性食物中毒的一种，主要是由于进食被细菌或细菌毒素污染的食物而引起的急性感染中毒性疾病，临床上以恶心、呕吐、腹痛、腹泻等急性胃肠炎表现为主要特征。多发生于气温高、细菌易繁殖的夏秋季，常集体发病，潜伏期短，呈突然暴发，患者进食同种可疑食物，多食者病情重，少食者病情轻，停止进食可疑食物流行可得到迅速控制。

发病机制

病因

当病原菌污染食物后可大量繁殖并产生肠毒素或释放内毒素，大量活菌或毒素伴随食物进入人体后，可引起人体剧烈的胃肠道反应。进食活菌的数量及毒素量的多少、人体免疫力的强弱均与是否发病及病情的轻重有关，主要致病因素如下：

1. 肠毒素 各类病原菌产生的肠毒素致病作用基本相似。由于肠毒素刺激肠壁上皮细胞并激活其腺苷酸环化酶，在活性腺苷酸环化酶的催化下，细胞质中的三磷腺苷脱去两个磷酸而成为环磷酸腺苷（cAMP）。cAMP浓度增高可促进细胞质内蛋白质磷酸化过程，激活细胞内特有的一系列酶反应，从而抑制肠壁上皮细胞对水、钠的吸收，促进肠液与氯离子的分泌，进而引起腹泻。

2. 内毒素 沙门菌菌体裂解后所释放的内毒素致病性较强，进入人体后能引起发热、胃肠黏膜炎症，并促进消化道蠕动加快，从而产生呕吐和腹泻症状。

3. 病原菌 沙门菌、副溶血弧菌、变形杆菌等进入肠道后能直接侵袭肠黏膜上皮细胞，引起局部肠黏膜充血、水肿、上皮细胞变性、坏死、脱落并形成溃疡。

4. 过敏反应 莫根变形杆菌能使蛋白质中的组氨酸脱羧形成组胺，引起过敏反应。

病理

重症病可有胃、小肠充血、糜、出血，部分病可发生结肠炎症及出血，肝、肾、肺等可有中毒性病变。
诊断

病史要点

根据进食同种疑食物后同食者短期内连续发病，突然暴发，出现急性胃肠炎症状，初步出临床诊断。

症状要点

胃肠型食物中毒临床表现以急性胃肠炎症状为主，起病急，可有恶心、呕、腹痛、腹泻等。腹痛以中上腹部持续或阵发性绞痛多见，呕物多为所进食物。常先后泻，腹泻轻重不一，每天数次至数十次不等，多为黄色稀便、水样或黏液便，腹泻严重者可导致脱水、酸中毒甚至休克。根据所感染细菌的种类不同，患者中毒症状也不完全相同。

1. 沙门氏菌食物中毒多在进食后4～24小时内发病，也可短至2小时内，长者可达2～3天。中毒症状包括忽冷忽热、呕、腹泻，大便为水样、恶臭，有时含有黏液和血液。经治疗多在3～5天恢复正常。

2. 葡萄球菌容易污染淀粉类食物（如剩米饭、面、粥）和乳类、肉类食物，一般在进食后2～5小时发病。中毒症状有恶心、上腹痛和腹泻，以呕最为显著，呕物含胆汁，有时带血和黏液。腹痛以上腹部及脐周多见。腹泻频繁，多为黄色稀便和水样便。经治疗可在1～2天内恢复。

3. 嗜盐菌主要污染海产品（如鱼、虾、蟹）和肉禽类腌制品。患者多在进食后数小时至2天内发病。除胃肠炎症状外，腹痛较明显，大便呈血水样。

4. 变形杆菌食物中毒包括过敏型和胃肠型两类，过敏型潜伏期为0.5～2小时，主要表现为颜面部和上身皮肤潮红、头晕、头痛及荨麻疹。胃肠型潜伏期为3～20小时，主要症状为恶心、呕、头晕、头痛、乏力、阵发性剧烈腹痛、腹泻、水样便伴黏液，有恶臭、无脓血，一日数次至十余次。多数病例在1～2天内迅速痊愈，短者仅数小时，长者可达数日。

辅助检查

实验室检查

1.血常规 沙门菌感染者白细胞计数及分类多在正常范围内，副溶血弧菌及金黄色葡萄球菌感染者白细胞计数可增高达10×109/L以上，中性粒细胞比例可增高。

2. 大便常规 稀水样便者镜检可见少量白细胞，血水样便者镜检可见较多红细胞及少量白细胞，血性黏液便者镜检可见较多红细胞及白细胞。

3. 细菌培养 对可疑食物、患者呕吐物及粪便采样进行细菌培养，可分离出病原菌。疑有败血症时应进行血培养。

4. 特异性核酸检查 近年来采用特异性核酸探针进行核酸杂交和特异性引物进行聚合酶链反应以检查病原菌，同时可做分型。

5. 血清学检查 急性期和恢复期双份血清与自身分离出的病原菌做凝集反应时，效价呈四倍增高者即可确诊。

鉴别诊断

1. 急性细菌性痢疾 以腹泻为主，恶心、呕吐较轻，腹泻为黏液脓血便，量少，伴里急后重，下腹部及左下腹明显压痛。粪便涂片镜检有大量红、白细胞，并可见吞噬细胞，粪便培养可分离出痢疾杆菌。

2. 霍乱和副霍乱 在流行区有接触史，多先泻后吐，吐泻严重，腹泻常为无痛性，呕吐常为喷射性与连续性。吐泻物为米泔水样，量多，易导致脱水、酸中毒及周围循环衰竭，粪便涂片荧光抗体染色镜检及培养可分离出霍乱弧菌或埃尔托弧菌。

3. 病毒性胃肠炎 此类患者无明显进食污染食物史，急性起病，潜伏期24～72小时，主要表现为发热、恶心、呕吐、腹痛、腹泻等症状，大便每日数次或十数次，多呈水样便，粪便培养无致病菌，粪便病毒分离有助于诊断。

4. 急性坏死性肠炎 全身中毒症状严重，患者脐周或上腹部剧痛，血水样粪便中常伴有坏死组织。易发生休克，重者可有肠麻痹及腹膜刺激征。

5. 非细菌性食物中毒 包括化学性食物中毒和有毒动植物食物中毒。主要根据病史鉴别，患者多有进食被化学毒物（有机磷农药、砷、汞、亚硝酸盐等）污染食物史，或进食毒蕈、河豚、生鱼胆、发芽马铃薯、苦杏仁等食物病史。潜伏期短（数分钟至数小时），临床表现除急性胃肠炎症状外，可有头晕、头痛、乏力、发绀、肢端麻木、惊厥等神经系统症状及肾功能损害等表现。可疑食物、呕吐物与粪便标本可检出毒物。

治疗

治疗原则

本病的病原菌或肠毒素多于短期内排出体外，自限性疾病，病程短，一1～3天，长者4～5天。临床上以对症治疗主，注意补充吐泻所致的水盐丢失。本病轻重差异极大，应特别强调不同病情不同对待。

一般治疗

患者须注意休息，早期饮以消化的流质或半流质物为主，病情好转后逐渐改为正常饮。

药物治疗

恶心者可口服甲氧氯普胺10mg/次，3～4次/天，呕吐、腹痛明显者可口服丙胺太林（溴丙胺太林）15～30mg，或给予阿托品0.5mg皮下或肌内注射，也可注射山莨菪碱10mg。能口服者均应给予口服补液盐，剧烈呕吐不能进食或腹泻频繁者给予静脉补液。酸中毒时应根据患者二氧化碳结合力的测定结果量补充5%碳酸氢钠或11.2%乳酸钠溶液。脱水严重甚至休克者应根据脱水程度积极补液、抗休克治疗。

一般轻症可不用抗生素，重症持续发热、腹泻频繁或有脓血便者可选用氟喹诺酮类、氨基糖苷类或β内酰胺类药物。病原菌明确者应根据细菌对药物的敏感度选用合的抗菌药物，疗程一般3～5天。

常见并发症
1. 溶血性尿毒综合征 多于疾病高峰期发生，由细菌毒素引起，主要表现为急性溶血、血小板减少和肾功能障碍。

2. 格林-巴利综合征 多于腹泻起病1～2周后出现，肢体出现迟缓性麻痹，常由下肢开始，迅速向上发展，进而出现呼吸肌麻痹，需人工呼吸机辅助呼吸，病死率较高。

3. 瑞特综合征（Reiter syndrome） 多表现为无菌性尿道炎、周围性关节炎及眼结合膜炎。

4. 血栓性血小板减少性紫癜。

预后
胃肠型食物中毒为自限性疾病，病程短，一般预后良好。年龄较大、原有慢性基础疾病者可发生脱水酸中毒、休克甚至及生命。
随访
。
预防
1. 认真贯彻《食品卫生法》，加强对饮食业的卫生监督和检查，包括原料的采购、加工、制作、运输、贮存到销售等所有环节。

2. 对食品制作人员加强卫生知培训，定期进行健康检查。

3. 对食品尽一切可能避免污染，包括煮熟、标准巴氏消毒等。熟食品食用前必须再加热。

4 . 做好饮食卫生的宣传教育。

5. 发现可疑病例及时上报，立即终止可疑食物的食用，制定防御措施，及早控制疫情。

6. 对密切接触者，可严格饭前、便后洗等预防措施。

疑难病例
