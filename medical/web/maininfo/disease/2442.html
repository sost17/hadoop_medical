急性肠炎
急性肠炎是消化系统疾病中常见的疾病,多在夏秋季突然发病，常由肠道感染包括肠道细菌（如大肠杆菌、沙门氏菌、金黄色葡萄球菌）、霍乱、病毒（如埃可病毒、柯萨奇病毒）和其他病毒、肠念球菌）、肠阿米巴、寄生虫等所致；也与误食不洁食物、化学品和药物、食物过敏等有关。部分患者发病前进食可疑不洁食物，同食者可一起发病。起病较急，常有突发的腹痛、腹泻、恶心、呕吐等症状，腹泻轻重不一，可为黄色稀便或稀水样便，也可为深黄色或带绿色，恶臭，一般不含脓血黏液。每日数次至10多次，可有泡沫或少量黏液，严重者可带少量脓血。可伴有腹部绞痛、发热、全身酸痛等症状，严重时可导致脱水，甚至休克。临床上往往恶心、呕吐、腹痛、腹泻同时并见，也称急性胃肠炎。本病可发生于任何年龄，公共卫生欠佳地区发病率较高。大多数患者经治疗痊愈。

发病机制

病因

1．进食腐败、污染的食，如天气热放置时间长的食未加热消毒，不新鲜的肉、海鲜，久存冰箱内的肉类食品，发酵变质的奶及奶制品。

2．对食产过敏反应。

3．肠道感染，常见的有嗜盐杆菌、沙门氏菌、大肠杆菌、变形杆菌及葡萄球菌等感染。

4、饮酒、饮冰饮料过多，或受凉之后，饮食习惯不良，暴饮暴食，进食过多的高脂高蛋白食。

5、抗素滥用：滥用头孢等抗素，这些抗素可以引起肠道菌群失调，刺激肠道，还使肠道内正常的大肠杆菌减少致有害菌大量繁殖。

病理

当过量的刺激性不易消化的食物或含有病原菌及其毒素的食物被食入后，胃肠道黏膜出现急性炎症性病变。病理表现为胃肠道黏膜充血、水肿、过量黏液分泌，有的可伴有黏膜出血及糜烂。炎症刺激消化道的感受器，冲动传入大脑延髓的呕吐中枢，致呕吐中枢兴奋，通过传出神经分别到达胃、膈肌、呼吸肌、腹肌及咽、腭、会厌等处，引起一系列协调运动，产生呕吐。由于肠黏膜的炎症刺激，肠腔内容物增多，引起肠蠕动增强，吸收功能下降，从而出现腹泻。

一些化学物，如非甾体类抗炎药阿司匹林等可通过抑制环氧化物酶，阻断内源性前列腺素E2和I2的合成，上皮分泌的碳酸氢钠及黏液减少，H+反弥散，从而破坏胃黏膜屏障，导致胃肠黏膜的炎症。另一方面还可抑制细胞线粒体内的氧化磷酸化，抑制细胞膜上的Na+-K+-ATP酶和主动运输系统，导致黏膜的渗透性增加，细胞内水钠潴留，细胞肿胀脱落。

有些致敏物刺激肠道，引起自身免疫反应，释放出自身免疫物而激发大量免疫细胞聚集在胃肠道黏膜表面，从而引起黏膜表面水肿充血及渗液等炎症反应。

急性出血性肠炎病变肠管一般呈节段性，表现为肠壁充血、水肿、广泛出血、炎性细胞浸润、坏死和溃疡形成，甚至穿孔。致肠管扩张，肠腔内充满血性坏死物。腹腔内可有混浊的血性渗液。

院前急救判断与处置

院前急救判断

1.电话询问

n 年龄？

n 突然发病？

n 起病前有无食入带有细菌或毒素的食物如变质、腐败、受污染的主副食品？

n 起病时是否出现恶心、呕吐、腹部疼痛？大便性状、是否含脓液和血？

n 同食者有无发病？

n 生命体征稳定吗？

2.途中思考

n 发病原因？

n 导致病情加重的可能原因是什么？

n 坏死性肠炎？

3.可危及生命的事件

n 失水、休克

n 酸中毒

n 肠坏死

4.现场评估

4.1临床症状

①恶心、呕吐、上腹部疼痛不适。

②腹痛，腹泻一日数次或十数次或更多，粪便为糊状或为黄色水样，可带有泡沫或少量黏液。坏死性肠炎可排腥臭血便。

③不同程口渴。

4.2查体所见

①脱水表现，如口干、眼眶凹陷、尿量减少等。

②腹部压痛、腹肌紧张、腹胀、肠鸣音亢进或减弱。

4.3急危重症评估

出现以下症状为危急重症：

4.3.1患者发病急，短时间内迅速出现脱水、休克。

4.3.2患者呼吸急促，酸中毒，

4.3.3腹部压痛、反跳痛且排腥臭血便（可能坏死性肠炎）。

院前急救处置

1．建立静脉输液通道，补液，维持有效血液循环。

2．严重酸中毒时可适当纠酸。

3．腹痛者可适当解痉。

4．严密监测意识状况、血压和呼吸等主要生命体征、腹部情况的变化和大便性状。

选择医疗机构与转运

1.选择医疗机构

就近选择能够进行粪便常规检查、静脉输液、24h急诊处理的医疗机构。

2.转运时注意

要及时转送，转送时对伴有呕吐的者要注意转送时者体位的摆放，呕吐时头偏向一侧，避免误吸。转送途中密切监测生命体征。危重者转运时，要建立至少2条以上的能快速输液的静脉通道，备足相应的急救药品。

急诊科诊断与治疗

初步评估与急救

1.就地诊治，卧床休息。

2.保持呼吸道通畅，必要时吸氧。

3.休克患者密切监测生命体征的变化、腹部情况的变化以及大便性状。

全面诊断与检查

1.症状

1.1起病形式：起病突然，多数患者发病前有明显诱因（食入带有细菌或毒素的食如变质、腐败、受污染的主副食品），同食者有发病。

1.2主要症状：恶心、呕吐、上腹部疼痛不适．腹泻一日数次或十数次，粪便为糊状或为黄色水样，可带有泡沫或少量黏液。坏死性肠炎可排腥臭血便。

2.体格检查

2.1腹部压痛、腹肌紧张、肠坏死可有腹部压痛、反跳痛；肠鸣音可亢进，感染严重或电解质紊乱可致肠蠕动减弱甚至麻痹，肠鸣音可消失。

2.2伴脱水者可有脱水体征，可出现不同程度的口渴。

3.实验室检查和其他检查

3.1血常规化验可正常，或仅有轻度的白细胞增高。

3.2大便化验，除粪汁稀薄外，常无明显异常。

3.3酸中毒、休克患者查血气分析、肾功能等。

4. 危重度评估

以下为儿科常用的判断方法：

①轻度脱水：患儿体重下降3%～5%相当于体液减少30～5Oml/kg。精神略差，略有烦躁不安；哭时有泪，口唇黏膜略干，尿量稍减少；皮肤稍干燥，弹性尚可，眼窝和前囟稍凹陷。

②中度脱水：患儿体重减少5%～10%相当于体液丢失50～lOOml/kg。精神萎靡，烦躁不安；哭时泪少，口唇黏膜干燥；四肢稍凉，尿量明显减少；皮肤苍白、干燥、弹性较差，眼窝和前囟明显凹陷。

③重度脱水：患儿多有超过10%的体重减少相当于体液丢失达100～120ml/kg。重病容，精神极度萎靡，表情淡漠，昏睡乃至昏迷；皮肤可见大理石花纹、弹性差；眼窝和前囟深凹陷，双目凝视，哭时无泪；口唇黏膜极干燥。可表现休克症状，脉搏细速、血压下降、四肢厥冷、尿极少甚至无尿。

成人出现口渴逐渐加重、神志变不清、血压逐渐下降、腹胀逐渐加重、腹部压痛、反跳痛、肠鸣音减弱甚至消失、中毒症状逐渐加重、尿量逐渐减少、腥臭血便、均为病情加重的表现。

5. 诊断与鉴别诊断

诊断：

1）常有进食不洁腐败变质食病史。 　　

2）多在食后短期内突然发病，多有上腹部不适、疼痛、食欲减退、恶心、腹泻，腹泻多为水样便，可含有未消化食，少量粘液，甚至血液等。

3）发病常有体温升高、头痛、周身不适及其他中毒症状。

4）以呕吐、腹泻严重主要表现者，可出现脱水、酸中毒，电解质紊乱，甚至休克等。 　　

5）腹部可有压痛，一般无肌紧张及反跳痛，多有肠鸣音亢进。 　　

6）可有大便常规白细胞、血常规白细胞增高。

鉴别诊断：

1）原发性小肠吸收不良综合征：该病典型症状为脂肪泻。大便量多，呈油脂状或泡沫状，颜色淡，常浮于水面多有恶臭味。多伴营养不良表现，病程长，病情时轻时重。行X线钡餐检查予鉴别。 　　 　　

2）肠结核：缓慢起病，多发于右下腹部，常有阵发性腹部绞痛，常有大便习惯改变，干、稀交替。肠鸣音亢进，轻者一般为稀便，重者可出现黏液脓血便。可行电子纤维肠镜检查鉴别。 　　

3）克隆病：起病缓慢，患者有消瘦、食欲差、乏力等表现。脐周或右下腹痛。腹泻初为间歇性，逐渐为持续性。每日3～6次，软便或半液状。右下腹压痛，可触及包块。可行肠系膜动脉造影、或腹腔镜及病理组织检查予鉴别。 　　

4）特发性溃疡性结肠炎：腹痛常在左下腹或全腹，可有压痛，肠鸣音亢进。患者食欲减退，体重下降及营养不良等。可行乙状结肠镜和病理组织检查予鉴别。

5）急性出血性肠炎：以儿童及青少年居多，常于夏秋季发病，多有不洁饮食史。起病急骤，腹痛多从脐周或上中腹开始，为阵发性绞痛，或为持续性伴阵发加剧。患者可发热、恶心、呕吐、腹泻和解腥臭血便。腹胀、腹肌紧张、压痛，肠鸣音弱。肠管坏死时，全身中毒症状、腹膜炎和肠梗阻症状加重，患者可出现休克。　 　　

6. 发病后的主要并发症

严重时出现失水、酸中毒、休克等。

急诊处理

1.卧床休息，无明显口渴或脱水，予口服葡萄糖-电解质液以补充体液的丢失。持续呕吐或脱水表现明显，则需静脉补充葡萄糖盐水及其他相关电解质。无呕吐无腹胀者，鼓励患者清淡流质饮食，防止继续脱水。

2.呕吐剧烈时可注射止吐药：例如肌内注射灭吐灵或氯丙嗪。腹泻时间长达数日、无腹胀、无中毒症状，可用止泻药：如思密达。

3.抗菌治疗：对于感染性腹泻，可适当选用有针对性的抗菌素，如黄连素或庆大霉素口服。休克、或中毒症状重、或腹部压痛、反跳痛或大便有脓血，应静脉用抗生素，但应防止抗菌素滥用。

4.出现严重失水、酸中毒、休克时，快速建立多条静脉通道进行液体复苏，尽早达到如下指标：（1）中心静脉压（CVP）：8～12 mmHg；（2）平均动脉压（MAP）：≥65 mmHg；（3）尿量：≥0.5 mL•kg-1•hr-1。（4）中心静脉（上腔静脉）氧饱和度ScvO2或者混合静脉氧饱和度SvO2分别≥70％或者≥65％。同时寻找病因，处理原发病。

疗效标准

1.治愈：症状体征消失。各项并发症均已治愈。

2.好转：病情明显好转，症状体征尚未完恢复正常，各项并发症尚未完治愈。

3.未愈：未达到上述标准者。

出院标准

凡达到临床治愈者出院，好转或未愈者需属签字要求出院者方转院或出院。
急诊建议
1.及时诊治，卧床休息。

2.保持呼吸道通畅，必要时吸氧。

3.休克患者密切监测生命体征的变化、腹部情况的变化以及大便性状。对于脱水、休克、酸中毒患者要积极补液，监测生命体征，防止病情进展。

4.呕吐剧烈时可注射止吐药，腹泻时间长达数日、无腹胀、无中毒症状，可止泻药。

5.对于感染性腹泻，可适当选有针对性的抗菌素，如黄连素或庆大霉素口服等。休克、或中毒症状重、或腹部压痛反跳痛、或大便有脓血等，应静脉抗生素。

6.出现严重失水、酸中毒、休克时，快速建立多条静脉通道进行液体复苏，尽早达到如下 指标：（1）中心静脉压（CVP）：8～12 mmHg；（2）平均动脉压（MAP）：≥65 mmHg；（3）尿量：≥0.5 mL·kg-1·hr-1。（4）中心静脉（上腔静脉）氧饱和度ScvO2或者混合静脉氧饱和度SvO2分别≥70％或者≥65％。同时寻找病因进行处理。
