慢性喉炎
慢性喉炎指喉部黏膜的非特异性感染引起的慢性炎症，是耳鼻喉常见疾病之一。由于不良的生活习惯（如抽烟、饮酒等）、过度发声、邻近部位的炎症迁延感染等导致慢性喉炎的患者逐渐增多。根据病变程度不同，分为慢性单纯性喉炎、慢性萎缩性喉炎、慢性肥厚性喉炎。

发病机制

病因

1. 急性喉炎反复发作或迁延不愈。

2. 邻近部位炎症直接向喉部蔓延或脓性分泌物刺激。如慢性鼻窦炎、慢性扁桃体炎、气管炎症、支气管炎症。

3. 反流机制：咽喉反流，此类患者除咽喉不适外，有嘶哑、反复清嗓、咳嗽等表现，要与胃食管反流相鉴别。

4. 过度用声或发音不当。

5. 自身抵抗力下降，导致局部易感染，如全身性疾病患者（糖尿病、慢性肾炎等）、放化疗患者。

6. 内分泌紊乱、缺乏维生素或微量元素、自主神经功能失调等，此因素与慢性萎缩性喉炎相关。

病理

1. 急性喉炎反复发作或迁延不愈。

2. 邻近部位炎症直接向喉部蔓延或脓性分泌物刺激。如慢性鼻窦炎、慢性扁桃体炎、气管炎症、支气管炎症。

3. 反流机制：咽喉反流，此类患者除咽喉不适外，有嘶哑、反复清嗓、咳嗽等表现，要与胃食管反流相鉴别。

4. 过度用声或发音不当。

5. 自身抵抗力下降，导致局部易感染，如全身性疾病患者（糖尿病、慢性肾炎等）、放化疗患者。

6. 内分泌紊乱、缺乏维生素或微量元素、自主神经功能失调等，此因素与慢性萎缩性喉炎相关。

诊断

病史要点

1. 慢单纯喉炎：表现为不同程度的声嘶为主要特点，初为间歇，轻度嘶哑，随着病情加重可发展为持续及程度较重的声嘶症状。喉部异物感，伴有轻微疼痛感觉及紧缩感。临床表现常有干咳。

2. 慢萎缩喉炎：喉部常有干燥不适，声嘶晨起时较重，咳嗽时伴有块或浓稠分泌物，分泌物可有血分泌物，咳出脓后咽痛加重。

3. 慢肥厚喉炎：症状同单纯喉炎，但声嘶较重，咳嗽较轻。

症状要点

1. 慢性单纯性喉炎：喉黏膜弥漫性充血、红肿，带失去原有的瓷白色，呈粉红色，边缘变钝。黏膜表面可见黏液附着。

2. 慢性萎缩性喉炎：喉黏膜干燥、变薄而发亮。杓间区、门下常有绿色或黑褐色干痂，干痂咳出后可见黏膜表面渗血，带变薄。3. 慢性肥厚性喉炎喉黏膜肥厚，以杓间区明显。带明显肥厚，向中线靠拢时有明显缝隙。室带常肥厚而遮盖部分带。

查体要点

1. 检查上呼吸道及咽部情况，注意有无鼻窦炎、鼻咽部、口咽部、扁桃体炎症。

2. 间接镜、维镜或电子镜检查观察杓会厌皱襞、杓间区、声带、室带、声门下黏膜情况，有无充血、增生、表面有无干痂等表现。

辅助检查

实验室检查

慢性喉炎本身并无特异性实验室检查，实验室检查主要是排除其他引起慢性喉炎类似症状的疾病，包括特异性炎症引起的喉结核、喉梅毒等。

影像学检查

慢性喉炎无特异性的影像学表现，影像学检查主要为排除引起类似慢性喉炎其他病变，如喉癌及其他喉部良性肿瘤。

其他检查

1. 频闪喉镜：在带水肿时振幅、黏膜波均有不同程度变化。

2. 电门图检查：在病变轻微时刻保持基本波形，带慢性充血时可见闭相延长，开相缩短。

临床类型

慢性单纯性、慢性萎缩性、慢性肥厚性区别已在病因、病理、病史、体查中予以阐述。

分期

鉴别诊断

1. 喉结核：可继发于肺结核或其他器官结核，临床表现以咽喉疼痛、声嘶为主要表现，咽喉疼痛较慢性喉炎严重，起病急，经抗生素治疗效果差。患者典型临床表现为午后低热、盗汗、咽喉痛剧烈、声嘶，病变常见于喉的后部，杓间区黏膜浅表性溃疡。通过抗结核抗体、结核斑点试验、结核菌素试验等辅助诊断，若有活动性肺结核表现可临床确诊。

2. 喉梅毒：临床表现为声嘶，无显咽痛，声音较为低沉。血清学检查可辅助诊断。

3. 喉白斑：主要表现为声嘶。喉镜检查可见声带前段不光滑新生物，病变局限，声带活动好，不易拭去，此病认为是癌前病变，应定期观察。

4. 喉癌：主要与声门型喉癌鉴别，声门型喉癌临床表现为声嘶，一般无喉痛，电子喉镜可见声带前段不光滑新生物，声带活动可不受限，随着病变程度加重出现声带受限、固定等表现。增强CT可见肿块不均匀强化，活检可确诊。

治疗

治疗原则

病因治疗，去因，增强体质。

一般治疗

养成良好生活及饮食习惯，不吸烟、酗酒。注意劳逸结合，增强体质，减少上呼吸及消化感。饮食搭配合理，补充必要微量元素。注意发声不能长久及高声尖叫，适当安排声带休息。

药物治疗

1. 病因治疗，目前认为胃食管反流可导致慢性喉炎，以咽喉不适为主诉的美国成年人中，10%～50%与胃食管反流有关，因此积极治疗胃食管反流疾病可明显减轻慢性喉炎不适的症状。根据严重程度不同分三期治疗，Ⅰ期治疗主要以改善生活、饮食习惯及加制酸剂（如铝碳酸镁）；Ⅱ期治疗在前期治疗基础上使抗组胺H2受体拮抗剂，如西咪替丁、法莫替丁等；Ⅲ期治疗质子泵抑制剂，如奥美拉唑等。

2. 急性喉炎期积极治疗炎症，疗程足够，减少急性喉炎反复发作转化为慢性喉炎。

3. 积极治疗喉部邻近组织病变，如慢性鼻窦炎、慢性扁桃体炎等。

4. 慢性萎缩性喉炎：碘化钾30mg tid或氯化胺口服，刺激喉部黏液分泌，减轻局部干燥。

5. 局部雾化治疗，予以激素或中草药。

手术治疗

对于药物无法控制的严重胃食管反流致的慢性喉炎患者可予以胃折叠手术。

其他治疗

发矫治包括有练习和发练习，需要业语言矫治师、言语疾病家进行矫治。
