糖尿病脑血管疾病
糖尿病患者（主要为2型糖尿病）除高血糖外，常伴有脑血管病的其他危险因素，如高血压、血脂异常、高黏血症、吸烟等，使糖尿病脑血管病的概率大大升高，患病率较正常人高3～5倍。发病年龄比非糖尿病患者提前5年左右，趋于年轻化。糖尿病患者血液流变学异常，血液黏度增高，血小板聚集黏附性增强，使微血管内血流不畅或栓塞、导致脑血栓形成，故缺血性脑血管病较多，约占89%，其中腔隙性脑梗死发生率较高，可梗死反复发生最终导致脑软化、脑萎缩，而致老年性痴呆，生存质量大大降低，由于糖尿病患者多伴有高血压，因此出血性脑血管病并非罕见，且死亡率可高达90%以上。

发病机制

病因

病因未完全清。

病理

动脉粥样硬化是其主要的病理改变。在临床类型上，糖尿病性脑血管病与非糖尿病性脑血管病无特异性差别。但在糖尿病性脑血管病中，主要为脑血栓形成，而脑出血较少。

另外的病理变化是腔隙性脑梗死。腔隙性脑梗死多因脑基底节区深部穿通动脉阻塞所致的小动脉病变引起。脑梗死后液化，1～3个月后，液化组织被吸收，遗留直径0.5～15mm的空腔。

诊断

病史要点

1. 起病情况

要注意采集患者发病时间、具体状况、病情演变等情况。例如无局灶神经系统体征的脑血管病通常发病时间不明确，症状逐渐发生发展，由于清晨血糖高，血液浓缩，血压也经常偏高，上午4时至9时是缺血性脑血管病的好发时间，症状逐渐加重。脑出血性疾病多发生于剧烈运动、酗酒、情绪激动后，突然发病，常伴头痛等。

2. 既往史

（1） 糖尿病病史及血糖控制情况：资料显示糖尿病是缺血性脑血管疾病的危险因素。

（2） 高血压病史及血压控制情况。

（3） 血脂异常。

（4） 吸烟饮酒史。

（5） 肥胖病史及日常体育运动情况。

症状要点

1. 脑动脉粥样硬化

发病时间和定位体征不明确。主要表现为头昏、头痛、失眠、乏力、健忘、注意力不集中、工作效率低下及情绪不稳定等神经衰弱症状，神经系统阳性体征多不明确。

2. Binswanger病

又称为皮质下动脉硬化性脑病，是较常见的糖尿病性脑血管病，临床以进行性痴呆为主要特征。

3. 无症状卒中

是糖尿病性脑血管病的一种常见类型，是指发现病灶却无临床症状或症状轻微而未引起注意，包括无症状脑梗死（较常见）及无症状脑出血，脑梗死又分为腔隙性脑梗死和非腔隙性脑梗死，这类患者主要因其他原因就诊检查而发现异常病灶或因脑血管病发现责任病灶以为的病变而又无相应临床表现。

4. 急性脑缺血发作

在糖尿病患者群的发病率较脑出血性疾病高，主要包括短暂性脑缺血发作（TIA）及脑梗死。TIA被认为是脑梗死的超级预警信号，TIA后早期发生脑梗死的主要危险因素之一是糖尿病，TIA发病突然，持续时间短暂，历时10～15min，多在1h内，可反复发作，一般在24h完全恢复，无后遗症，多表现为单侧视觉或大脑半球症状，视觉症状表现为一过性黑矇、雾视、视野中有黑点或有时眼前有阴影。大脑半球症状多为一侧面部或肢体的乏力或麻木，可能出现失语、认知及行为功能的改变，一过性单眼黑矇可在眼动脉缺血时出现。椎-基底动脉系统短暂性脑缺血发作(TIA)主要表现为一过性眩晕、头昏、语言不清、倒发作、共济失调、异常的眼球运动、复视、交叉性运动或感觉障碍、偏盲或双侧视力丧失。TIA的诊断需符合下列诊断标准：①突然发病；②局灶性脑或视网膜功能障碍的症状；③持续时间短暂，一般十多分钟，多在1h内，最长不超过24h；④恢复完全，且无后遗症。多数脑梗死均在静态下急性起病，心源性脑梗死以动态起病多见，部分病例在发病前可有TIA发作，病情多在几小时或几天内达到高峰，部分患者症状可进行性加重或波动。梗死灶的大小和部位决定临床表现，主要为局灶性神经功能缺损的症状和体征，如偏瘫、偏身感觉障碍、失语、共济失调等，部分患者可出现头痛、呕吐、昏迷等全脑症状。

5. 脑出血性疾病

糖尿病在脑出血发病机制中的影响是有争议的，甚至有报道非糖尿病患者群脑出血的发病率高于糖尿病患者，但也有研究显示脑出血患者中糖尿病患病率较其他人群高。脑出血多在动态下急性起病，包括脑出血及蛛网膜下腔出血。脑出血突发出现局灶性神经功能缺损症状，常伴有头痛、呕吐，可伴有血压增高、意识障碍和脑膜刺激征。大多蛛网膜下腔出血突发剧烈头痛，进行性加重，持续不能缓解，多伴有恶心、呕吐，可伴有短暂的意识障碍及烦躁、谵妄等精神症状，少数可有癫痫发作。

6. 颅内静脉系统血栓形成较脑动脉血栓少见，临床表现容易与良性颅内压增高、颅内占位病变、缺血或出血性卒中、脑脓肿、脑炎、代谢性脑病等多种疾病混淆，故易误诊、漏诊。随着近年来CT、MRI和MRA的广泛应用，使本病大多能及时准确地诊断。各年龄组均可发病，多数为年轻患者，常无高血压、动脉硬化病史。发病形式多样，多数（73%）以亚急性（48h～30d）和慢性（30d以上）起病。临床表现复杂多样而不典型，除海绵窦血栓形成外均缺乏特征性，可表现为单纯颅内压增高、伴或不伴有颅内高压的局灶性脑功能受累的表现（瘫痪、癫痫、失语、偏盲、感觉障碍等）、意识障碍为主等。由于脑静脉间吻合丰富，特别当大脑皮质静脉受累时，症状体征多变呈波动性，单侧或双侧均可见，也可左右交替，分布也不符合动脉血栓致供血区功能障碍的特点，可伴发脑实质出血。

查体要点

1. 监测体温、脉搏、呼吸、血压等生命体征。

2. 检查营养状况、意识状况、语调与语态、体位。

3. 检查颅神经、运动功能、感觉功能。

4. 神经系统浅反射、深反射、病理反射、脑膜刺激征。

5. 自主神经功能检查。

辅助检查

实验室检查

1. 空腹及餐后血糖，GHbAlc,评估血糖的控制水平，检测血胰岛素、C肽水平、血脂、肝肾功能。

影像学检查

1. 头影像学检查，头颅CT或MRI

急性脑梗死CT首选,较MRI更普及和经济，扫描时间短，对诊断早期除外脑出血更容易。

2.测量颈动脉超声和颈动脉内膜厚度，动脉粥样硬化病变的程度可以从颈动脉内膜中层厚度了解。

其他检查

1. 血管病变诊断：经颅多普勒超声可以发现头颅血管的畅和血流情况。

2.心电图、超声心动图、肝肾超声检查。

临床类型

1.脑动脉粥样硬化

研究表明，糖尿病患者病程5年以下的发生率为31%，病程5年以上发生了率可达70%，由于脑动脉粥样硬化狭窄尚无完全闭塞，脑血供减少所致。临症状通常无特异性。

2.Binswanger病

是一种特殊类型的血管性痴呆，发生在脑动脉粥样硬化的基础上，发病机制是由于脑白质深穿支小动脉硬化，管腔狭窄，血流减少，白质血管广泛变性，致使脑室周围和半卵圆中心白质进行性缺血及神经纤维脱髓鞘，甚至导致发性腔隙性脑梗死。

3.无症状卒中

是糖尿病脑血管病变的一种常见类型，因病灶太小或未累及重要的运动、感觉传导束而无症状或症状轻微被忽视；或者是未被认定的卒中，即无明确卒中发作史，神经系统症状和体征不明确；或检查时发现责任病灶以外的病变而又无相应病史者。无症状卒中包括无症状脑梗死和无症状脑出血，以前者较常见，前者又以腔隙性脑梗死常见。

4.急性脑缺血发作

糖尿病性脑梗死与非糖尿病性脑梗死部位及类型也显著不同，前者发生后循环梗死及腔隙性脑梗死，而后者则以前循环梗死见。糖尿病微血管病变引起微循环障碍及微血管瘤形成，导致基底膜增厚，尤其是豆纹动脉、丘脑深穿支小动脉闭塞，这也是导致糖尿病腔隙性脑梗死和后循环脑梗死见的原因。糖尿病患者动脉粥样硬化性血栓性脑梗死和发性脑梗死的发生率也较高。在临表现上，糖尿病脑梗死易于发生运动功能障碍、构音困难、言语不清，非糖尿病脑梗死以失语及吞咽困难见。

5.糖尿病性脑出血

糖尿病性脑出血患者的出血量大、病死率高和治愈率低，预后差。可能与糖尿病患者血管胶原蛋白的非酶促糖化使血管脆性增加、出血后愈合能力下降等有关。

鉴别诊断

1. 糖尿病酮症酸中毒昏迷

为最常见的糖尿病急症，1型糖尿病有自发DKA倾向，多为年轻患者，2型糖尿病患者在一定诱因下也可发生DKA，常见诱因有感染、胰岛素治疗中断或不适当减量、各种应激如创伤、手术等。患者早期有明显的多尿、烦渴多饮和乏力，随后出现食欲减退、恶心、呕吐等，呼吸深快，呼气中有烂苹果味，后期严重失水、尿量减少、皮肤干燥，至晚期时不同程度意识障碍，反射迟钝、消失，嗜睡以至昏迷，检测血糖和酮体升高，尿糖和尿酮体强阳性，CO2结合力降低，血PH下降可确诊。

2. 低血糖昏迷

老人低血糖可伴发脑功能障碍，临床症状为精神不振、头晕、思维迟钝、视物不清、步态不稳，可有幻觉、躁狂、肌张力增高性痉挛、昏迷、甚至“植物人”。老年糖尿病患者有时可直接陷入昏迷或惊厥，因其年老体衰、意识能力差、频繁发作低血糖致脑功能障碍等可缺乏低血糖的自主神经警告症状。若低血糖降低程度较重且历时较久，治疗延误，脑细胞可发生不可逆脑损害如点状出血、脑水肿、坏死等，因此糖尿病患者出现昏迷必须紧急处理：尽快检测血糖，如无条件检测可直接静脉注射或口服葡萄糖10g,如果为低血糖则可及时抢救，若无效则需进一步检查昏迷原因。

3. 高渗性非酮症糖尿病昏迷

老年人多见，常见诱因有急性感染、脑血管意外以及某些药物如糖皮质激素、免疫抑制剂等，患者水摄入不足或失水，致高血钠和血浆渗透压升高引起脑细胞脱水，从而引起突出的神经精神症状。起病缓慢，最初常有多尿、多饮，而多食不明显，渐出现神经精神症状，表现为嗜睡、幻觉、定向障碍、偏盲、上肢拍击样粗震颤、癫痫样抽搐（多为局限性发作或单瘫、偏瘫）等，最后陷入昏迷。患者血糖达到或超过33.3mmol/L，血钠可高达155mmol/L,血浆渗透压一般为330～460mmol/L。由于脑血管意外也可以引起高渗性非酮症糖尿病昏迷，因此老年患者需行头颅CT或MRI检查以排除脑血管病变。

4. 糖尿病乳酸性酸中毒昏迷

糖尿病患者合并心肺疾病导致的缺氧或肝肾功能不全时，口服二甲双胍可致其蓄积，可能使葡萄糖的氧化分解减弱，而无氧酵解增强，乳酸产生增多。起病较急，伴酸中毒深大呼吸（不伴烂苹果味）、神志模糊、嗜睡、木僵、昏迷等症状，缺氧者可伴发绀、休克等。实验室检查可有血乳酸升高（多大于5mmol/L）,动脉血气分析提示为代谢性酸中毒，CO2结合力降低、pH下降、阴离子间隙升高，而血酮体不高，尿酮体阴性。

5. 中毒

过量毒物因误食或意外接触有毒物质、用药过量、自杀或谋害等原因进入体内引起中毒，临床表现为昏迷、惊厥、呼吸困难、休克等，这类疾病多可以发现毒物接触史或自杀倾向等，皮肤黏膜可出现化学物质的灼伤、发绀或黄疸等，阿托品、莨菪碱类中毒可使双侧瞳孔扩大；有机磷、氨基甲酸酯类中毒可使双侧瞳孔缩小。除神经系统表现外，通常伴有呼吸系统、循环系统、泌尿系统和血液系统等表现，可通过检测残留实物或血液的毒物分析确诊。

6. 高血压脑病

重症高血压患者多见，由于血压过高突破了脑血流自动调节范围，脑组织血流灌注过多导致脑水肿。临床表现以脑病的症状与体征为特征，表现为严重弥漫性头痛、呕吐、意识障碍、精神错乱，甚至昏迷、局灶性或全身抽搐。脑CT或MRI检查提示为无出血或缺血灶的脑水肿。

治疗

治疗原则

1.尽快明确诊断及评估病情，及时治疗。

2.根据病变部位、范围、病理生理改变制订个体化治疗方案。

3.注意监测全身情况及生命体征、血压、血糖、血脂等危险因素，及时处理。

4.预防与处理并发症。

5.强调早期的康复治疗与活动。

一般治疗

1. 监控血糖、血压、血脂及吸氧等，避免血糖、血压过高和过低，控制动脉粥样硬化的进展。

2. 生活应健康规律，戒烟限酒，糖尿病饮食，少食高脂肪、高胆固醇等食品，科学适度地运动，控制体重。

3. 讲究个人卫生，预防感染。

4. 在一般情况下，脑血管病患者如出现肢体、语言等功能障碍时，应尽早做康复治疗，但训练应科学安排、循序渐进。

药物治疗

1.糖尿病合并缺血性脑血管病

(1) 血小板聚集抑制药：常用阿司匹林、双嘧达莫（潘生丁）、噻氯匹定和氯吡格雷等，还可静脉应用奥扎格雷钠。

(2) 抗凝治疗及降纤治疗：常用华法林、低分子肝素、水蛭素和降纤酶、巴曲酶、安克洛酶等。

(3) 溶栓治疗：发病在3～6h且符合适应证者可用尿激酶（UK）、组织型纤维蛋白溶解酶原激活剂（t-PA）等。

(4) 改善微循环：常用银杏叶制剂、川芎、红花、丹参制剂等。

(5) 脑保护治疗：可用钙离子拮抗药如尼莫地平、氟桂利嗪等及胞磷胆碱、脑活素、小牛血清去蛋白注射液、依达拉奉等。

(6) 减轻脑水肿：大面积脑梗死患者，为减轻脑水肿、降低颅内压，可选用20%甘露醇注射液、甘油果糖、呋塞米等，应用上述药物时要注意维持水电解质平衡，尤其是要适当补钾。

(7) 扩血管治疗：梗死灶小、无明显脑水肿、水肿消退后，可用罂粟碱30～60mg或倍他司丁20mg加入生理盐水中静脉滴注，也可口服烟酸0.1～0.2g或倍他司丁6～20mg，1日3次。出血性梗死或血压低者禁用。

(8) 高压氧治疗：部分收缩压低于160mmHg（21.3kPa）的患者在脑水肿消退后，应用高压氧治疗有效。

2.糖尿病合并脑出血等出血性脑血管病：高血压病控制不良可引起脑出血，脑动脉瘤或动静脉畸形可引起蛛网膜下腔出血。

手术治疗

某些重脑卒中患者，使用多种内科治疗，却仍无法控制脑损害的发展，可采用开颅血肿清除术，以使患者转危为安，部分患者甚至获得较好的功能恢复，对于某些老年人等难以承受开颅术者，可采用血肿碎吸引流术，也可获得较理想的效果。大片脑梗死有脑疝危险且药物无法控制者，去骨瓣减压或切除部分坏死组织有可能会挽救患者生命。大量蛛网膜下腔出血和脑室出血，对脑出血尤其伴破入脑室者，脑脊液引流术可排出部分血液，降低颅内压，减轻脑积水，有一定的改善作用。

其他治疗

1. 脑出血除了安静、卧床休息、吸氧、加强护理、保持二便通畅、避免褥疮等一般治疗外，根据出血量可选用适当剂量20%甘露醇注射液、甘油果糖、呋塞米等积极降低颅内压、减轻脑水肿，在应用上述药物时要注意维持水电解质平衡，特别是要适度补充钾。

2. 营养脑细胞，可选用脑活素、小牛血清去蛋白注射液等。

3. 预防并发症，包括选用适宜抗生素防治感染及应用质子泵抑制剂等防治应激性溃疡等。

4. 使用冰毯或冰帽物理降低体温，可使脑代谢率降低、耗氧量减少，有利于保护脑细胞，减轻脑水肿。

5.应用胰岛素治疗糖尿病，并注意监测血糖、电解质和肾功能等。

6.蛛网膜下腔出血除了脑出血治疗外，还需用氨基己酸（EACA）或氨甲苯酸（PAMBA）进行止血治疗及应用钙离子拮抗药，如尼莫地平等防治脑血管痉挛和对症止痛治疗等。

2. 诊断与治疗

根据病史及查体，患者存在高血压、高血脂、高血糖、高尿酸血症及糖尿病大血管、微血管并发症。入院后给予糖尿病相关知识教育，改善饮食结构及生活方式，给予胰岛素皮下注射控制血糖，灯盏花素静点改善循环，VitB1、VitB12营养神经，给予尤诺、碳酸氢钠口服降尿酸，碱化尿液；阿托伐他汀及普罗布考调脂、抗氧化及稳定斑块等治疗；给予阿司匹林抗血小板治疗；高血压波动在129～160/68～99mmHg，给予依那普利降压治疗。经上述治疗后头晕症状较前缓解。未给予血管内支架治疗。
