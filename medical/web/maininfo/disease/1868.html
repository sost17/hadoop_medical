败血症性鼠疫
鼠疫是鼠疫杆菌引起的烈性传染病，系广泛流行于野生啮齿动物间的一种自然疫源性疾病。鼠为其主要传染源，鼠蚤是其传播媒介，在一定条件下在一定条件下通过疫鼠、疫蚤传染给人造成人间鼠疫。本病传染性强，病死率高，属于甲类传染病。世界历史上曾有3次鼠疫大流行，造成数以亿计人死亡。至今亚洲、美洲、非洲等一些国家仍有散发流行，目前我国仅在青海等个别地区有少数散发病例。鼠疫的发生有明显的季节性，鼠疫的分布于其自然疫源地的地理分布相一致。按病变部位和病理变化不同，临床上可分为不同类型，原发性败血症鼠疫很少见，但病情最凶险。多继发于肺鼠疫或腺鼠疫，病初有肺鼠疫和腺鼠疫的相应表现而病情进一步加重[。

发病机制

病因

鼠疫的病原为鼠疫耶尔森菌，鼠类和其他啮齿类动物为主要传染源和保菌动物，家鼠为人间鼠疫重要传染源。鼠疫经鼠蚤传播，经皮肤黏膜、呼吸道、消化道等受染。原发性败血症鼠疫很少见，多继发于肺鼠疫或腺鼠疫。病情严重的患者，型鼠疫均可发展为败血症和毒血症，病原菌可进入血循环并大量繁殖，并可波及干、脾及全身淋巴结。

病理

基本病变为血管和淋巴管的急性出血和坏死。败血症型鼠疫各个器官均可充血水肿、出血或坏死，浆膜腔内可有血性液。镜检可见充血、水肿、出血、细胞退行性变性坏死，炎细胞浸润及细胞团块等。
诊断

病史要点

1. 患者发病前10天内到过物鼠疫流行区。

2. 在10天内接触过自鼠疫疫区的疫源物、物制品、进入过鼠疫实验室或接触过鼠疫实验用品。

3. 患者发病前10天内接触过具有临床表现

（1）突然发病，高热，白细胞剧增，在未用抗菌药物或仅用青霉素族抗菌药物情况下，病情迅速恶化，在48小时内进入休克或更严重的状态。

（2）急性淋巴结炎，淋巴结肿胀，剧烈疼痛并出现强迫体位。

（3）出现重度毒血症、休克综合征而无明显淋巴结肿胀。4)咳嗽、胸痛、咳痰带血或咯血。特征的患者，并发生具有类似表现的疾病。

4. 起病迅速，常有高热或体温不升、极度衰竭、神志不清、谵妄或昏迷等中枢神经系统症状。

症状要点

原发性败血症鼠疫很见，但病情凶险，多继发于肺鼠疫或腺鼠疫。起病极为迅速，全身中毒症状及中枢神经系统症状极为明显，并有出血倾向。表现为高热、寒战或体温不升，伴严重毒血症症状，面色苍白，皮肤黏膜、脏器广泛出血，神志不清甚至昏迷，呼吸急促，血压下降，可数小时至2~3天因感染性休克、DIC及出血死亡。继发性败血症可有肺鼠疫、腺鼠疫发展而来，症状轻重不一。

查体要点

查体发现不整、心律不齐、心音微弱、血压下降、皮肤黏膜有出血点，颜面呈恐怖、痛苦、狰狞表情。

辅助检查

实验室检查

1.常规检查

（1）血象：外周血白细胞总数大多升高，常达（20~30）×109/L以上，以中性粒细胞为主，明显核左移，还可见红细胞、血红蛋白和血小板减少。

（2）尿常规：可见蛋白尿及血尿，尿沉渣中可见红细胞、白细胞和细胞管型。

（3）便常规：大便潜血可阳性。

2.细菌学检查 是确诊本病的依据。可取淋巴结穿刺液、脓、痰、血或脑脊液等，涂片、压片或印片染色镜检，亦可做细菌培养或动物接种。

3.血清学检查 荧光抗体试验、PHA、ELISA、RIP等也敏感、快速、准确。

4.分子生物学检测 主要有DNA探针和聚合酶链反应，快速、敏感、特异，应用广泛。

临床类型

1. 原发性败血症 亦称暴发型鼠疫很少见，但病情最凶险。

2. 继发性败血症 多继发于肺鼠疫或腺鼠疫。

鉴别诊断

1.败血型炭疽 可有皮肤焦痂溃疡及其周围广泛的无痛性非凹陷性水肿，血培养和皮肤病灶分泌物涂片和培养可检出炭疽杆菌。

2. 金黄葡萄球菌败血症 可有原发感染病灶，血培养为金黄色葡萄球菌 。

3. 流行性出血热 有发热、出血及肾损害，典型病人有发热期、低血压休克期、少尿期、多尿期及恢复期5期经过。检测特异性IgM做早期诊断 。

4. 钩端螺旋体病有疫水接触史，全身疼痛和腓肠肌压痛，全身出血倾向和肺弥漫性出血、腹股沟淋巴结肿痛较轻，钩端螺旋体病原学和血清学试验阳性。

治疗

治疗原则

由于本病为烈性传染病，发病急、病情危重，死亡率高，因要求早发现、早诊断、早隔离、早治疗以及疫早处理。

一般治疗

1. 严格的隔离消毒 患者应严格隔离于隔离病院或隔离病区，病区内必须做到无鼠无蚤。入院时对病人做好卫生处理（更衣、灭蚤及消毒）。病区、室内定期进行消毒，病人排泄物和分泌物应用漂白粉或来苏液彻底消毒。工作人员在护理和诊治病人时应穿连衣裤的“五紧”防护服，戴棉花纱布口罩，穿第筒胶鞋，戴薄胶手套及防护眼镜。

2. 饮食与补液 急性期应给患者流质饮食，并供应充分液体，或给予葡萄糖，生理盐水静脉滴注，以利毒素排泄。

3. 护理 严格遵守隔离制度，做好护理工作，消除病人顾虑，达到安静休息目的。

4. 对症治疗 急性期要绝对卧床休息，烦躁不安或疼痛者用镇静止痛剂，呼吸困难者，给氧，出现休克、DIC、心衰者给予相应处理。

药物治疗

早期足量选用有效抗菌药物是治疗本病的关键。

1.链霉素 为治疗各型鼠疫特效药。成人首剂量1g，以后每次0.5g，每4小时1次，肌注，1～2天后改为每6小时1次。小儿20～40mg/（kg体重·d），新生儿10～20mg/（kg体重·d），分2～4次肌注。对严重病例应加大剂量，最初二日，每日4g，继以每日2g，分4次肌注。链霉素可与磺类或四环素等联合应用，以提高疗效。疗程一般7～10天，甚者用至15天。

2.庆大霉素 每日24～32万U，分次稀释后静脉滴入，持续7～10天。

3.四环素 对链霉素耐药时可使用。轻症者初二日，每日2～4g，分次口服，以后每日2g；严重者宜静脉滴注，第1次0.75～1g，每日 2～3g，病情好转后改为口服。疗程7～10天。

4.氯霉素 每日3～4g，分次静脉滴入或口服，退热后减半，疗程5～6天。对小儿及孕妇慎用。

5.磺类药物 首剂5g，4小时后2g，以后每4小时1g，与等量碳酸氢钠同服，用至体温正常3日为止。不能口服者，可静脉注射。磺只对腺鼠疫有效，严重病例不宜单独使用。

预防
1. 严格控制传染源

（1）管理患者：发现疑似或确诊患者，应立即按紧急疫情上报，同时将患者严密隔离，禁止探视及病人互相往来。病人排泄物应彻底消毒，病人死亡应火葬或深埋。接触者应检疫9天，对曾接受预防接种者，检疫期应延至12天。

（2）消灭动物传染源：对自然疫源地进行疫情监测，控制鼠间鼠疫。广泛开展灭鼠爱国卫生运动。旱獭在某些地区是重要传染源，也应大力捕杀。

2. 切断传播途径

（1）灭蚤： 灭蚤必须彻底，对猫、狗，家畜等也要喷药。

（2）加强交通及国境检疫：对来自疫源地外国船只、车辆、飞机等均应进行严格国境卫生检疫，实施灭鼠、灭蚤消毒，对乘客进行隔离留检[12]。

3. 保护易感者

（1）预防接种：主要对象是疫区及周围人群，参加防疫工作人员及进入疫区医务工作者。自鼠间开始流行时，对疫区及其周围居民、进入疫区工作人员，均应进行预防接种。常用为EV无毒株干燥活菌苗，皮肤划痕法接种，即2滴菌液，相距3~4cm。2周后可获免疫。一般每年接种一次，必要时6个月后再接种一次。中国新研制06173菌苗免疫动物后产生F1抗体较EV株效果高1倍。

（2）个人防护进入疫区医务人员，必须接种菌苗，两周后方能进入疫区。工作时必须着防护服，戴口罩、帽子、手套、眼镜、穿胶鞋及隔离衣。接触患者后可服下列一种药物预防，四环素每日2g，分4次服；磺胺嘧啶每日2g，分4次服；或链霉素每日1g，分1～2次肌注，连续6天。

（3）预防服药:与患者接触可口服磺胺嘧啶，每次1g，每日2次，亦可用四环素，每次0.5g，每日4次，均连用6天 。
