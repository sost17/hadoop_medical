低钠血症
低钠血症是常见的电解质紊乱，椐估计2.5%的住院患者存在低钠血症。血清钠降低与各种内分泌、肾、神经和呼吸功能紊乱有关，也与药物及其他治疗等有关；一旦血清钠极度低下，则表现为神志改变（低钠血症性脑病）、惊厥发作和死亡率增加，必须细致地纠正严重的低钠血症，并避免进一步的并发症。

发病机制

病因

低钠血症可见于三种不同的临床形式，这三种形式中细胞外液量分别是减少、增高或正常的（表1）。

表1 水平衡紊乱：低钠血症
血浆渗透压正常 假性低钠血症（高脂血症）；罕见于用离子特异性的钠离子电极测定 血浆渗透压增高 高糖血症 甘露醇、甘油、放射造影剂 血浆渗透压降低 尿最大程度稀释： 1． 溶质排泄减少（蛋白摄入减少） 2． 水摄入过多 尿未最大程度稀释： 1． 细胞外液量正常 a．SIADH：肺病、CNS疾病、药物、焦虑 b．肾上腺功能不全（也可能存在容量缺失） c．甲状腺机能减退症 2． 细胞外液量减少 a．肾外途径丢失 b．经肾丢失：利尿、失钠性肾病 3． 细胞外液量增加 a．充血性心力衰竭 b．肝硬化 c．肾病综合征

病理

低血钠→血浆低渗→血液浓缩、血浆蛋白增加→细胞胀大→血液严重浓缩、血容量减少。
院前急救判断与处置

院前急救判断

1.电话询问

n 何时出现症状

n 是否突发气促

n 有无发绀

n 生命体征

n 是否曾就诊、有无用药史

n 有无吸氧

2.途中思考

n 引起低钠的原因包括肾内丢失、肾外丢失、激素异常、水潴留。

n 伴随症状：注意力或计算能力下降、呼吸抑制、反应力低下、二便失禁、幻觉、去皮层强直和惊厥、昏迷。

3.床边诊治

n 快速目测：一般情况是否稳定

n 生命体征：对于休克要早期识别，一看（神志、面色）、二摸（脉搏、肢湿）、三测（血压）、四量（尿量）；掌握经典的5P 即苍白(Pallor)虚脱 (Prostration)冷汗 (Perspiration)无脉 (Pulselessness)呼吸衰竭 (Pulmonaey insuffciency)

n 询问病史

院前急救处置

1．建立静脉通路，给予盐水补液，快扩容、必要时使用升压药。

2．重视病因治疗，不同类型的低钠血症治疗原则不同。

选择医疗机构与转运

1.选择医疗机构

就近转运到有救治能力的综合医疗机构。

2.转运时注意

一般平卧位，下肢略抬高，在密切监护下，迅速将患者转送到医进一步救治。

急诊科诊断与治疗

初步评估与急救

1. 临床表现：低血容量性休克。Na+＞125，多无症状，Na+＜110，可有昏迷。

2. 诊断：①失钠史；②血钠＜135；③急性血容量减少、血压降低、慢性者有失盐、失水症状；④血浓缩且MCV升高；⑤尿量正常（晚期下降）、尿比重正常、尿钠＜25。

3. 急救：治疗的办法基本上有两种，补充钠盐及限水。补钠以口服氯化钠最安全，计算出补钠量，将食盐放于胶囊中，静脉补充时，可用3％氯化钠溶液。不同类型的低钠血症治疗原则不同。

全面诊断与检查

1. 临床表现

伴渗透压降低的低钠血症往往在血清钠降至125mmol/L以下时才表现出症状，而血清钠变化的速度是很重要的，其快速的发展可导致严重后果。轻症可出现轻微神经症状，如注意力或计算能力下降，而低钠血症急性发展并且血清钠＜115mmol/L或慢性低钠血症时血清钠＜105mmol/L，就可以出现严重的症状，包括神志改变、惊厥、恶心、呕吐、精神恍惚和昏迷等。低钠血症性脑病可表现为呼吸抑制、反应力低下、二便失禁、幻觉、去皮层强直和惊厥等。有些慢性低钠血症患者即使血清钠低至100mmol/L，也可能清醒和定向力正常，这些患者存在缓慢发展的低钠血症。

应寻找基础疾病的临床表现。饮水过多可以由病史和多尿的出现来确定，但ICU中可能会不加注意地给予患者大量的水分。低容量血症患者可有容量不足的表现如低血压、心动过速、皮肤张力减退或体重减少；而高容量血症患者有水肿和体重增加。

2. 实验室检查与其他检查

测定血清电解质、血糖、肌酐、尿素氮、血清渗透压、尿渗透压、尿钠和尿肌酐（计算钠排泄分数）。血清渗透压低（＜280mOsm/kg）可确定低钠血症由水相对于溶质过多引起。如果存在高糖血症，须纠正血清钠值。已发现低钠血症与低钾血症和严重的钾丢失有关，低钾血症也可能使低钠血症患者容易发生渗透性脱髓鞘综合征和脑病。

在过多水摄入引起的低钠血症患者中，尿渗透压是低的（＜300mOsm/kg），而低容量血症患者的尿钠浓度（＜20mmol/L）和钠排泄分数（＜1%）低，这也可见于细胞外液量增加和血管内容量减少的患者。

3. 诊断与鉴别诊断

（1）血清钠＜135mmol/L。

（2）神志改变。

（3）大多数病例由常规血清电解质测定发现。

急诊处理

低钠血症的严重性（Na+＜120mmol/L）、发展进程和神经系统症状的出现决定疗的快慢和积极程度。如果患者无明显症状或低钠血症程度轻或缓慢发展，那么就无须紧急疗，并禁忌积极的干预。

1. 估计水过多

水过多可由下式估计：TBW（L）=正常TBW（L）×140/[Na+]（140mmol/L为正常[Na+]）。例如，70kg的个体其正常TBW为42L（0.6L/kg），如果[Na+]为110mmol/L，那么估计实际TBW为42×140/110=53.5L，水过多为53.5－42=11.5L。

2. 决定是否需要快速或积极的纠正

低钠血症引起精神改变或惊厥的患者需要快速的疗。血清[Na+]严重降低（＜105mmol/L）的患者即使没有症状也应注意。有症状的低钠血症往往与[Na+]严重降低有关。

3. 纠正基础疾病

在导致低钠血症的潜在紊乱中最容易纠正的是低容量血症。心因性水中毒和大量静脉补液的患者应最大程度地排泄水分，如果没有其他医疗问题，那么去除水分摄入可使[Na+]快速恢复正常。大多数患者停止噻嗪类利尿剂可快速恢复最大尿稀释。低钾血症应纠正，因其与低钠血症的并发症及其疗有关。高容量血症伴低钠血症的处理较为困难，但严重的低钠血症不常见。

4. 低张性低钠血症的特异性疗

低张性低钠血症的特异性疗应限制水分摄入并加强水分排泄。对无症状或轻度的低钠血症通常限制水分就足够了，高张盐水和呋塞米适用于有症状的低钠血症和血清钠＜105mmol/L的无症状低钠血症。

（1）限制水摄入：限制经口和胃肠外途径的水摄入可改善任何原因引起的低钠血症，除了伴有低容量血症，所有低钠血症患者均应考虑这一方法。血清钠在125～135mmol/L之间且患者无明显症状时往往将水摄入限制在1000～1500ml即可成功地逆转低钠血症。更为积极的水限制在某些患者可能有用。

（2）高张盐水和呋塞米：最有效的联合疗低钠血症的方法是应用高张盐水（常用3%NaCl）和呋塞米。单用呋塞米（给予40～80mg通常足以产生明显的利尿作用）可增加钠和氯的排泄，并通过抑制Henle氏袢升支的溶质转运使尿稀释，尽管可使水分排除，但同时钠和氯也将丢失。因此，疗中应替代溶质的丢失。

5. 慢性低钠血症的其他疗

可逆性CNS或肺部疾病患者一般对纠正或解决基础疾病有反应。轻度到中度的水限制是必要的。有些患者还需要额外的帮助促进水排泄。

疗效标准

1．治愈：恢复至正常水平。

2．未愈：未达到上述水准者。

出院标准

达到临床治愈好转，病情相对稳定者可出院。
急诊建议
1.应该首先正确区分是哪种类型的低钠血症。

稀释性低钠血症患者对利尿剂的反应很差，血浆渗透压低，因此选用渗透性利尿剂甘露醇利尿效果要好于其他利尿剂，同时联合应用强心剂和袢利尿剂疗效明确。甘露醇100～250ml缓慢静滴，时间控制在2～3h，在输注到半左右时用强心药（如西地兰），再过20min左右根据患者情况静脉推注呋塞米100～200mg。缺钠性低钠血症对利尿剂的效果也很差，可以采用联合应用大剂量袢利尿剂和输注小剂量高渗盐水的方法。

2.治疗期间要严密观察心肺功能变化，以调节剂量和滴速，般以分次补给为宜，否则会引起神经症状和加重心衰。

3.另外还可以采用强化利尿并高张盐水的方法治疗，给予2.72%高张盐水500 ml（生理盐水400ml，10%氯化钠100ml）。同时在给药期间每日限钠120 mmol（食盐7 g），限水1000ml。

4.对于药物治疗无效者可以考虑采用高渗性腹膜透析或动、静脉血液滤过的方法。
