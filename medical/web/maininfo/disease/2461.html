急性气管-支气管炎
急性气管-支气管炎（Acute tracheobronchitis）是气管、支气管黏膜及周围组织因生物性或非生物性因素而引起的急性炎症。

发病机制

病因

1.生物性病原体 主要是病毒（如腺病毒、冠状病毒、流感病毒、柯萨奇病毒及鼻病毒等），病毒感染后可以抑制肺泡吞噬和纤毛上皮细胞的活力。细菌感染常继发于病毒感染后，使肺炎支原体、肺炎链球菌及流血嗜血杆菌、卡他莫拉菌和葡萄球菌等有入侵的机会，这些感染因素均可引起气管、支气管粘膜的损伤。

2.物理和化学因素 如吸烟、冷空气、粉尘、刺激性气体，这些理化因素都可以损伤气管、支气管黏膜，破坏气管上皮细胞，纤毛运动减弱，最终导致气道净化能力下降。香烟烟雾可以使中性粒细胞释放蛋白酶，抑制抗胰蛋白酶系统，破坏肺弹力纤维，最终引起肺气肿。

3.其他因素 免疫、龄均与气管、支气管炎症有关。老人肾上腺皮质功能减退，引起免疫力低下，从而可以引起气管、支气管反复感染，儿童应首先排除少见疾病如肺囊性纤维化或低免疫球蛋白血症。

病理

气管、支气管上皮细胞变性、坏死、脱落，支气管黏膜充血、水肿，鳞状上皮化生，纤毛变短、粘连、倒伏、脱失，黏液腺体肥大，分泌物增加，并有淋巴细胞和中性粒细胞浸润。炎症消退后气管、支气管黏膜结构和功能可恢复。如病情发展，炎症可沿支气管壁向周围组织扩散，黏膜和支气管周围组织增生，肺泡弹性差，可形成阻塞性肺疾病。
院前急救判断与处置

院前急救判断

1.电话询问

n 患者年龄

n 有无气促

n 有无发绀

n 是否曾就诊

n 是否有COPD或哮喘病史

n 生命体征

n 有无吸氧

2.途中思考

伴随症状：胸痛、咳血、发热、咳嗽、咳痰、昏迷与否。

3.可危及生命的事件

缺氧

4.现场评估（危重程度判断）

SpO2＜90%：面罩100%氧气6～10L/min。

SpO2＜95%：鼻导管2L/min给氧。

SpO2＞95%：详细了解病史和查体。

院前急救处置

1．保气道通畅，吸氧。

2. 患者取舒适位，坐、卧或半坐位。

3. 监测血压、心率、呼吸、氧饱和度。

4. 开放静通道，对症用药。

选择医疗机构与转运

1.选择疗机构

对轻中度气管-支气管炎的慢性呼吸功能不全患者，以呼吸困难为主诉表现的，可送往日常就诊的疗机构。

2.转运时注意

转运途中严密监控神志、呼吸、血压、心、心律和SpO2等。

急诊科诊断与治疗

初步评估与急救

1．吸氧：先使用导管或面罩吸氧（2-6L/min），同时监测脉搏氧饱和度。

2．体位：取坐位或半坐位。

3.心电监测：血压、心率、呼吸、经皮脉搏氧饱和度等。

4.开放静脉：保持静脉通道开放。

5.采血：作常规、生化、电解质、凝血、血气分析。

6.对症处理。

全面诊断与检查

生命体征稳定后，做全面的诊断检查。

1.症状

1.1起病形式：起病突然，多数患者发病前有明显诱因（如病毒、遇冷空气、花粉、过敏病史等）。

1.2主要症状：起病急，全身症状轻，也可有发热，初期为干咳或少量咳黏液痰，随后可出现咳嗽加剧，延续2～3周，如不愈可形成慢支气管炎。

2.体格检查：双侧肺可闻及散在干湿啰音，部位不固定，咳嗽后可减少或消失。

3.实验室检查和其他检查

3.1血常规：白细胞计数可升高，中粒细胞升高，血沉增快。

3.2痰培养：24～48h可以确定病原菌。痰标本送检应注意器皿洁净无菌，在使用抗菌药之前，漱口后采集深部的痰液。

3.3X线及胸部CT检查：肺纹理增强，少数无异常发现。

急诊处理

1.对症治疗：咳嗽无痰或少痰，可用右美沙芬、咳必清镇咳。咳嗽痰不易咳出，可选用氨溴索、溴己新化痰，也可用雾化痰。如有气道痉挛，可用氨茶碱、受体激动剂等。发热可用物理降温或解热镇痛药对症处理。

2.如痰培养有细菌感染证据，可给予大环内酯类、青霉素类，也可选用头孢菌类或喹诺酮类的药物，口服即可。

3.多休息，多饮水，避免劳累。

疗效标准

1.治愈：无咳嗽咳痰，热退，无呼吸困难，肺部啰音消失，各项并发症均已治愈，血常规白细胞恢复正常。

2.好转：病情明显好转，症状体征尚未完全恢复正常。

3.未愈：未达到上述标准者。

出院标准

凡达到临床治愈者可出，好或未愈要求出者需家属签字方可或出。
急诊建议
增强体质，避免劳累，止感冒。改生活卫生环境，止空气污染。清楚鼻、咽部位的病灶。
