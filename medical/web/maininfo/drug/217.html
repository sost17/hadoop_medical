结构式

适应证
1．抑症。

2．治疗伴有或不伴有广场恐怖症的惊恐障碍。

用法用量
口服

1．治疗抑郁症：每日10mg，每日1次，据个体反应情况，每日最大剂量可增加至20mg。通常2～4周即可获得抗抑郁疗效，症状缓解后，应持续治疗至少6个月以巩固疗效。

2．伴有或不伴有广场恐怖症的惊恐障碍：每日1次，建议起始剂量为每日5mg，持续一周后增加至每日10mg，根据个体反应，剂量还可以继续增大至每日20mg。

不良反应
1.常见不良反应 食欲减退，恶心、口干、腹泻、便秘等。

2.可发生激泌紊乱、心动过速、味觉异常。

禁忌证
1、对西酞普兰、艾司西酞普兰过敏者禁用。

2、禁忌与非选择性、不可逆性单胺氧化酶抑制合用，

3、禁忌与匹莫齐特合用。

注意事项
1. 不与MAOIs合。

2. 使过程定期检测心电图、肝肾功能、血常规。

3. 孕妇、青少年。

4. 药期间从事需要精神高度紧张、集中地工作和活动者，需要重使本药。

5、抗抑郁剂不适于儿童和十八岁以下的青少年。

药物相互作用
1、禁忌合用：非选择性、不可逆MAOIs、匹莫齐特

2、需谨慎合用：可逆性、选择性MAO-A抑制剂、司来吉兰、5-羟色胺药物、降低癫痫发作阈值的药物、锂盐、色氨酸、圣约翰草，口服抗凝剂、酒精。

用药说明
1．剂量个体化，可与食物同服。

2．不良反应多发生在开始治疗的第1～2周，持续治疗后不良反应的严重程度和发生率都会降低。

3．乳期妇女不应接受本品治疗或在用药期停止乳。

4.服药期应避免从事精细或动作协调性工作。
