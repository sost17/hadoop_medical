盐酸异丙肾上腺素
1．药效学 ①作用于ß1肾上腺素受体，心率加快，使心肌收缩力增强，并进而增加心脏传导系统的传导速度，缩短窦房结的不应期。②作用于ß2肾上腺素受体，松弛支气管平滑肌，抑制组胺等介质的释放。③扩张外周血管，减轻心负荷，纠正低排血量。④加快糖原和脂肪的分解，使组织耗氧量增加。 2．药动学 雾化吸入吸收完全。2～5分钟起效。作用可维持0.5～2小时。生物利用度为80%～100%。有效血浓度为0.5～2.5mg/ml。表观分布容积（Vd）为0.7L/kg。5%～15% 以原形排出。

结构式

适应证
1. 心脏骤停。

2. 支气管哮喘。

3. 抗休克。

4. 房室传导阻滞。

用法用量
1. 心脏骤停：心腔内注射0.5～1mg。

2. 支气管哮喘：已渐被其他药物取代。（1）气雾吸入：成人：以0.25％气雾剂每次吸入1～2揿，每天2～4次。喷吸间隔时间>2小时。极量一次0.4mg，每天2.4mg。（2）舌下含服：成人：一次10～15mg，每天3次。极量一次20mg，每天60mg。（3）儿童用法与用量：1）舌下：5岁以上，一次2.5～lOmg，每天2～3次。2）以0.25％气雾剂每次吸入1揿，每天2～3次。

3. 休克：以0.5～1mg加于5%葡萄糖溶液200ml中，静滴，滴速0.5～2μg/分，根据心率、血压调整剂量。

4. Ⅲ度房室传导阻滞：心率每分钟不及40次时，可以本品0.5～1mg加在5%葡萄糖注射液200~300ml 内缓慢静滴。

不良反应
1.神经系统头晕、目眩、震颤、乏力、多汗。

2.消化系统恶心、口咽发干。

3.循环系统

面部潮红、心率增快、心悸。

禁忌证
1. 对本品过敏者。

2. 心绞痛。

3. 心肌梗死。

4. 甲状腺功能亢进症。

5. 嗜铬细胞瘤。

注意事项
1. 妊娠期妇女 慎用。

2. 哺乳期妇女 慎用。

3. 慎用疾 ①心律失常并伴有心动过速；②心血管疾；③糖尿；④高血压；⑤洋地黄中毒。

4. 其 服药后若感心律失常或胸痛，应予以重视。

药物相互作用
1.其他拟肾上腺素药物与本品，可增效，但不良反应也增多。

2.普萘洛尔与本品，后者作用受到拮。

3.洋地黄类药物与本品，加剧心动过速。

4.茶碱与本品，降低前者的血药浓度。

5.钾盐与本品，易致心律失常，故呈配伍禁忌。

6.三环类抑郁药丙咪嗪、丙卡巴肼与本品，增加后者的不良反应。

用药说明
雾化吸入时，12小时内已喷吸药物3～5次而疗效不著应换药。
制剂与规格
片剂：每片10mg。

气雾剂：每瓶总量14克，内含盐酸异丙肾上腺素35毫克；每揿含盐酸异丙肾上腺素0.175毫克。

注射液：每支1mg(2ml)。
