复方磺胺甲噁唑
1.药效学 SMZ-TMP有协同抑菌或杀菌作用。磺胺药作用于二氢叶酸合成酶，干扰叶酸合成，而TMP进一步作用于叶酸合成，选择性抑制二氢叶酸还原酶的作用，二者双重阻断细菌的叶酸合成。两者的协同抗菌作用较单药增强，耐药菌株减少。 2.药动学 应用SMZ-TMP复合剂时，两者的血药浓度比例为1:20，尿药浓度差异较大。24小时内两者分别约有50%原形经尿排出。

适应证
1. 成人慢性支气管炎急性细菌感染。

2. 儿童急性中耳炎。

3. 细菌性尿路感染。

4. 肠道感染。

5. 肺孢菌病。

6. 诺卡菌病。

7. 耶尔森结肠炎

8. 单核细胞增多性李斯特菌感染。

用法用量
1.成人：口服。①细菌性感染：每次TMP160mg，SMZ800mg，每12小时服用1次。②肺孢菌病：每次TMP160mg，SMZ800mg，每8小时1次,连用21天。 21天后，AIDS患者长期抑制治疗，起始剂量TMP160mg，SMZ800mg，每周3次。

2.儿童用法用量：口服。①细菌感染：2个月以上、体重40kg以下婴幼儿，按体重每次SMZ 20～30mg/kg、TMP4～6mg/kg，每12小时1次。体重≥40kg的儿童，剂量同成人。②肺孢菌病：按体重每次SMZ18.75～25mg/kg、TMP 3.75～5mg/kg，每8小时1次，连用21天。

不良反应
1. 神经系统 头痛、乏力、肌肉疼痛。

2. 消化系统 恶心、呕吐、胃纳减退、腹泻。

3. 泌尿系统 可见结晶尿、管型尿，偶见间质性肾炎或肾小管坏死的严重不良反应。

4. 血液系统 白细胞减少、血小板减少，严重者可见粒细胞缺乏症、再生障碍性贫血。

5. 循环系统 可见出血倾向、溶血性贫血、血红蛋白尿。

6. 过敏反应 常见药疹、荨麻疹，偶见严重的渗出性多形性红斑。

禁忌证
1. 对本品或磺胺类药物过敏。

2. 对呋塞米、砜类、噻嗪类利尿药、磺脲类、碳酸酐酶抑制药过敏者。

3. 肝功能损害者。

4. 肾功能减退者。

5. 巨幼红细胞性贫血。

6. 血液病。

注意事项
1. 老年患者 禁用。

2. 妊娠期妇女 禁用。

3. 哺乳期妇女 禁用。

4. 儿童 新生儿及2个月以下婴儿禁用。

5. 慎用疾病 ①葡萄糖-6-磷酸脱氢酶缺乏症；②血卟啉症；③失水；④休克。

6. 对诊断的干扰 ①交叉过敏反应：对一种磺胺药过敏的患者，对其他磺胺药也可能过敏。②AIDS患者应用磺胺药发生严重不良反应的风险增加。③TMP引起叶酸缺乏时，可同时服用叶酸制剂。如有骨髓抑制征象发生，应停用TMP，并给予叶酸3～6mg，肌注，每天1次，共3天；或根据需要用药至造血功能恢复正常。对长期、大剂量使用本品者可给予高剂量叶酸并延长叶酸的疗程。

7. 随访检查 ①疗程中严密观察，当出现皮疹或其他严重不良反应早期征兆时立即停药。②全身应用磺胺类药时应定期检查周围血象，发现异常及时停药。③疗程中至少每周检查尿常规2～3次，发现结晶尿或血尿时给予碳酸氢钠及饮用大量水，直至结晶尿和血尿消失。③肝、肾功能检查。

药物相互作用
1.碱化尿液的药物与本品，增强后者在碱性尿中的溶解度，促进药物排出。

2.对氨基苯甲酸及其衍生物与本品，拮抗后者抑菌作用。

3.口服抗凝药、口服降糖药、甲氨蝶呤、苯妥英钠、硫喷妥钠等药物与本品，后者与血浆蛋白结合，抑制前者代谢，导致前者作用增强甚至有毒性反应。需调整本品使用剂量。

4. 能引起骨髓抑制的药物与本品，增多白细胞、血小板减少的机会。。

5.口服含雌激素避孕药与本品，避孕失败，增加经期外出血的机会。

6.溶栓药物与本品，增加前者潜在的毒性作用。

7.具有肝毒性药物与本品，可能增高肝毒性发生率。

8.光敏药物与本品，可能增加光敏反应的发生风险。

9. 保泰松与本品，增强前者的作用

10.乌洛托品与本品，增加结晶尿的危险性。

11.磺吡酮与本品，减少后者自肾小管的分泌，血药浓度升高而持久，可发生毒性反应。合用时须调整本品剂量。磺吡酮疗程较长时，须监测本品的血药浓度。

12.氨苯砜与本品，两者血药浓度均升高，前者可致不良反应增多且加重。

13.利福平与本品，增加后者的清除，缩短半衰期。

14.环孢素与本品，增加肾毒性。

15.苯妥英钠与本品，干扰前者的肝内代谢，延长其半衰期，并减少其清除率。

16.普鲁卡因胺与本品，减少前者肾清除，血药浓度增高。

17.抑制华法林与本品，抑制前者的代谢，增强其抗凝作用。

18. 磺胺药与本品，使细菌的叶酸合成代谢遭到双重阻断，有协同抗菌作用，使抑菌作用转为杀菌作用，减少耐药菌株产生。

用药说明
1.静脉滴注 注射剂临用前以本品每支5ml于5%葡萄糖注射液75～125ml中供静脉滴注，滴注时间60～90分钟以上。

2.不宜应用本品：①中耳的预防。②A组血性链球菌扁桃体和咽，不易清除细菌。
