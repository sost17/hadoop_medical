急性胃扩张
急性胃扩张是指短时间内大量气体、液体或食物积聚潴留，胃和十二指肠上段的高度扩张而致的一种综合征。通常为某些内外科疾病或者麻醉手术患者的严重并发症。国内报告多因暴饮暴食所致，病死率较高。发病年龄以21～40岁，男性多见。

发病机制

病因

急性胃扩张的病因多种多样，某些器质性疾病和功能性因素均可引起急性胃扩张，常见下列因素有关。

1.暴饮暴食 占本病发病率的绝大多数，由于短时间内大量进食，使胃腔过度充盈，胃壁肌肉突然受到过度牵伸而呈现反射性麻痹扩张。

2.疾病状态 如胃扭转、嵌顿性食管裂孔疝、十二指肠肿瘤、异物等均可引起急性胃扩张。脊柱畸形、环状胰腺、胰腺癌等也可压迫胃的输出道引起急性胃扩张。情绪紧张、剧烈疼痛、精神抑、营养不良等均可引起自主神经功能紊乱，导致胃张力减低或排空延迟而致急性胃扩张。糖尿病神经病变、抗胆碱能药物的应用、水电解质失衡、中枢神经系统损伤、尿毒症、严重感染等均可影响胃的张力和胃的排空，导致出现急性胃扩张。

3.手术与创伤 腹部大手术患者发生急性胃扩张，可能与麻醉时过多的空气或氧气进入胃内以及手术时胃肠牵拉有关。创伤尤其是上腹部挫伤或严重复合伤，其发生与腹腔神经丛受强烈刺激有关。

4.先天性畸形 此种病因多发生于小儿，如先天性胃肌缺损环状胰腺、幽门肥厚等。

病理

胃肠壁神经性麻痹和机械性梗阻是引起急性胃扩张的发病机制。胃壁肌肉张力减弱引起胃扩张，使食管与贲门，胃与十二指肠交界处形成锐角，阻碍胃内容物的排出。膨大的胃可压迫十二指肠，并将系膜及小肠挤向盆腔，因此，牵拉肠系膜上动脉而压迫十二指肠。胃内食物、咽入的空气及胃十二指肠分泌液、胆汁、胰液积存，刺激胃泌素的分泌，又可以刺激胃十二指肠黏膜分泌液体，加重胃扩张，还可以进一步牵拉肠系膜，加重胃十二指肠麻痹和梗阻，于是形成恶性循环，使胃扩张逐步加重。扩张的胃还可以压迫门静脉，使血液郁滞于腹腔；也可压迫下腔静脉，使回心血量减少，最后可导致周围循环衰竭。由于大量呕吐、禁食和胃肠减压，可引起水和电解质紊乱。
院前急救判断与处置

院前急救判断

1.电话询问

n 突然发病？

n 起病前有无暴饮暴食？

n 起病时是否腹胀、腹痛？

n 有无恶心、呕吐、意识障碍等？

n 是否停止排气排便？

n 是否有类似病史？

n 生命体征稳定吗？

2.途中思考

n 单纯性胃扩张和胃穿孔？

3.可危及生命的事

n 呕吐物窒息

n 机械性胃梗阻

n 脓毒症休克

4.现场评估

4.1临床症状

①腹胀、上腹部或脐周隐痛、胀痛.

②恶心、持续呕吐，呕吐后症状不减轻。

③呕出液常具有典型特性，开始为深棕绿色液体后呈咖啡渣样，但无血块，亦无粪臭味。即应怀疑为急性胃扩张。

4.2查体所见

①上腹部膨胀，可见无蠕动胃轮廓，局部有压痛，叩诊过度回响，有振水音。

②巨胃窦征：脐右偏上出现局限性包块，外观隆起，触之光滑而有弹性，有轻度压痛，其右下边界较清，这是急性胃扩张的特有体征。

③腹膜炎征：严重病例可出现。

4.3急危重症评估

出现以下症状为危急重症.

4.3.1烦躁不安、手足抽搐。

4.3.2呼吸急促，发绀。

4.3.3血压下降和休克。

院前急救处置

1．取半卧头侧位，利于呕吐物流出。

2．吸氧，保持呼吸道通，及时清理呼吸道分泌物，防止误吸。

3．开通静脉通道，血流动力学不稳定者可开通2～3条通道，积极抗休克。

4．心电护，严密测生命体征的变化。

选择医疗机构与转运

1.选择医疗机构

怀疑急性胃扩张时，要考虑到发病后可能需要手术治疗。诊断延迟，转运不及时可能会导致预后不良，无论患者症状轻重，都应该选择转送到能24h对患者进行急诊处理及急诊手术的医疗机构。

2.转运时注意

要及时转送，转送中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1．卧床休息，取半卧位。

2．禁食水。

3．放置胃管持续胃肠减压

4．留置尿管，记每小时尿量。

5．休克患者放置中心静脉导管，使用血管活性药物持有效血流灌注。

全面诊断与检查

1.症状

1.1起病形式：起病突然，急性胃扩张可在数小时内发生。

1.2主要症状：早期出现腹胀、上腹部或脐周隐痛、胀痛。恶心、持续呕吐，呕吐后症状不减轻呕出液常具典型特性，开始为深棕绿色液体后呈咖啡渣样，但无血块，亦无粪臭味。即应怀疑为急性胃扩张。如未能及时做出诊断和处理，随着患者病情的加重，会逐渐出现水和电解质紊乱症状。可出现口渴、尿少、脱水征，患者烦躁不安、呼吸急促、脉搏快速而微弱，血压下降和休克。

2.体格检查

2.1上腹膨胀，可见无蠕动的胃轮廓、局部压痛，振水音。肠鸣音减弱或消失。

2.2巨胃窦征是急性胃扩张的特重要体征，表现为脐右偏上可见局限性包块，边界较清，可见腹部隆起，触之光滑而弹性，轻压痛，这是临床诊断的力佐证。

2.3腹膜炎征：全腹压痛、反跳痛，腹肌紧张。

3.实验室检查和其他检查

3.1 X线腹部平片可见左上腹巨大液平面和充满腹腔的特大胃影及左膈肌抬高。

3.2腹部B超可见胃高度扩张，胃壁变薄，若胃内为大量潴留，可测出其量的多少和在体表的投影，但若为大量气体，与肠胀气不易区分。

3.3实验室检查：白细胞计数升高，血液浓缩、低血钾、低血氯和碱中毒、尿素氮升高

4．诊断

存在上述提到的病因，例如手术后初期创伤、感染或过分饱食后，出现上述溢出性呕吐和上述特征的呕吐物。并上腹部胀满，振水音阳性，即应考虑为急性胃扩张，如腹部X 线平片见上腹部巨大液气平面及特大胃影，或胃管吸出大量液体，即可确诊。

5.鉴别诊断

急性胃扩张应与机械性肠梗阻、弥漫性腹膜炎和幽门梗阻区别。机械性肠梗阻可腹胀、呕吐，但常较明显腹痛，腹部体格检查可见肠型，肠鸣音多亢进，立位腹部x线平片可见小肠积气，并可见肠腔内多个液平面，胃管抽吸无大量胃内容物。弥漫性腹膜炎常由腹腔内脏器穿孔引起，起病急骤，腹痛剧烈，腹部肌肉紧张，压痛及反跳痛．肝脏浊音界可消失，肠鸣音消失，患者体温常升高，白细胞增多。腹部X线检查可发现膈下游离气体。消化性溃疡、胃窦部肿瘤引起的幽门梗阻也可导致胃扩张的发生，但一般起病缓慢，患者呕吐物无胆汁，上腹部可见到胃型及胃蠕动，很少出现脉搏快速而微弱、血压下降等，胃镜检查或X线钡剂造影可明确诊断。

急诊处理

1．对症治疗

1.1食。

1.2持续肠减压。

1.3尽量吸出内容物。

1.4纠正水、电解质和酸碱失衡。

1.5病情稳定24h后可于管内注入少量液体，如无潴留，即可开始少量进食。

2．手术治疗

2.1手术指征：①饱餐后极度扩张，内容物无法吸出。②内科治疗8～12h后，症状改善不明显。③十二指肠机械性梗阻因素存在，无法解除。④合并有穿孔或出现大量出血。⑤功能长期不能恢复，静脉高营养不能长期维持者。

2.2手术方法：手术切开壁清除内食物，全层缝合壁，术后予管继续减压，特别是对已有腹腔感染、休克、穿孔或疑有壁坏死者，应及时行手术治疗，手术方法以简单有效为原则。

疗效标准

1.治愈：除胃管给予进食，临床症状消失。无恶心、呕吐，无腹痛，无解黑便、呕血，无发热、畏寒，无血流动力学稳定等并发症临床表现。

2.好转：病情明显好转，症状体征尚未完全恢复正常，各项并发症尚未完全治愈。

3.未愈：未达到上述标准者。

出院标准

凡到临床治愈者可出院，好转未愈者需家属签字要求出院者方可转院出院。
急诊建议
1．本病死亡率可高20%～60%，一旦发生极少能自行痊愈，延误治疗死亡率可高达80%。本病疗效在于早期治疗干预。

2．应选用较粗胃管置入有利于减压和洗胃。

3．洗胃时应注意避1次用水量过大或用力过猛，造成胃穿孔。

4．积极宣传暴饮暴食的危害性。

5．术中麻醉操作要避使患者吞咽大量空气；术中减少创伤，避对组织的过度牵拉；术后预防腹腔感染，注意给予营养支持。

6．术后要经常翻身及变换体位，以促进患者胃肠功能的恢复。
