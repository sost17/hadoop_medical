发病机制

病因

急进性肾小球肾炎是以急性肾炎综合征、肾功恶化为主要临床表现，并迅速发展为少性急性肾功衰竭为特征。病理改变特征表现为广泛的新月体形成，新月体形成的肾小球数目占全部肾小球数目的50%以上。故又称新月体型肾炎。

因此早期诊断，及时、合理的治疗直接影响到患者的预后。

病理

急进性肾小球肾炎根据免疫病理可分为三型，其病因和发病机制各不相同。①Ι型又称为抗肾小球基底膜（GBM）型肾小球肾炎，占10%～20%，患者血检测有抗GBM抗体。抗GBM病包括两种情况，即单纯局限肾脏的抗GBM肾炎和同时损害累及肺脏的Goodpasture综合征。后者同时伴有肺出血。常见于20～30岁和60～70岁年龄段。20～30岁年龄段以男性多见，肺出血发生率较高，60～70岁以女性常见，肺出血发生率低。②Ⅱ型又称为免疫复合物型，大多数继发于免疫复合物型肾炎，少数是原发免疫复合物型急进性肾炎。本型是我国常见的急进型新月体肾炎，主要见于青少年。血检测到免疫复合物，血清补体C3降低。与Ι型相比临床和病理改变稍轻；③Ⅲ型为非免疫复合物型，又称寡免疫型（pauci-immune）急进性肾炎，多见于中老年人。有一半患者伴有全身性血管炎改变，血清抗中性粒细胞胞浆抗体（antineuturophil cytoplasmic antibody， ANCA）阳性。急进性肾小球肾炎进展很快如不及时诊断和治疗，患者很快进入不可逆，转入终末期肾衰竭。
院前急救判断与处置

院前急救判断

1.电话询问

n 患者年龄？

n 是否突然发病？

n 起病前有无呼吸道感染、接触有机溶剂、服用甲亢药物等诱因？

n 起病后是否出现少尿、肉眼血尿、无尿、血压升高、恶心、呕吐、水肿？

n 有无发热、皮疹、腹痛、咯血、乏力、意识障碍等？

n 生命体征稳定吗？

2.途中思考

n 是急性还是非急性？

n 是原发性还是继发性？

3.可危及生命的事件

n 肺出血致窒息。

n 肾衰竭致急性左心衰、致命性高钾血症、心律失常、重度代谢性酸中毒。

4.现场评估

4.1临床症状

①从未有过的突然发生的排尿改变（包括尿量、次数和性状)。患者可伴有肾区疼痛或压痛，气促、咳血及血尿。

②可出现头晕、头痛，血压升高，乏力、食欲减退，恶心、呕吐、咳血、胸闷、气促等。

③部分患者呈肾病综合征的表现，如水肿、蛋白尿、高血压等，并迅速进展为尿毒症。

4.2查体所见

①一般查体：面色苍白，贫血貌，肢体及颜面水肿、高血压，可有体温升高。

②呼吸急促或深长，肺部听诊可有或无湿性啰音，呼吸音可减弱。

③部分患者有心率快伴奔马律，或有急性左心衰体征。

4.3急危重症评估

出现以下症状为危急重症：

4.3.1患者发病急，短时间内出现急性心肺衰竭表现。

4.3.2患者出现呼吸气促、咳血多改变，心律改变。

4.3.3无尿2d以上。

4.3.4血压急剧高或出现剧烈头痛、呕吐。

院前急救处置

1．取半卧位休息.

2．保持呼吸道通畅，及时清理呼吸道分泌物，防止误吸。如出现咯血者应鼓励咳出血块避免堵住气管致息。

3．给予吸氧，保持供氧充分。

4．严密监测意识状况、心率、血压和呼吸等主要生命体征的变化。

5．维持有效血液循环，可适当利尿。

6．控制血压：合理使用降压药，控制在180/120mmHg以下。

7．出现急性左心衰者按急性左心衰处理。

8．行心电图检测了解有无高耸而基底较窄的T波，注意高钾血症，及时处理如用排钾利尿剂。

选择医疗机构与转运

1.选择医疗机构

怀疑急进性肾小球肾炎时，要考到发病后出现水钠潴留、水电解质紊乱及酸碱失衡，血压升高，严重者可出现Goodpasture综合征、心律失常、急性肺水肿及左心衰、上消化道出血、代谢性脑病等危及生命的并发症。诊断延迟，转运不及时可能会导致预后不良，无论患者症状轻重，都应该选择转送到能24h对患者进行血生化检查、具有血液净化治疗、能行肾脏病理活检、可急诊处理的医疗机构。

2.转运时注意

要及时转送，转送时移动应避免刺激，尽量注意安静状态转送。在搬运患者时，可使用转送担架，转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1.就地诊治，卧床休息。

2.持呼吸道通畅，吸氧，必要时行气管插管或行气管切开术。

3.有条件应争取监护治疗，密切监测生命体征和神经系统体征的变化。

4.注意严重高血压、神经系统损及心血管事件发生。

全面诊断与检查

1.症状

1.1起病形式：急进性肾小球肾炎可有呼吸道前驱感染，起病多较急，病情急剧进展。

1.2主要症状：进行性少尿或无尿，全身症状较重， 如疲乏、无力、精神萎靡， 体重下降，可伴有发热、腹痛、皮疹。血尿为本病较突出的临床表现， 多数患者疾病一开始即有肉眼血尿， 持续时间长；可有轻、中度水肿， 严重水肿者少见。代谢性酸中毒表现为恶心、呕吐、疲乏、嗜睡、呼吸深快、食欲不振和甚至昏迷。

2.体格检查

2.1血压升高，肢体及颜面水肿，贫血貌，可有呼吸深快。可出现胸腹水体征或皮疹、紫癜、蝶形红斑。

2.2呼吸系统症状：部分患者出现咯血及全身血管炎症状应注意肺出血-肾炎（Goodpasture）综合征，多数病情严重，死亡率高。

3.实验室检查和其他检查

3.1 血尿素氮、血清肌酐进行性升高， 反映肾脏病变严重， 随着尿少、血清肌酐升高， 可出现代谢性酸中毒及高钾血症， 必须严密观察。随时追查血气结果。肾功能衰竭可以出现高磷低钙。

3.2 尿沉渣可见大量红细胞， 以畸形红细胞占多数的肾小球源性血尿， 常见红细胞管型和透明管型，尿蛋白定量多数超过2g /24h； 应每日测量体重， 如体重增加表示水、钠潴留；记24h尿量， 如尿量少于400mL 为少尿， 少于100mL为无尿， 均提示病情严重。

3.3 影像学检查：①肾脏B超双肾增大或正常大小， 轮廓整齐， 但皮、髓质交界不清， 与肾脏水肿有关。 核素肾图显示肾脏灌注和滤过减少；一般急进性肾损伤没指征不做造影。

②胸部X线或胸部CT：抗GBM型可有片絮状或团块状阴影，慢性血管炎可见肺间质纤维化改变。

3.4 肾活组织检查光镜：肾小球囊内细胞增生， 形成新月体， 受累肾小球的数量超过50%，比较广泛，新月体形成是肾小球毛细血管伴严重损害的结果；免疫荧光：免疫病理是区别3种急进性肾炎的主要依据。IgG沿肾小球毛细血管基底膜呈细线状沉积是抗GBM肾炎的最特征性表现。免疫复合物型急进性肾炎的免疫荧光主要表现为IgG和C3呈粗颗粒状沉积。由于该型可继发于各种免疫复合物肾炎，因此，继发于免疫复合物肾炎的急进性肾炎同时还有原发病的免疫荧光表现；Ⅲ型急进性肾炎肾脏免疫荧光染色一般呈阴性或微弱阳性。偶尔可见散在IgM和C3沉积。在新月体或血栓中可有纤维蛋白原染色阳性。需要强调的是， RPGN 是临床诊断，新月体肾炎是病理诊断，两者并非完全一致。对临床疑似急进性肾小球肾炎应尽快行肾活检。

3.5 相关检查：疑似新月体性肾炎应做：补体、循环免疫复合物、ANCA、抗GBM抗体、ANA检测； 患者可呈贫血、血红蛋白降低，肺出血时贫血加重， 有时与存在微血管病性溶血性贫血有关， 可查血涂片， 见破碎的头盔状红细胞可确诊；血液免疫学检查：Ⅰ型患者可检出抗肾小球基膜抗体阳性；Ⅱ型患者可有免疫复合物及冷球蛋白和类风湿因子阳性、血清总补体及C3可降低；Ⅲ 型可查到ANCA阳性； ANCA滴度的变化可反映原发性小血管炎的活动， 并可预示疾病的复发。ANA阳性考虑系统性红斑狼疮引起的继发性免疫复合物肾炎。

4. 危重度评估

4.1对肾功能损害的评估：GFR的估算；

4.2 对全身状况的评估：如高血压分级、贫血分度、心功能不全分期、酸中毒分度、水及电解质紊乱的程度评价。

5. 诊断与鉴别诊断

5.1诊断

5.1.1　根据临床表现：凡急性肾炎综合征伴肾功能急剧恶化，无论是否达到少尿性急性肾衰竭应疑似本病，应及时排除肾后性梗阻等因素后进行肾活检。

5.1．2根据病理检查：可靠的诊断分型有赖于病理组织检查。若病理证实为新月体肾小球肾炎，病理诊断标准必须强调两点: 新出现的新月体为闭塞肾小球囊腔50%以上的大新月体，又能除外是由系统性疾病引起者，同时检测血清抗GBM抗体、ANCA、诊断即成立。

5.2鉴别诊断

5.2.1急性肾小管坏死：本病有以下特点：①常有明确的发病原因，如中毒因素(药物、鱼胆中毒等)、休克、挤压伤、异型输血等；②病变主要在肾小管，故尿少而且尿比重低于1.010，肾小管回吸收钠功能受损，尿钠常超过20～30mEq/L。可见特征性的大量肾小管上皮细胞。

　5.2.2梗阻性肾病：常见于肾盂或输尿管双侧性结石，或一侧无功能肾伴另侧结石梗阻，膀胱或前列腺肿瘤压迫或血块梗阻等。本病特点为：①原来尿量正常，突然骤减以至无尿者。②有肾绞痛或明显腰痛史。③超声波检查发现膀胱或肾盂积水；X线平片可有结石及肾脏增大；核素肾图有排泌段障碍。④膀胱镜及逆行肾盂造影可发现梗阻病损与部位。

　5.2.3急性间质性肾炎：亦可以急性肾功能衰竭起病，但常伴发热、皮疹等过敏表现，血及尿中嗜酸性粒细胞增高等，可资鉴别。药物过敏所致的急性间质性肾炎血尿不明显。必要时依靠肾活检确诊。

　5.2.4血栓性微血管病：如溶血性尿毒症综合征，血栓性血小板减少性紫癜，恶性高血压产后引起的HELP综合征等。这一类疾病的临床特点是微血管病性溶血(贫血、血涂片可见破碎的头盔状红细胞)及血小板减少。病理形态改变有助确诊。

　5.2.4 原发性肾小球疾病常见于重型急性肾小球肾炎， 其病变较重和( 或) 持续， 个别情况亦可出现新月体， 临床上呈现急进性肾炎综合征， 有血尿、蛋白尿、水肿、高血压， 表现为进行性肾损害， 还要与慢性肾炎急性发作或其他类型肾小球病变鉴别， 如膜性肾病、膜增生性肾炎、IgA 肾病等基础上形成广泛新月体， 即其他病理类型转化而来的新月体肾炎进行鉴别。常常需要做肾活检协助诊断。

6. 发病后的主要并发症

6.1泌尿系统：由于少尿期体内水、钠的蓄积，患者可出现高血压、肺水肿和心力衰竭。当蛋白质的代谢产物不能经肾脏排泄，造成含氮物质在体内积聚时出现氮质血症。如同时伴有感染、损伤、发热，则蛋白质分解代谢加快，血中尿素氮、肌酐升高，即形成尿毒症。

6.2水、电解质紊乱和酸碱平衡失常：水过多见于摄入量或补液量过多，出水量如呕吐、出汗、伤口渗透量等估计不准确，以及液量补充时忽略计算内生水。随少尿期延长，易发生水过多，表现为稀释性低钠血症、软组织水肿、体重增加、高血压、急性心力衰竭和脑水肿等。

6.3心血管系统：①高血压。②急性肺水肿和心力衰竭：是少尿期常见死亡原因。它主要为体液潴留引起，但高血压、严重感染、心律失常和酸中毒等均为影响因素。③心律失常、心包炎。

6.4消化系统：食欲减退、恶心、呕吐、腹胀、腹泻等。上消化道出血是常见的晚期并发症。

6.5神经系统：轻型患者可无症状，部分患者早期表现疲倦、精神较差。若早期出现意识淡漠、嗜睡或烦躁不安，甚至昏迷，提示病情重笃，不宜拖延透析时间。

6.6贫血和低蛋白血症常伴中度或严重贫血， 贫血出现早， 进展快。血红蛋白进行性下降。

急诊处理

1.对症治疗

卧床休息，给无盐或低盐饮食，以维持和调整容量与电解质平衡，纠正代谢性酸中毒如碳酸氢钠，严格控制高血压如CCB类降压药等，病情需时可利用利尿剂和血管扩张剂。肾功能损害严重者伴全身症状者，早期安排连续性血液净化治疗。及早安排肾脏病理穿刺活检明确病理损害类型。

2.强化治疗

2.1　强化血浆置换疗法：该法是用血浆置换机分离患者的血浆和血细胞， 弃去血浆，然后用等量正常人的血浆或血浆成分(如白蛋白)和患者的血细胞重新输入体内， 每日或隔日置换1次，每次置换2～4L。直到血清抗体转阴或病情好转，一般需10次左右此法清除致病抗体及循环免疫复合物的疗效肯定， 已被临床广泛应用。一般需配合糖皮质激素及环磷酰胺治疗，以抑制有害抗体再生。

常口服泼尼松或泼尼松龙: 起始量足1mg/(kg•d)，不过最大剂量常不超过60mg/d；减、撤药慢(足量服用2～3月后开始减药)。细胞毒药物常用环磷酰胺， 每日口服2～3mg/(kg•d)，累积量不超过8g。必须注意骨髓抑制及肝脏损伤等不良反应。适用于各型急进性肾炎，但主适用于Ⅰ型和就诊时急性肾衰竭需透析的Ⅲ型。

2.2　甲泼尼龙冲击联合环磷酰胺治疗： 甲泼尼龙0.5～1.0g/次，3次为一疗程，必时可间隔3～5ｄ后开始下一个疗程，一般总共不超过3个疗程。对Ⅱ、Ⅲ型效果较好，对Ⅰ型效果较差。需配合糖皮质激素及环磷酰胺常规口服。用每月一次的环磷酰胺冲击治疗代替常规剂量每日口服，有助于减少环磷酰胺的不良反应，但有待进一步研究评估其风险及收益。

2.3　其他免疫抑制药：麦考酚酸酯抑制免疫疗效肯定， 而不良反应较细胞毒药物轻， 已被广泛应用于肾病治疗， 包括Ⅱ及Ⅲ型RPGN。起始剂量1～2g/d(常为1.5g /d)， 以后每半年减0.5g/d，最后以0.5g/d剂量维持半年至1年。硫唑嘌呤常用于新月体肾炎的维持性治疗。

3.替代治疗：如果患者肾功能急剧恶化达到透析指征时， 应尽早进行透析治疗(包括血液透析或腹膜透析)， 以维持生命，赢得治疗时间。如果治疗过晚， 疾病已进入不可逆性终末期肾衰竭，则应予患者长期维持透析治疗或肾移植。肾移植应在病情静止半年至1年、血中致病抗体( 抗GBM 抗体、ANCA等) 阴转后才进行，以免术后移植肾再发RPGN。

疗效标准

1.治愈：患者肾能完全恢复，各项并发症均已治愈；或肾能稳定，没有需要住院处理的并发症。

2.好转：病情明显好转，症状体征尚未完全恢复正常，或肾能等查处在好转中，各项并发症尚未完全治愈。

3.未愈：未达到上述标准者。

出院标准

凡达到床治愈者可出院；肾功能稳定，没有需要住院处理的并发症可出院；病情好转或愈者需家属签字要求出院者方可转院或出院。
急诊建议
1．出现急性肾损伤后患者有时会出现血压升高、心衰、肺水肿等表现，使病情加重或恶化，要详细向患者或家属说明，争取合作。

2．少数患者可出现高钾血症至心律失常或心脏骤停。

3．肾脏病理活检为侵入性检查，有一定创伤，可出现血尿、血肿、肾撕裂伤、肾动静脉瘘等并发症，要详细向患者或家属说明，争取合作。

4．动态观察肾功能变化，如发展至终末期肾衰竭或已有严重并发症，尽早行血液净化治疗挽救生命。

5. 条件具备的医院应争取做肾脏病理活检，如患者不愿做该检查也要完善血清免疫学检测。
