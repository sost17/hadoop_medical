食道、胃静脉瘤破裂
位于食道黏膜下部、胃的上部的静脉壁膨胀、血管形成瘤状的疾病，当压力逐渐升高，就会导致静脉瘤破裂大出血。感染、外伤以及血管疾病是造成静脉瘤的常见的原因。在静脉瘤治疗方面，其方法与静脉扩张相似，静脉过度扩张,血管壁变薄而破裂出血。因此，对于静脉瘤，手术治疗是多数病人较有效的方法。

发病机制

病因

感染、外伤以及血管疾病是造成静瘤的常见的原因。当静过度扩张，血管壁变薄，静内的压力可使血管壁向食管腔内突出，当进食硬或粗糙的食物时，血管壁受摩擦可破裂而引起大出血。剧烈的情绪波动可致血压升高，静充盈，静压升高，静壁破裂出血。肝硬化等肝脏疾病，造成的门静血压升高，使食管胃底扩张，类似静瘤，破裂后引起大出血。

病理

发病初期，食管、胃脉瘤呈青蓝色紫蓝色圆形卵圆形扁平状隆起，边界清楚，表面黏膜完好。随着病情发展，黏膜可因炎症水肿，因此较小的外力脉压升高，可引起脉壁破裂出血。当脉瘤为多发时，脉瘤之间的食管、胃黏膜可正常，脉瘤可能是间隔而非连续性的。
院前急救判断与处置

院前急救判断

1.电话询问

n 年龄、性别、既往有无感染、外伤以及血管疾病、肝硬化病史？

n 是否突然发病？

n 起病前有无情绪激动或用力、进食速度过快等诱因？

n 有无头晕、胸闷、气促、呕血、意识障碍等？

n 生命体征是否正常、稳定？

2.途中思考

n 可能引起呕血的病因，避免发生误诊或漏诊。

3.可危及生命的事件

n 呕吐物堵塞气道窒息

n 大出血致失血性休克

4.现场评估

4.1临床症状

头晕、面色苍白、腹部胸部闷胀、心跳加速、恶心、解黑便或血便、甚至出现大量呕血等。

4.2急危重症评估

在短时间内大量呕血、出现休克症状患者。

院前急救处置

1．取平卧头侧位，利于口腔呕吐物流出。

2．保持呼吸道通畅，及时清理呼吸道分泌物，防止误吸，必要时气管插管通畅气道。

3．给予吸氧，保持供氧充分。

4．严密监测压、心率、呼吸等主要生命体征的变化。

5．维持有效液循环（静脉补液、输）。

6．使用止药物。

选择医疗机构与转运

1.选择医疗机构

怀疑食道、胃静脉瘤破裂患者应该尽可能收治到内视镜下行食道、胃静脉瘤注射硬化剂或结扎术治疗的医疗机构。

2.转运时注意

要及时转送，转送时移动应避免刺激；对伴呕血的患者要注意转送时患者体位的摆放；给以吸氧、密切监测命体征。

急诊科诊断与治疗

初步评估与急救

1.就地诊治，卧床休息，避免声光刺激。

2.保持呼吸道通畅，平卧头侧位，以利于口腔呕吐物流出，保持气道通畅，必要时行气管插管。

3. 维持有效血液循（给以静脉补液、输血）；使用止血物。

全面诊断与检查

1.状

头晕、面色苍白、腹部胸部闷胀、心跳加速、恶心、解黑便或血便、甚至出现大量呕血等。

3.实验室检查和其他检查

急诊查血常规、生化、凝血指标、交叉配血等。病情允许行内镜检查，此外还进行CT诊察、超声内镜检查，为鉴别诊断提供依据，进一步了解引起出血的能原因，判断治疗效果等。一般来说，在掌握适应的情况下，利用内镜观察食管静脉瘤的出血、糜烂以及周围的血管等，为诊断和治疗提供较靠的依据。

急诊处理

1.对症治疗

1.1使用止血药。

1.2放置鼻胃管洗胃及引流血液。

1.3放置食道球直接压迫止血。

1.4经内镜食道、胃静脉瘤注射硬化剂或实施结扎术治疗。

①硬化疗法

利用内镜确认静脉瘤，予静脉瘤注入硬化剂，如鱼肝油酸钠，使静脉瘤硬化而达到治疗目的。　

②静脉瘤结扎术

利用内镜把要结扎的静脉瘤入到结扎器中，用橡皮圈套扎在静脉瘤基底部，达到止血目的。该方法所致的痛苦较少、简便安全，但发率较高。

疗效标准

1.治愈：①临床主要症状消失。②胃镜复查活动出血消失，食管、胃静脉瘤表面黏膜完好，无新近或陈旧出血灶。

2.好转：①临床主要症状消失或基本消失。②胃镜复查活动出血消失，食管、胃静脉瘤表面粘膜较完好，无新近或陈旧出血灶。

3.未愈：未达到上述准者。

出院标准

凡达到临床治愈者可出，好转或未愈者需家属签字要求出者可转或出。
急诊建议
1.急性食管、胃静脉瘤破裂患者因详细问清病史，既往疾病，寻找病因，避免出现误诊或漏诊。

2.严密监测患者生命体征，动态监测，防范严重大出血误吸至窒息发生。

3.在大出血患者中因加强补液输血扩容抗休克处理，尽早在内镜下行食道、胃静脉瘤注射硬化剂或实施结术治疗。
