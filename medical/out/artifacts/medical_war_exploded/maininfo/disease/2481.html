流行性出血热
流行性出血热是一种自然疫源性疾病，鼠为主要传染源。冬春季多见。本病潜伏期4～46 天，一般为7～14 天，以2 周多见。临床上以发热、休克、充血出血和急性肾衰竭为主要表现，又称为肾综合征出血热。

发病机制

病因

1.宿主动物和传染源

是小型啮齿动物，包括姬鼠属（主要为黑线姬鼠）、大鼠属（主要为褐家鼠、大白鼠）、鼠（棕背、红背）、田鼠属（主要为东方田鼠）、仓鼠属（主要为黑线仓鼠）和小鼠属（小家鼠，小白鼠）。我国已查出30种以上动物可自然携带本病毒。除啮齿动物外，一些家畜也携带EHFV，包括家猫、家兔、狗、猪等，证明有多宿主性。这些动物多偶然性携带，只有少数几个鼠种从流行病学证明为本病的传染源，其中在我国黑线姬鼠为野鼠型出血热的主要宿主和传染源，褐家鼠为城市型（日本、朝鲜）和我国家鼠型出血热的主要传染源，大林姬鼠是我国林区出血热的主要传染源。至于其他携带本病毒的鼠类在流行病学上的作用有待进一步观察研究。

2.传播途径

主要传播为动物源性，病毒能通过宿主动物的血及唾液、尿、便排出。鼠向人的直接传播是人类感染的重要途径。本病的传播途径有以下5种。

（1）呼吸道传播 鼠类携带病毒的排泄物如尿、粪、唾液等污染尘埃后形成的气溶胶，能通过呼吸道而感染人体。

（2）消化道传播 进食被鼠类携带病毒的排泄物所污染的食物，可经口腔或胃肠粘膜而感染。

（3）接触传播 被鼠咬伤或破伤口接触病毒的鼠类血液和排泄物亦可导致感染。

（4）母婴传播 孕妇感染本病后，病毒可经胎盘感染胎儿。

（5）虫媒传播 有研究表明寄生于鼠类身上的革螨等具有传播作用。

3.人群易感性

一般认为人群普遍易感，隐性感染率较低，在野鼠型多为3～4%以下。但家鼠型疫区隐性感染率较高，有报告为15%以上。一般青壮年发病率高，二次感染发病罕见。病后在发热期即可检出血清特异性抗体，1～2周可达很高水平，抗体持续时间长 。

EHF主要表现的发病机制

1.低血压及休克

1.1小血管壁损害，血浆大量外渗；致血液浓缩，血容量减少。

1.2小动脉扩张，外周阻力降低；毛细血管开放，微循环淤血，回心血量进一步减少至低血压休克。

2.出血

2.1毛细血管壁脆性增加，管壁坏死引起出血。

2.2血小板减少：由于血小板在骨髓中生成受抑制，血小板变性和聚集，以及并发DIC，使血小板减少引起出血。

3.急性肾功能衰竭

3.1肾血管损害和肾小管上皮细胞变性坏死。

3.2肾小球和肾小管基底膜上有免疫复合物（IC）沉着，肾小球基底膜孔增大，负电荷丢失；肾小管对蛋白质再吸收功能障碍，致蛋白漏出是蛋白质生成的主要原因。

3.3血管通透性增加，血浆外渗，有效循环血量下降，肾血流量减少，致肾小管缺血，加重肾小管上皮细胞损害，导致肾功能衰竭。

4.高血容量综合征

4.1少尿期的中后阶段，尿量减少，血管壁病损恢复外渗停止，外渗液体主要由淋巴系统回吸收入血循环，致血容量增多。

4.2心负荷增加导致心力衰竭。

5.水肿

由于小血管扩张淤血，血管通透性增强，血浆外渗所致；此外尿毒症时加重了血管通透性亦为原因之一。

病理

院前急救判断与处置

院前急救判断

1.电话询问

n 有无野外史。有无进食含出血热病毒的鼠排泄物污染的食物。

n 有无畏寒发热，全身中毒症状，高度乏力，全身酸痛，头痛和剧烈腰痛、眼眶痛。

n 有无颜面、颈部、胸部发红？

n 尿量情况？

n 呼吸困难？

n 生命体征稳定吗？

2.途中思考

n 登革热？

n 流行性脑膜炎？

n 原发性血小板减少性紫癜？

n 急性肾脏疾患？

3.可危及生命的事件

n 休克

n 肾功能不全

n 呼吸衰

n 多器官衰

4.现场评估

6.1患者有野外史，开放性损伤感染史。

6.2出现全身中毒症状，高度乏力，全身酸痛，头痛和剧烈腰痛、眼眶痛，称为“三痛”。

6.3出现低血压、休克、少尿、无尿及严重出血等症状典型表现。

6.4皮肤粘膜有出血。

院前急救处置

1．卧床休息、吸氧、迅速建静脉通道。

2．心电监护、血压、血氧饱和度监护、查心电图、血糖。

3．发热期的治疗

3.1 改善中毒症状：高热采用物理降温，中毒症状重者可给予肾上腺皮质激素如地塞米松5~10ml静脉滴注。

3.2 减轻外渗：可大剂量维生素C降低血管通透性，用甘露醇提高血浆渗透压。

3.3 预防DIC：予低分子右旋糖苷500ml静脉滴注以降低血液粘滞性。

4.低血压休克期治疗

4.1 积极扩容：液体应晶、胶体结合，以平衡液为主；休克重者常用双渗平衡盐液。

4.2 纠正酸中毒：常用5%碳酸氢钠。

4.3 改善循环：可用血管活性药物（多巴胺、盐酸山莨菪碱等）。

5．保持呼吸道通畅：呼吸道分泌物多且排出困难者、痉挛时间长或已发生窒息者应及早行气管切开，注意吸痰。

6．控制抽搐使用安定、鲁米那等。

7. 出血 ：有消化道出血者可口服云南白药、三七粉等，无效时可用去甲肾上腺素10mg置于生理盐水100~200ml，以鼻管灌胃。

选择医疗机构与转运

1.择医疗构

怀疑流行性出血热时，要考虑到发病后早期转运不及时可能会导致预后不良，无论患者症状轻重，都应该择转送到能24小时对患者进行处理的医疗构。

2.转运时注意

要及时转送，转送时移动应避免刺激，尽量注意安静状态下转送。在搬运患者时，要注意转送时患者体位的摆放。转送途中密切监测生命体征。

急诊科诊断与治疗

初步评估与急救

1．曾有被鼠咬伤、鼠类排泄物、分泌物直接与破损的皮肤、黏膜接触。

2．发热期 初始尿检无异常，蛋白尿常在第5 天左右突然出现，严重者达3＋，并伴有镜下血尿。血尿素氮(BUN)和血清肌酐(SCr)偶有升高。肾小管功能检查可见尿NAG 酶、溶菌酶升高，尿比重下降。

3．患者潜伏期超过10天以上，起病又慢，及时治疗预后良好。若潜伏期未过7天，在3天内，多项症状出现，提示预后不良。

4．病情危重指标

4.1高热体温≥40℃。

4.2全身中毒症状严重。或出现中毒性精神症状者。

4.3出现休克。

4.4出血现象严重 皮肤瘀斑，腔道出血，甚至重要器官出血。

4.5肾功能损害严重 少尿≥5天或无尿≥2天，氮质血症严重及肾功能衰竭。

4.6高血容量综合征。

4.7肺水肿、心功能衰竭。

4.8中枢神经系统并发症

急诊处理

本病以综合疗法为主，早期应用抗病毒治疗，中晚期则针对病理生理进行对症治疗。 “三早一就”仍为本病治疗的原则。即早期发现，早期休息，早期治疗与就近治疗。治疗中要注意防治休克、肾功能衰竭和出血。

1) 发热期 控制感染，减轻外渗，改善中毒症状。在发病4日以内的患者可应用利巴韦林每日1克静滴，进行抗病毒治疗。早期卧床休息，以体液疗法减轻外渗。高热以物理降温为主；中毒症状重者可给予地塞米松改善中毒症状。采用低分子右旋糖酐、丹参和肝素等进行止血抗凝治疗。预防肾功能衰竭。

2) 低血压休克期 补充血容量，注意酸中毒的纠正和改善微循环功能。用醋酸钠液等补充血容量。用5％的碳酸钠溶液纠正酸中毒。用多巴胺等血管活性物质使血压稳定。

3) 少尿期 稳定机体内环境，促进利尿，导泄和透析治疗。补充体液，促进机体内环境的稳定。用甘露醇减轻肾间质水肿，给予利尿药物促进排尿。为防止高血容量综合征和高血钾，少尿期可进行导泻。对有明显氮质血症，高血钾或高血容量综合征患者，可应用血液透析或腹膜透析。

4) 多尿期 维持水和电解质平衡，防治继发感染。

5) 恢复期 补充营养，逐步恢复工作。定期检查肾功能、血压和脑垂体功能。

疗效标准

1.治愈：症状完全缓解，生命体稳定；需进一步检查的其他情况。

2.未愈：未达到上述标准者。

出院标准

凡达到临床治愈者可院，未愈者需家属签字要求院者方可转院或院。
急诊建议
1、灭鼠是防止本病流行的关键，在流行地区要大力组织群众，在规定的时间内同时进行灭鼠。灭鼠时机应选择在本病流行高峰（5～6月和10～12月）前进行。春季应着重灭家鼠，初冬应着重灭野鼠。

2、灭螨、防螨要保持屋内清洁、通风和干燥，经常用滴滴畏等机磷杀虫剂喷洒灭螨。清除室内外草堆。
3、加强食品卫生做好食品卫生、食具消毒、食物保藏等工作，要防止鼠类排泄物污染食品和食具。剩饭菜必须加热或蒸煮后方可食用。
4、做好消毒工作对发热病人的血、尿和宿主动物尸体及其排泄物等，均应进行消毒处理，防止污染环境。
5、注意个人防护在疫区不直接用手接触鼠类及其排泄物，不坐卧草堆，劳动时防止皮肤破伤，破伤后要消毒包扎。在野外工作时，要穿袜子，扎紧裤腿、袖口，以防螨类叮咬。
